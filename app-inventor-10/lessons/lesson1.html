1.2 什么是 App Inventor？1.2 What is App Inventor?

MIT App Inventor 是由麻省理工学院（MIT）媒体实验室开发的一个在线可视化编程平台。它完全在浏览器中运行，不需要安装任何软件。你只需要一个 Google 账号就能登录使用。通过拖拽"积木块"（Blocks）的方式来编写程序逻辑，即使你从来没写过一行代码，也能轻松上手！ MIT App Inventor is an online visual programming platform developed by MIT's Media Lab. It runs entirely in your browser — no software installation needed. All you need is a Google account to log in. You program by dragging and snapping together "blocks," so even if you've never written a line of code before, you can get started easily!

App Inventor 有两个主要视图：

Designer 视图（设计视图）：在这里你可以设计 App 的外观。左边是组件面板，你可以拖入按钮、文本框、图片等 UI 组件到中间的手机预览区。右边是属性面板，用来设置组件的颜色、文字、大小等。

Blocks 视图（编程视图）：在这里你用积木块编写 App 的逻辑。比如"当按钮被点击时 → 改变文字内容"。积木块像拼图一样咬合在一起，不同颜色代表不同功能类别。 App Inventor has two main views:

Designer View: This is where you design your app's look. The left panel has components (buttons, text boxes, images, etc.) you drag onto the phone preview in the center. The right panel lets you set properties like color, text, and size.

Blocks View: This is where you code your app's logic using blocks. For example, "When Button is clicked → change the text." Blocks snap together like puzzle pieces, and different colors represent different categories of functionality.

App Inventor 支持制作 Android App（也有实验性的 iOS 支持）。你做出来的 App 可以直接安装到手机上使用，也可以生成 APK 文件分享给朋友。全世界已经有超过 195 个国家的学生使用 App Inventor 来学习编程和制作 App！ App Inventor supports building Android apps (with experimental iOS support too). The apps you create can be installed directly on your phone, or you can generate an APK file to share with friends. Students from over 195 countries worldwide already use App Inventor to learn coding and build apps!

1.3 动手实践1.3 Hands-On

现在让我们一步步制作你的第一个 App —— "HelloAI"！这个 App 会在你点击按钮时显示一条 AI 相关的消息。 Now let's build your first App step by step — "HelloAI"! This App will display an AI-related message when you tap a button.

步骤 1：打开 App Inventor 并登录Step 1: Open App Inventor and Sign In

在浏览器中打开 ai2.appinventor.mit.edu，使用你的 Google 账号登录。如果你没有 Google 账号，请先请家长帮你注册一个。登录后你会看到"我的项目"（My Projects）页面。 Open ai2.appinventor.mit.edu in your browser and sign in with your Google account. If you don't have a Google account, ask a parent to help you create one. After signing in, you'll see the "My Projects" page.

步骤 2：创建新项目Step 2: Create a New Project

点击左上角的 "新建项目"（Start new project） 按钮，在弹出框中输入项目名称 HelloAI，然后点击"确定"。你会进入 Designer 视图，看到一个空白的手机屏幕。 Click the "Start new project" button in the top-left corner. In the popup, enter the project name HelloAI and click "OK." You'll enter the Designer view and see a blank phone screen.

步骤 3：添加 UI 组件Step 3: Add UI Components

从左侧的 User Interface 面板中，依次拖入以下组件到手机屏幕上： From the User Interface palette on the left, drag the following components onto the phone screen:

A

Label（标签）Label

拖入一个 Label，在右侧属性面板中将 Text 改为 Hello, AI World!，字体大小设为 24，勾选 FontBoldDrag in a Label. In the Properties panel on the right, set Text to Hello, AI World!, font size to 24, and check FontBold

B

Button（按钮）Button

拖入一个 Button，将 Text 改为 Click Me，可以修改 BackgroundColor 让按钮更好看Drag in a Button. Set Text to Click Me. You can change BackgroundColor to make it look nicer

C

Image（图片）Image

拖入一个 Image 组件，点击 Picture 属性，上传一张你喜欢的图片（比如一个 AI 机器人图标）Drag in an Image component. Click the Picture property and upload an image you like (e.g., an AI robot icon)

步骤 4：切换到 Blocks 视图，编写逻辑Step 4: Switch to Blocks View and Add Logic

点击右上角的 "Blocks" 按钮切换到编程视图。现在我们要给按钮添加一个"点击事件"： Click the "Blocks" button in the top-right to switch to the programming view. Now let's add a "click event" to the button:

积木块逻辑Block Logic 
// 积木块伪代码 / Block pseudocode

When Button1.Click
  do set Label1.Text to "AI is Amazing!"

操作步骤：在左侧点击 Button1，找到 when Button1.Click do 积木块，拖到工作区。然后点击 Label1，找到 set Label1.Text to 积木块，拼接到 Click 积木块内部。最后从 Text 类别中拖入一个文本块，输入 AI is Amazing!。 How to do it: On the left, click Button1 and find the when Button1.Click do block — drag it to the workspace. Then click Label1, find the set Label1.Text to block, and snap it inside the Click block. Finally, from the Text category, drag in a text block and type AI is Amazing!.

步骤 5：连接手机测试Step 5: Connect Your Phone to Test

现在让我们在真实手机上看看效果！你需要在手机的应用商店中下载 MIT AI2 Companion App。 Now let's see your App on a real phone! You need to download the MIT AI2 Companion app from your phone's app store.

如何在手机上测试：

1. 在手机上安装 MIT AI2 Companion（Android 在 Google Play 搜索，iPhone 在 App Store 搜索）
2. 确保手机和电脑连接同一个 Wi-Fi 网络
3. 在 App Inventor 网页上点击菜单栏的 Connect → AI Companion
4. 屏幕上会出现一个二维码和 6 位代码
5. 打开手机上的 MIT AI2 Companion，扫描二维码或输入代码
6. 几秒后，你的 App 就会出现在手机上！

没有手机？你也可以使用内置模拟器：点击 Connect → Emulator。 How to test on your phone:

1. Install MIT AI2 Companion on your phone (search Google Play for Android, or App Store for iPhone)
2. Make sure your phone and computer are on the same Wi-Fi network
3. In the App Inventor website, click Connect → AI Companion from the menu bar
4. A QR code and a 6-character code will appear on screen
5. Open MIT AI2 Companion on your phone and scan the QR code or enter the code
6. In a few seconds, your App will appear on your phone!

No phone? You can also use the built-in emulator: click Connect → Emulator.

点击手机上的 "Click Me" 按钮，你应该会看到文字从 "Hello, AI World!" 变成了 "AI is Amazing!" —— 恭喜你，你的第一个 App 成功了！ Tap the "Click Me" button on your phone. You should see the text change from "Hello, AI World!" to "AI is Amazing!" — Congratulations, your first App works!

1.4 挑战任务1.4 Challenge

完成了基本的 HelloAI App 后，试试下面的挑战来提升你的技能吧！ Now that you've completed the basic HelloAI App, try these challenges to level up your skills!

挑战 A：消息轮播Challenge A: Message Cycle

修改按钮逻辑，让每次点击按钮时在 3 条不同的消息之间循环切换：
① "AI is Amazing!"
② "AI 正在改变世界！"
③ "I love building Apps!"

提示：你需要用一个变量（Variable）来记住当前显示的是第几条消息，再用 if / else if 积木块来判断应该显示哪一条。 Modify the button logic so each tap cycles through 3 different messages:
① "AI is Amazing!"
② "AI is changing the world!"
③ "I love building Apps!"

Hint: You'll need a variable to track which message is currently displayed, then use if / else if blocks to decide which one to show next.

挑战 B：变色背景Challenge B: Color-Changing Background

添加第二个按钮 "Change Color"，每次点击时将 Screen 的背景颜色（BackgroundColor）在红色、蓝色、绿色之间切换。

提示：在 Blocks 视图中，你可以用 set Screen1.BackgroundColor to 来改变背景颜色。颜色值可以从 Colors 积木块类别中获取。 Add a second button "Change Color" that changes the Screen's background color (BackgroundColor) cycling between red, blue, and green on each tap.

Hint: In the Blocks view, use set Screen1.BackgroundColor to to change the background. You can find color values in the Colors block category.

课堂测验Lesson Quiz

1. MIT App Inventor 是什么？1. What is MIT App Inventor?

MIT App Inventor 是由麻省理工学院开发的在线可视化编程平台，完全在浏览器中运行，通过拖拽积木块来制作手机 App。MIT App Inventor is an online visual programming platform developed by MIT. It runs entirely in the browser and lets you build mobile apps by dragging and dropping blocks.

2. App Inventor 的 Designer 视图和 Blocks 视图分别用来做什么？2. What are the Designer view and Blocks view used for in App Inventor?

Designer 视图用来设计 App 的界面（拖入按钮、文字等组件），Blocks 视图用来编写 App 的逻辑（用积木块定义点击按钮后会发生什么）。The Designer view is for designing the app's interface (dragging in buttons, labels, etc.), while the Blocks view is for programming the app's logic (using blocks to define what happens when a button is clicked).

3. 如何在手机上实时测试你的 App？3. How do you test your App on a phone in real time?

使用 MIT AI2 Companion App 可以实时在手机上测试。确保手机和电脑在同一 Wi-Fi 网络，然后在 App Inventor 中选择 Connect → AI Companion，扫描二维码即可。Use the MIT AI2 Companion app for real-time testing on your phone. Make sure your phone and computer are on the same Wi-Fi network, then select Connect → AI Companion in App Inventor and scan the QR code.

4. 在 Blocks 视图中，when Button1.Click 积木块的作用是什么？4. In the Blocks view, what does the when Button1.Click block do?

when Button1.Click 是一个事件处理积木块，它定义了当用户点击 Button1 时程序要执行的操作。你可以在它内部拼接其他积木块来设定具体行为。when Button1.Click is an event handler block. It defines what actions the program should perform when the user clicks Button1. You snap other blocks inside it to set the specific behavior.

5. 如果你没有手机，还能怎么测试 App Inventor 项目？5. If you don't have a phone, how else can you test an App Inventor project?

App Inventor 提供了内置的模拟器（Emulator），可以在电脑上模拟手机环境来测试你的 App。点击 Connect → Emulator 即可使用。App Inventor includes a built-in emulator that simulates a phone environment on your computer. Click Connect → Emulator to use it.
