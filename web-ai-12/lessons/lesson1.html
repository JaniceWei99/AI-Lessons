1.2 什么是 HTML？ 1.2 What Is HTML?

HTML 是 HyperText Markup Language（超文本标记语言）的缩写。它不是一种编程语言，而是一种标记语言——用来告诉浏览器"这里是标题""这里是段落""这里是图片"。 HTML stands for HyperText Markup Language. It's not a programming language — it's a markup language that tells the browser "this is a heading," "this is a paragraph," "this is an image."

你可以把 HTML 想象成网页的"骨架"。就像人的骨骼支撑着身体一样，HTML 定义了网页的结构和内容。 Think of HTML as the "skeleton" of a webpage. Just as bones support the human body, HTML defines the structure and content of a page.

标签、元素与属性Tags, Elements & Attributes

🏷️

标签 (Tags)Tags

用尖括号包裹的关键词，如 <p>、<h1>。大多数标签成对出现：开始标签和结束标签。Keywords wrapped in angle brackets, like <p>, <h1>. Most tags come in pairs: an opening tag and a closing tag.

📦

元素 (Elements)Elements

开始标签 + 内容 + 结束标签 = 一个完整的元素。例如 <p>你好</p> 就是一个段落元素。Opening tag + content + closing tag = a complete element. For example, <p>Hello</p> is a paragraph element.

⚙️

属性 (Attributes)Attributes

在开始标签中添加额外信息，如 <a href="..."> 中的 href 就是属性。Extra info added to opening tags, like href in <a href="..."> is an attribute.

HTML 基本结构Basic HTML Structure

每个 HTML 文件都遵循一个固定的基本结构。下面是最简单的 HTML 页面： Every HTML file follows a standard structure. Here's the simplest HTML page:

HTML 
<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8">
    <title>我的第一个网页</title>
  </head>
  <body>
    <h1>你好，世界！</h1>
    <p>这是我的第一个网页。</p>
  </body>
</html>

🔍 解读：<!DOCTYPE html> 告诉浏览器这是 HTML5 文档。<head> 包含页面元信息（如标题），<body> 包含用户看到的所有内容。 🔍 Breakdown: <!DOCTYPE html> tells the browser this is an HTML5 document. <head> contains metadata (like the title), and <body> contains everything the user sees.

1.3 动手实践 1.3 Hands-on Practice

现在让我们来创建一个包含标题、段落和图片的完整网页！ Now let's create a complete webpage with headings, paragraphs, and images!

标题标签 h1 到 h6Heading Tags h1 to h6

HTML 提供了 6 级标题，<h1> 最大最重要，<h6> 最小。每个页面通常只有一个 <h1>。 HTML provides 6 levels of headings. <h1> is the largest and most important, <h6> the smallest. Each page usually has only one <h1>.

HTML 
<h1>一级标题（最大）</h1>
<h2>二级标题</h2>
<h3>三级标题</h3>
<h4>四级标题</h4>
<h5>五级标题</h5>
<h6>六级标题（最小）</h6>

段落和图片Paragraphs and Images

<p> 标签用来创建段落。<img> 标签用来插入图片——注意它是一个自闭合标签，不需要结束标签。 The <p> tag creates paragraphs. The <img> tag inserts images — note that it's a self-closing tag with no closing tag needed.

HTML 
<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8">
    <title>关于我</title>
  </head>
  <body>
    <h1>大家好！</h1>
    <p>我叫小明，今年 14 岁，正在学习做网页。</p>
    <h2>我的爱好</h2>
    <p>我喜欢编程、画画和打篮球。</p>
    <img src="my-photo.jpg" alt="我的照片">
  </body>
</html>

⚠️ 重要：文件必须保存为 .html 后缀（例如 index.html），然后用浏览器打开。alt 属性是给图片加上文字说明，当图片无法显示时会显示这段文字。 ⚠️ Important: Save your file with a .html extension (e.g. index.html), then open it in a browser. The alt attribute provides a text description of the image, shown when the image can't load.

💡 小技巧：你可以使用任何文本编辑器来写 HTML，推荐使用 VS Code——它免费、强大，还有很多好用的插件。 💡 Pro tip: You can use any text editor to write HTML. We recommend VS Code — it's free, powerful, and has tons of helpful extensions.

课堂测验Lesson Quiz

1. HTML 的全称是什么？1. What does HTML stand for?

HTML 全称为 HyperText Markup Language（超文本标记语言），是构建网页的基础语言。HTML stands for HyperText Markup Language — the foundational language for building webpages.

2. <h1> 标签的作用是什么？2. What does the <h1> tag do?

<h1> 是最高级别的标题标签，通常用于页面的主标题。<h6> 才是最小的标题。<h1> is the highest-level heading tag, typically used for the page's main title. <h6> is the smallest heading.

3. <img> 标签必须包含哪个属性来描述图片内容？3. Which attribute must the <img> tag include to describe the image?

alt 属性为图片提供替代文本，当图片无法加载时显示，也有助于屏幕阅读器理解图片内容。The alt attribute provides alternative text for the image, shown when the image can't load, and helps screen readers understand the image.

4. 以下哪个是正确的 HTML 基本结构？4. Which of the following is the correct basic HTML structure?

正确顺序是 <html> 包裹一切，里面先是 <head>（元信息），再是 <body>（可见内容）。The correct order is <html> wrapping everything, with <head> (metadata) first, then <body> (visible content).

5. <!DOCTYPE html> 的作用是什么？5. What is the purpose of <!DOCTYPE html>?

<!DOCTYPE html> 不是 HTML 标签，而是一个声明，告诉浏览器当前文档使用 HTML5 标准解析。<!DOCTYPE html> is not an HTML tag but a declaration that tells the browser to parse the document using the HTML5 standard.
