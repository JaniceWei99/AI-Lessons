第 2 课 Lesson 2

HTML 基础（下）— 链接、列表与表单 HTML Basics (Part 2) — Links, Lists & Forms

学习如何用链接连接页面、用列表组织信息、用表单收集用户输入。 Learn to connect pages with links, organize info with lists, and collect user input with forms.

2.1 课程目标 2.1 Lesson Goals

在上一课中，我们学会了用标题、段落和图片构建网页。这一课，我们将学习三个新的重要 HTML 功能： In the last lesson, we learned to build pages with headings, paragraphs, and images. This lesson covers three important new HTML features:

🔗

超链接Hyperlinks

让用户可以在不同页面之间跳转，也可以链接到外部网站。Let users navigate between pages and link to external websites.

📋

列表Lists

用有序列表和无序列表来清晰地组织和展示信息。Use ordered and unordered lists to organize and display information clearly.

📝

表单Forms

创建输入框、下拉菜单和按钮，让用户可以与网页交互。Create input fields, dropdowns, and buttons so users can interact with your page.

2.2 超链接 <a> 2.2 Hyperlinks <a>

<a> 标签（anchor，锚点）用来创建超链接。href 属性指定链接的目标地址。 The <a> tag (anchor) creates hyperlinks. The href attribute specifies the destination URL.

HTML 
<!-- 链接到外部网站 -->
<a href="https://www.example.com">访问 Example 网站</a>

<!-- 链接到本地页面 -->
<a href="about.html">关于我们</a>

<!-- 在新标签页打开链接 -->
<a href="https://www.example.com" target="_blank">
  在新窗口打开
</a>

💡 小技巧：使用 target="_blank" 可以让链接在新标签页中打开，这样用户不会离开你的网站。 💡 Tip: Use target="_blank" to open links in a new tab, so users don't leave your site.

你也可以创建页内锚点链接，让用户跳转到同一页面的不同位置： You can also create anchor links that jump to different sections on the same page:

HTML 
<!-- 导航链接 -->
<a href="#hobbies">跳到爱好部分</a>

<!-- 目标位置 -->
<h2 id="hobbies">我的爱好</h2>

2.3 列表 2.3 Lists

HTML 提供两种基本列表类型：无序列表（项目符号）和有序列表（编号）。 HTML provides two basic list types: unordered lists (bullet points) and ordered lists (numbered).

无序列表 <ul>Unordered List <ul>

HTML 
<ul>
  <li>苹果</li>
  <li>香蕉</li>
  <li>橙子</li>
</ul>

显示为圆点列表，用于顺序不重要的项目。Displays bullet points — used when order doesn't matter.

有序列表 <ol>Ordered List <ol>

HTML 
<ol>
  <li>打开编辑器</li>
  <li>编写代码</li>
  <li>保存文件</li>
</ol>

显示为编号列表，用于有先后顺序的步骤。Displays numbered items — used when order matters (like steps).

💡 注意：每个列表项都使用 <li>（list item）标签。列表还可以嵌套——在 <li> 里放入另一个 <ul> 或 <ol>。 💡 Note: Each list item uses the <li> (list item) tag. Lists can be nested — put another <ul> or <ol> inside an <li>.

2.4 表单基础 2.4 Form Basics

表单是网页中收集用户信息的重要工具。登录页面、搜索栏、联系表单——这些都是用 HTML 表单构建的。 Forms are essential for collecting user information on the web. Login pages, search bars, contact forms — they're all built with HTML forms.

常用表单元素Common Form Elements

✏️

<input> 输入框<input> Input Field

最常用的表单元素。type 属性决定输入类型：text（文本）、email（邮箱）、password（密码）、number（数字）等。The most common form element. The type attribute determines input type: text, email, password, number, etc.

📄

<textarea> 文本区域<textarea> Text Area

用于多行文本输入，适合留言、评论等较长内容。For multi-line text input — great for messages, comments, and longer content.

📃

<select> 下拉菜单<select> Dropdown

提供一个下拉选择列表，用户从预设选项中选择一个。Provides a dropdown list where users pick from predefined options.

🔘

<button> 按钮<button> Button

用户点击按钮来提交表单或触发操作。Users click buttons to submit forms or trigger actions.

完整的联系表单示例Complete Contact Form Example

HTML 
<form>
  <label for="name">姓名：</label>
  <input type="text" id="name" placeholder="请输入你的姓名">

  <label for="email">邮箱：</label>
  <input type="email" id="email" placeholder="example@mail.com">

  <label for="subject">主题：</label>
  <select id="subject">
    <option value="question">问题咨询</option>
    <option value="feedback">意见反馈</option>
    <option value="other">其他</option>
  </select>

  <label for="message">留言：</label>
  <textarea id="message" rows="4" placeholder="写下你的留言..."></textarea>

  <button type="submit">发送</button>
</form>

⚠️ 注意：<label> 标签的 for 属性必须和对应 <input> 的 id 一致。这不仅方便用户点击标签聚焦到输入框，还能提升无障碍访问体验。 ⚠️ Note: The for attribute of <label> must match the id of the corresponding <input>. This lets users click the label to focus the input and improves accessibility.

2.5 动手实践 2.5 Hands-on Practice

让我们把学到的链接、列表和表单结合起来，制作一个完整的个人信息页面！ Let's combine links, lists, and forms to build a complete personal info page!

HTML 
<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8">
    <title>我的个人主页</title>
  </head>
  <body>
    <!-- 导航菜单 -->
    <nav>
      <a href="#about">关于我</a> |
      <a href="#hobbies">我的爱好</a> |
      <a href="#contact">联系我</a>
    </nav>

    <h1>欢迎来到我的主页</h1>

    <h2 id="about">关于我</h2>
    <p>大家好！我是一名热爱编程的学生。</p>
    <p>点击这里访问
      <a href="https://www.example.com" target="_blank">
        我喜欢的网站
      </a>。
    </p>

    <h2 id="hobbies">我的爱好</h2>
    <ul>
      <li>编程</li>
      <li>画画</li>
      <li>打篮球</li>
      <li>阅读科幻小说</li>
    </ul>

    <h2 id="contact">联系我</h2>
    <form>
      <label for="visitor-name">你的名字：</label>
      <input type="text" id="visitor-name">
      <label for="visitor-msg">留言：</label>
      <textarea id="visitor-msg" rows="3"></textarea>
      <button type="submit">发送留言</button>
    </form>
  </body>
</html>

💡 试一试：把上面的代码保存为 profile.html，然后在浏览器中打开。点击导航链接，看看页面是否会跳到对应的部分！ 💡 Try it: Save the code above as profile.html and open it in your browser. Click the nav links and see if the page scrolls to the corresponding sections!

课堂测验Lesson Quiz

1. 哪个属性用来指定超链接的目标地址？1. Which attribute specifies the destination of a hyperlink?

href（Hypertext Reference）是 <a> 标签中用来指定链接目标的属性。href (Hypertext Reference) is the attribute in the <a> tag that specifies the link destination.

2. 哪个标签用来创建无序列表？2. Which tag creates an unordered list?

<ul>（Unordered List）创建无序列表，显示为项目符号。<ol> 是有序列表。<ul> (Unordered List) creates a bullet-point list. <ol> is for ordered (numbered) lists.

3. 在表单中，哪个标签用来创建多行文本输入？3. Which form tag creates a multi-line text input?

<textarea> 创建可以输入多行文本的区域。<input type="text"> 只能输入一行文字。<textarea> creates a multi-line text area. <input type="text"> is single-line only.

4. target="_blank" 属性的作用是什么？4. What does the target="_blank" attribute do?

target="_blank" 让浏览器在新的标签页或窗口中打开链接，而不是在当前页面跳转。target="_blank" tells the browser to open the link in a new tab or window instead of navigating away from the current page.

5. <label> 标签的 for 属性应该与什么匹配？5. What should the for attribute of <label> match?

<label> 的 for 属性需要与对应表单元素的 id 属性保持一致，这样点击标签就能聚焦到输入框。The for attribute of <label> must match the id of the corresponding form element, so clicking the label focuses the input.
