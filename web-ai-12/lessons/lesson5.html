5.2 什么是 JavaScript？5.2 What Is JavaScript?

如果 HTML 是网页的"骨架"，CSS 是"衣服"，那么 JavaScript 就是网页的"大脑"。它让网页变得有互动性——按钮可以点击、内容可以动态更新、表单可以验证。If HTML is the "skeleton" of a web page and CSS is the "clothing," then JavaScript is the "brain". It makes pages interactive — buttons can be clicked, content can update dynamically, forms can be validated.

如何引入 JavaScriptHow to Include JavaScript

有两种常见方式：There are two common ways:

HTML 
<!-- 方式一：直接在 HTML 中写 JS -->
<script>
  console.log("Hello from inline script!");
</script>

<!-- 方式二：引入外部 JS 文件（推荐） -->
<script src="script.js"></script>

最佳实践：把 <script> 标签放在 </body> 结束标签之前，这样 HTML 内容会先加载完毕，JS 再执行，避免页面空白。Best practice: Place the <script> tag just before the </body> closing tag. This ensures HTML loads first, then JS runs, avoiding blank pages.

console.log() — 你的调试好帮手console.log() — Your Debugging Best Friend

console.log() 会把信息输出到浏览器的开发者工具控制台。按 F12 打开开发者工具，切到 Console 标签页就能看到。console.log() prints information to the browser's developer tools console. Press F12 to open DevTools and switch to the Console tab to see it.

JavaScript 
console.log("Hello, World!");      // 输出字符串
console.log(42);                   // 输出数字
console.log(10 + 20);              // 输出计算结果: 30
console.log("My age is", 15);     // 输出多个值

5.3 变量与数据类型5.3 Variables & Data Types

变量就像一个"盒子"，可以存储数据。JavaScript 中用 let 和 const 来声明变量。A variable is like a "box" that stores data. In JavaScript, use let and const to declare variables.

JavaScript 
// let — 可以重新赋值
let score = 0;
score = 100;         // ✅ 可以改变

// const — 不可重新赋值
const name = "Alex";
// name = "Bob";     // ❌ 报错！const 不能改

let vs const：如果变量的值之后会变，用 let；如果是固定不变的值，用 const。尽量多用 const，需要改值时才用 let。let vs const: Use let if the value will change later; use const for values that stay fixed. Prefer const by default, and use let only when you need to reassign.

三种基本数据类型Three Basic Data Types

字符串 StringString

文本数据，用引号包裹。Text data, wrapped in quotes.

let greeting = "Hello!";

数字 NumberNumber

整数或小数，不需要引号。Integers or decimals, no quotes needed.

let age = 15;

布尔值 BooleanBoolean

只有 true 或 false 两个值。Only two values: true or false.

let isStudent = true;

模板字符串Template Literals

用反引号 ` 和 ${} 可以在字符串中嵌入变量，比拼接字符串更方便：Use backticks ` and ${} to embed variables in strings — much easier than string concatenation:

JavaScript 
const name = "Alex";
const age = 15;

// 老方法：字符串拼接
console.log("Hi, I'm " + name + " and I'm " + age);

// 新方法：模板字符串（推荐！）
console.log(`Hi, I'm ${name} and I'm ${age}`);

常用运算符Common Operators

JavaScript 
// 算术运算符
let sum = 10 + 5;    // 15
let diff = 10 - 5;   // 5
let product = 10 * 5; // 50
let quotient = 10 / 3; // 3.333...
let remainder = 10 % 3; // 1（取余数）

// 比较运算符
console.log(5 === 5);   // true（严格相等）
console.log(5 !== 3);   // true（不等于）
console.log(10 > 5);    // true
console.log(3 <= 3);    // true

5.4 函数5.4 Functions

函数是一段可以重复使用的代码块。把常用的逻辑封装成函数，需要时直接调用即可。A function is a reusable block of code. Wrap common logic in a function, and call it whenever you need it.

函数声明Function Declaration

JavaScript 
// 定义函数
function greet(name) {
  return `Hello, ${name}! Welcome to my website.`;
}

// 调用函数
let message = greet("Alex");
console.log(message);  // "Hello, Alex! Welcome to my website."

// 带多个参数的函数
function add(a, b) {
  return a + b;
}
console.log(add(3, 7));  // 10

箭头函数Arrow Functions

箭头函数是更简洁的函数写法，用 => 代替 function 关键字：Arrow functions are a shorter syntax, using => instead of the function keyword:

JavaScript 
// 普通函数
function double(x) {
  return x * 2;
}

// 箭头函数（完整写法）
const double = (x) => {
  return x * 2;
};

// 箭头函数（简写：一个参数 + 一行代码）
const double = x => x * 2;

console.log(double(5));  // 10

什么时候用箭头函数？对于简单的、短小的函数，箭头函数更简洁。对于复杂的逻辑，普通函数声明更清晰。两种写法都可以，先掌握普通函数，再慢慢习惯箭头函数。When to use arrow functions? For simple, short functions, arrow functions are more concise. For complex logic, regular function declarations are clearer. Both work fine — master regular functions first, then get comfortable with arrow functions.

5.5 事件处理5.5 Event Handling

事件是用户与网页交互时触发的动作，比如点击按钮、输入文字、移动鼠标等。JavaScript 可以"监听"这些事件并做出反应。Events are actions triggered when users interact with a page — clicking buttons, typing text, moving the mouse, etc. JavaScript can "listen" for these events and respond.

方式一：onclick 属性Method 1: onclick Attribute

HTML + JavaScript 
<button onclick="sayHello()">Click Me</button>

<script>
function sayHello() {
  alert("Hello! You clicked the button!");
}
</script>

方式二：addEventListener（推荐）Method 2: addEventListener (Recommended)

HTML 
<button id="myBtn">Click Me</button>
<p id="output"></p>
JavaScript 
const btn = document.getElementById("myBtn");
const output = document.getElementById("output");

btn.addEventListener("click", function() {
  output.textContent = "Button was clicked!";
});

为什么推荐 addEventListener？因为它可以给同一个元素添加多个事件监听器，而 onclick 属性只能设置一个处理函数。而且把 JS 和 HTML 分开是更好的代码组织方式。Why prefer addEventListener? It lets you attach multiple listeners to the same element, while onclick can only have one handler. Also, separating JS from HTML is better code organization.

常用事件类型Common Event Types

click（点击）click

鼠标点击元素时触发。Fires when element is clicked.

mouseover（鼠标悬停）mouseover

鼠标移入元素时触发。Fires when mouse enters element.

keydown（按键）keydown

键盘按键按下时触发。Fires when a key is pressed down.

input（输入）input

输入框内容变化时触发。Fires when an input value changes.

5.6 动手实践：简易计数器5.6 Hands-On: Simple Counter

让我们综合运用变量、函数和事件，制作一个点击计数器！Let's combine variables, functions, and events to build a click counter!

HTML 
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Click Counter</title>
  <style>
    body {
      display: flex;
      flex-direction: column;
      align-items: center;
      justify-content: center;
      min-height: 100vh;
      font-family: sans-serif;
      background: #f0f4f8;
    }
    #count {
      font-size: 80px;
      font-weight: bold;
      color: #4f46e5;
    }
    .btn-group { display: flex; gap: 12px; margin-top: 20px; }
    button {
      padding: 12px 28px;
      font-size: 18px;
      border: none;
      border-radius: 8px;
      cursor: pointer;
      color: white;
    }
    .btn-minus { background: #ef4444; }
    .btn-plus  { background: #22c55e; }
    .btn-reset { background: #6b7280; }
  </style>
</head>
<body>
  <p id="count">0</p>
  <div class="btn-group">
    <button class="btn-minus" id="minusBtn">- 1</button>
    <button class="btn-reset" id="resetBtn">Reset</button>
    <button class="btn-plus" id="plusBtn">+ 1</button>
  </div>

  <script>
    let count = 0;
    const countDisplay = document.getElementById("count");

    document.getElementById("plusBtn").addEventListener("click", () => {
      count++;
      countDisplay.textContent = count;
    });

    document.getElementById("minusBtn").addEventListener("click", () => {
      count--;
      countDisplay.textContent = count;
    });

    document.getElementById("resetBtn").addEventListener("click", () => {
      count = 0;
      countDisplay.textContent = count;
    });
  </script>
</body>
</html>

试一试：把上面的代码保存为 counter.html，在浏览器中打开，点击按钮看看效果！Try it: Save the code above as counter.html, open it in your browser, and click the buttons to see it in action!

课堂测验Lesson Quiz

1. JavaScript 在网页中的主要作用是什么？1. What is JavaScript's main role in a web page?

JavaScript 是网页的"大脑"，负责实现交互功能，如响应点击、动态更新内容等。HTML 负责结构，CSS 负责样式。JavaScript is the "brain" of a web page, responsible for interactive features like responding to clicks and dynamically updating content. HTML handles structure, CSS handles styling.

2. 以下哪个是声明一个不可重新赋值的变量的正确方式？2. Which of the following correctly declares a variable that cannot be reassigned?

const 声明的变量不能重新赋值。let 可以重新赋值，var 是旧语法也可以重新赋值，fixed 不是 JavaScript 关键字。const declares a variable that cannot be reassigned. let allows reassignment, var is old syntax that also allows reassignment, and fixed is not a JavaScript keyword.

3. 箭头函数 const add = (a, b) => a + b; 的返回值是什么？3. What does the arrow function const add = (a, b) => a + b; return?

箭头函数如果只有一行表达式且没有大括号，会自动返回该表达式的值。所以 (a, b) => a + b 自动返回 a + b 的结果。If an arrow function has a single expression without curly braces, it automatically returns the value of that expression. So (a, b) => a + b automatically returns the result of a + b.

4. 使用 addEventListener 给按钮添加点击事件的正确写法是？4. What is the correct way to add a click event to a button using addEventListener?

addEventListener("click", handler) 是标准的 DOM 事件监听方法。第一个参数是事件类型，第二个参数是处理函数。addEventListener("click", handler) is the standard DOM event listener method. The first argument is the event type, the second is the handler function.

5. 模板字符串使用什么符号包裹？5. What character is used to wrap template literals?

模板字符串用反引号 ` 包裹，可以在其中用 ${} 嵌入变量和表达式。Template literals are wrapped in backticks `, and you can embed variables and expressions inside using ${}.
