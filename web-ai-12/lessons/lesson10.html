图像提示词技巧Image Prompt Engineering

好的提示词（Prompt）是生成高质量图片的关键。以下是一些技巧： Good prompts are the key to generating high-quality images. Here are some tips:

💡 提示词公式：主体 + 风格 + 细节 + 氛围。例如："一只穿着宇航服的猫，像素风格，太空背景，色彩鲜艳"比"一只猫"能得到好得多的结果。 💡 Prompt formula: Subject + Style + Details + Mood. For example, "a cat in a spacesuit, pixel art style, space background, vibrant colors" gives much better results than just "a cat."

10.3 调用图像生成 API 10.3 Calling the Image Generation API

我们使用 OpenAI 的图像生成 API。请求需要包含 prompt（描述）、size（尺寸）和 n（生成数量），返回结果中包含图片的 URL。 We'll use OpenAI's image generation API. The request includes prompt (description), size (dimensions), and n (number of images). The response contains the image URL.

JavaScript 
async function generateImage(prompt) {
  const response = await fetch("https://api.openai.com/v1/images/generations", {
    method: "POST",
    headers: {
      "Content-Type": "application/json",
      "Authorization": "Bearer YOUR_API_KEY"
    },
    body: JSON.stringify({
      prompt: prompt,
      n: 1,
      size: "512x512"
    })
  });

  const data = await response.json();
  return data.data[0].url;
}

⚠️ 安全提醒：永远不要把 API Key 直接写在前端代码中！在真实项目中，应该通过后端服务器代理请求。这里仅作学习演示。 ⚠️ Security warning: Never put your API key directly in frontend code! In real projects, proxy requests through a backend server. This is for learning purposes only.

响应格式：API 返回的 JSON 中，data 数组包含生成的图片信息，每个元素有一个 url 字段指向生成的图片。 Response format: The returned JSON contains a data array with generated image info. Each element has a url field pointing to the generated image.

10.5 动手实践 10.5 Hands-on Practice

把 HTML、CSS 和 JavaScript 组合在一起，完成你的 AI 图像生成器！下面是完整的 JavaScript 逻辑： Combine the HTML, CSS, and JavaScript to complete your AI image generator! Here's the full JavaScript logic:

JavaScript 
const promptInput = document.getElementById("image-prompt");
const generateBtn = document.getElementById("generate-btn");
const loading = document.getElementById("loading");
const imageResult = document.getElementById("image-result");

generateBtn.addEventListener("click", async () => {
  const prompt = promptInput.value.trim();
  if (!prompt) return alert("Please enter a description!");

  loading.style.display = "block";
  generateBtn.disabled = true;
  imageResult.innerHTML = "";

  try {
    const url = await generateImage(prompt);
    const img = document.createElement("img");
    img.src = url;
    img.alt = prompt;
    imageResult.appendChild(img);
  } catch (error) {
    imageResult.innerHTML = "<p>Error: " + error.message + "</p>";
  } finally {
    loading.style.display = "none";
    generateBtn.disabled = false;
  }
});

💡 提示：生成图片通常需要 5-15 秒，所以一定要给用户展示加载状态！try...catch...finally 可以优雅地处理错误和重置界面。 💡 Tip: Image generation usually takes 5-15 seconds, so always show a loading state! try...catch...finally gracefully handles errors and resets the UI.

课堂测验Lesson Quiz

1. DALL·E 是由哪家公司开发的？1. Which company developed DALL·E?

DALL·E 是由 OpenAI 开发的文生图模型，可以通过 API 调用来生成图片。DALL·E is a text-to-image model developed by OpenAI, accessible via API for image generation.

2. 调用 OpenAI 图像生成 API 时，哪个参数用于指定要生成的图片数量？2. Which parameter specifies the number of images to generate when calling the OpenAI image API?

OpenAI 图像 API 使用参数 n 来指定生成图片的数量，例如 n: 1 表示生成 1 张。The OpenAI image API uses the parameter n to specify how many images to generate, e.g. n: 1 generates one image.

3. 以下哪个是好的图像生成提示词？3. Which of the following is a good image generation prompt?

好的提示词应该包含具体的主体、风格和细节。越具体的描述，AI 生成的图片越符合预期。Good prompts should include a specific subject, style, and details. The more specific the description, the better the AI output matches your expectations.

4. 为什么不应该把 API Key 直接写在前端代码中？4. Why shouldn't you put your API key directly in frontend code?

前端代码对所有用户可见，把 API Key 放在前端意味着任何人都能看到并滥用它，产生高额费用。应该通过后端服务器代理请求。Frontend code is visible to all users. Putting an API key there means anyone can see and misuse it, incurring high costs. You should proxy requests through a backend server.

5. OpenAI 图像 API 返回的结果中，图片信息存储在哪个字段？5. In the OpenAI image API response, where is the image information stored?

OpenAI 图像 API 的返回结果中，生成的图片信息存储在 data 数组中，每个元素包含一个 url 字段。In the OpenAI image API response, generated image info is stored in the data array, with each element containing a url field.
