12.2 准备演示 12.2 Preparing Your Demo

好的演示不仅是展示你做了什么，更是讲一个引人入胜的故事。以下是演示的最佳流程： A great demo isn't just showing what you built — it's telling a compelling story. Here's the best demo flow:

1️⃣

项目介绍（30 秒）Project Intro (30 sec)

用一句话说明你的网站是做什么的、解决了什么问题。In one sentence, explain what your website does and what problem it solves.

2️⃣

现场演示（2 分钟）Live Demo (2 min)

展示核心功能：AI 聊天、图片生成等。提前准备好输入内容，确保演示流畅。Show core features: AI chat, image generation, etc. Prepare inputs in advance to ensure a smooth demo.

3️⃣

技术亮点（1 分钟）Technical Highlights (1 min)

简要介绍你用了哪些技术、API 和部署方式。Briefly describe the technologies, APIs, and deployment methods you used.

4️⃣

问答环节（1 分钟）Q&A (1 min)

回答观众的问题。不知道答案时诚实说"我还在学习"也很好。Answer audience questions. It's perfectly fine to say "I'm still learning about that" when unsure.

💡 演示技巧：一定要有备用方案！提前录制一段演示视频或截图，万一现场网络出问题也能展示你的成果。 💡 Demo tip: Always have a backup plan! Record a demo video or take screenshots in advance, so you can still showcase your work if the network fails during the live demo.

课堂测验Lesson Quiz

1. 在项目演示中，以下哪项是最重要的准备？1. What is the most important preparation for a project demo?

成功的演示关键在于提前准备好展示内容，确保演示流畅，并准备好备用方案以应对突发情况。The key to a successful demo is preparing your content in advance, ensuring smooth delivery, and having a backup plan for unexpected issues.

2. GitHub 仓库中的 README.md 文件应该包含哪些内容？2. What should a GitHub repository's README.md file contain?

一个好的 README 应该包含项目简介、功能列表、使用的技术、在线演示链接和截图，让其他人快速了解你的项目。A good README should include a project overview, feature list, technologies used, live demo link, and screenshots so others can quickly understand your project.

3. 在这门课程中，我们学习了哪三种核心 Web 技术？3. Which three core web technologies did we learn in this course?

本课程的三大核心技术是 HTML（结构）、CSS（样式）和 JavaScript（交互），它们是所有网页开发的基础。The three core technologies in this course are HTML (structure), CSS (styling), and JavaScript (interactivity) — the foundation of all web development.

4. 以下哪个是学完本课程后推荐的进阶方向？4. Which is a recommended next step after completing this course?

掌握 HTML/CSS/JS 基础后，React（前端框架）和 Node.js（后端）是最受欢迎的进阶方向，能帮你构建更复杂的全栈应用。After mastering HTML/CSS/JS basics, React (frontend framework) and Node.js (backend) are the most popular next steps, enabling you to build more complex full-stack applications.

5. 把你的 AI 网站项目用于竞赛或申请时，最应该强调什么？5. When using your AI website project for competitions or applications, what should you emphasize most?

评审官和招生官最看重的是你的学习过程、解决问题的能力和独立思考能力，而不是代码量或使用的语言数量。Reviewers and admissions officers value your learning process, problem-solving ability, and independent thinking most — not code volume or number of languages used.
