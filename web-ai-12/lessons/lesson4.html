4.2 Flexbox 基础4.2 Flexbox Basics

Flexbox（弹性盒布局）是 CSS 中最强大的布局工具之一。只需要在父元素上设置 display: flex，子元素就会自动排列。Flexbox (Flexible Box Layout) is one of the most powerful layout tools in CSS. Simply set display: flex on the parent element, and child elements will arrange themselves automatically.

CSS 
.container {
  display: flex;            /* 开启 Flexbox */
  flex-direction: row;      /* 水平排列（默认） */
  justify-content: center;  /* 主轴居中 */
  align-items: center;      /* 交叉轴居中 */
  gap: 16px;                /* 子元素间距 */
}

主轴 vs 交叉轴：flex-direction: row 时，主轴是水平方向，交叉轴是垂直方向。flex-direction: column 时正好相反。Main Axis vs Cross Axis: With flex-direction: row, the main axis is horizontal and the cross axis is vertical. With flex-direction: column, it's the opposite.

flex-direction 的四个值Four Values of flex-direction

row（行）row

从左到右水平排列（默认值）。Arrange horizontally from left to right (default).

row-reverse（反向行）row-reverse

从右到左水平排列。Arrange horizontally from right to left.

column（列）column

从上到下垂直排列。Arrange vertically from top to bottom.

column-reverse（反向列）column-reverse

从下到上垂直排列。Arrange vertically from bottom to top.

justify-content 常用值Common justify-content Values

CSS 
/* 主轴对齐方式 */
justify-content: flex-start;    /* 靠起点对齐 */
justify-content: flex-end;      /* 靠终点对齐 */
justify-content: center;        /* 居中 */
justify-content: space-between; /* 两端对齐，中间均匀分布 */
justify-content: space-around;  /* 每个元素两侧等距 */
justify-content: space-evenly;  /* 所有间距完全相等 */

4.3 实用 Flexbox 布局4.3 Practical Flexbox Layouts

导航栏布局Navigation Bar Layout

用 Flexbox 可以轻松创建一个左边是 Logo、右边是链接的导航栏：With Flexbox, you can easily create a nav bar with a logo on the left and links on the right:

HTML 
<nav class="navbar">
  <a href="#" class="logo">My Site</a>
  <div class="nav-links">
    <a href="#">Home</a>
    <a href="#">About</a>
    <a href="#">Contact</a>
  </div>
</nav>
CSS 
.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 12px 24px;
  background: #1a1a2e;
  color: white;
}

.nav-links {
  display: flex;
  gap: 20px;
}

.nav-links a {
  color: white;
  text-decoration: none;
}

卡片网格布局Card Grid Layout

使用 flex-wrap: wrap 可以让卡片自动换行，配合 flex 属性控制每个卡片的宽度：Use flex-wrap: wrap to let cards automatically wrap to new lines. Combine with the flex property to control each card's width:

CSS 
.card-grid {
  display: flex;
  flex-wrap: wrap;
  gap: 20px;
}

.card {
  flex: 1 1 300px;  /* 最小宽度300px，自动填充 */
  padding: 20px;
  border-radius: 12px;
  background: #f8f9fa;
  box-shadow: 0 2px 8px rgba(0, 0, 0, 0.1);
}

flex 简写：flex: 1 1 300px 表示 flex-grow: 1（可以伸展）、flex-shrink: 1（可以收缩）、flex-basis: 300px（基准宽度 300px）。flex shorthand: flex: 1 1 300px means flex-grow: 1 (can stretch), flex-shrink: 1 (can shrink), flex-basis: 300px (base width of 300px).

完美居中Perfect Centering

Flexbox 让居中变得前所未有地简单：Flexbox makes centering easier than ever before:

CSS 
.center-everything {
  display: flex;
  justify-content: center;
  align-items: center;
  min-height: 100vh;  /* 至少占满整个视口高度 */
}

4.4 响应式设计4.4 Responsive Design

响应式设计让你的网页在手机、平板和电脑上都能完美显示。关键工具是 媒体查询（Media Queries）。Responsive design ensures your page looks great on phones, tablets, and desktops. The key tool is Media Queries.

别忘了 viewport meta 标签！在 HTML <head> 中必须加上：<meta name="viewport" content="width=device-width, initial-scale=1.0">，否则手机上不会正确缩放。Don't forget the viewport meta tag! You must add this in the HTML <head>: <meta name="viewport" content="width=device-width, initial-scale=1.0">, otherwise the page won't scale correctly on mobile.

媒体查询语法Media Query Syntax

CSS 
/* 默认样式（适用于所有屏幕） */
.container {
  display: flex;
  gap: 20px;
}

/* 平板和更小屏幕（宽度 <= 768px） */
@media (max-width: 768px) {
  .container {
    flex-direction: column;  /* 改为垂直排列 */
  }
}

/* 手机屏幕（宽度 <= 480px） */
@media (max-width: 480px) {
  .container {
    padding: 10px;
    gap: 10px;
  }
}

常用断点Common Breakpoints

手机Mobile

max-width: 480px

平板Tablet

max-width: 768px

小型笔记本Small Laptop

max-width: 1024px

大屏桌面Large Desktop

min-width: 1200px

移动优先设计Mobile-First Design

最佳实践是先写手机端样式，再用 min-width 媒体查询添加大屏样式：Best practice is to write mobile styles first, then use min-width media queries for larger screens:

CSS 
/* 移动优先：默认为手机样式 */
.grid {
  display: flex;
  flex-direction: column;
  gap: 12px;
}

/* 当屏幕 >= 768px 时切换为水平布局 */
@media (min-width: 768px) {
  .grid {
    flex-direction: row;
    gap: 24px;
  }
}

4.5 动手实践：响应式作品集页面4.5 Hands-On: Responsive Portfolio Page

现在让我们综合运用 Flexbox 和媒体查询，构建一个完整的响应式作品集页面！Let's combine Flexbox and media queries to build a complete responsive portfolio page!

HTML 
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My Portfolio</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <nav class="navbar">
    <span class="logo">Portfolio</span>
    <div class="nav-links">
      <a href="#projects">Projects</a>
      <a href="#about">About</a>
      <a href="#contact">Contact</a>
    </div>
  </nav>

  <section class="hero-section">
    <h1>Hi, I'm Alex</h1>
    <p>Young Developer & AI Enthusiast</p>
  </section>

  <section id="projects" class="projects">
    <h2>My Projects</h2>
    <div class="project-grid">
      <div class="project-card">
        <h3>AI Chatbot</h3>
        <p>A chatbot powered by OpenAI</p>
      </div>
      <div class="project-card">
        <h3>Weather App</h3>
        <p>Real-time weather using API</p>
      </div>
      <div class="project-card">
        <h3>Todo List</h3>
        <p>A task manager with localStorage</p>
      </div>
    </div>
  </section>
</body>
</html>
CSS 
* { margin: 0; padding: 0; box-sizing: border-box; }
body { font-family: 'Segoe UI', sans-serif; }

/* 导航栏 */
.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 16px 24px;
  background: #0f0f23;
  color: white;
}
.nav-links { display: flex; gap: 20px; }
.nav-links a { color: #a5b4fc; text-decoration: none; }

/* Hero 区域 */
.hero-section {
  display: flex;
  flex-direction: column;
  justify-content: center;
  align-items: center;
  min-height: 60vh;
  background: linear-gradient(135deg, #667eea, #764ba2);
  color: white;
  text-align: center;
}

/* 项目卡片网格 */
.projects { padding: 40px 24px; }
.project-grid {
  display: flex;
  flex-wrap: wrap;
  gap: 20px;
  margin-top: 20px;
}
.project-card {
  flex: 1 1 280px;
  padding: 24px;
  border-radius: 12px;
  background: #f1f5f9;
  border: 1px solid #e2e8f0;
}

/* 响应式调整 */
@media (max-width: 768px) {
  .navbar { flex-direction: column; gap: 12px; }
  .project-grid { flex-direction: column; }
  .hero-section { min-height: 40vh; padding: 20px; }
}

4.6 挑战任务4.6 Challenges

挑战 A：三栏布局Challenge A: Three-Column Layout

创建一个三栏布局（侧边栏 + 主内容 + 侧边栏），在手机上自动变成单栏堆叠。使用 flex-wrap 和媒体查询实现。Create a three-column layout (sidebar + main content + sidebar) that automatically stacks to a single column on mobile. Use flex-wrap and media queries.

挑战 B：响应式导航Challenge B: Responsive Navigation

设计一个导航栏，在大屏幕上水平显示链接，在小屏幕上变为垂直菜单（汉堡按钮概念）。只用 CSS 实现布局切换。Design a nav bar that shows links horizontally on large screens and switches to a vertical menu on small screens (hamburger concept). Use CSS only for the layout switch.

提示：挑战 A 中，给侧边栏设置 flex: 0 0 200px（固定宽度），主内容设置 flex: 1（自动填充）。在 768px 断点将所有元素设为 flex: 1 1 100% 即可堆叠。Hint: For Challenge A, set sidebars to flex: 0 0 200px (fixed width) and main content to flex: 1 (fill remaining space). At the 768px breakpoint, set all elements to flex: 1 1 100% to stack them.

课堂测验Lesson Quiz

1. 要让一个容器的子元素水平排列，需要设置哪个 CSS 属性？1. Which CSS property makes a container's children arrange horizontally?

display: flex 开启 Flexbox 弹性布局，默认 flex-direction: row 就是水平排列。display: flex activates Flexbox layout. The default flex-direction: row arranges children horizontally.

2. justify-content: space-between 的效果是什么？2. What does justify-content: space-between do?

space-between 让第一个和最后一个元素分别贴住容器的两端，其余元素之间的间距完全相等。space-between pushes the first and last elements to the container edges, with equal spacing between all remaining elements.

3. 下列哪个是正确的媒体查询语法？3. Which of the following is correct media query syntax?

@media (max-width: 768px) 是标准的 CSS 媒体查询语法，当屏幕宽度小于等于 768px 时生效。@media (max-width: 768px) is the standard CSS media query syntax, which applies when the screen width is 768px or less.

4. "移动优先"设计意味着什么？4. What does "mobile-first" design mean?

移动优先意味着默认样式为手机设计，然后用 min-width 媒体查询为更大的屏幕添加样式。Mobile-first means default styles are designed for mobile, then min-width media queries add styles for progressively larger screens.

5. flex: 1 1 300px 中的 300px 代表什么？5. What does 300px represent in flex: 1 1 300px?

flex 简写的第三个值是 flex-basis，即元素在分配剩余空间之前的初始大小（基准宽度）。The third value in the flex shorthand is flex-basis, which is the element's initial size before remaining space is distributed (base width).
