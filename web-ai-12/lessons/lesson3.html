3.2 什么是 CSS？ 3.2 What Is CSS?

CSS 是 Cascading Style Sheets（层叠样式表）的缩写。如果说 HTML 是网页的骨架，那么 CSS 就是皮肤、衣服和化妆——它决定了网页看起来是什么样子。 CSS stands for Cascading Style Sheets. If HTML is the skeleton, CSS is the skin, clothes, and makeup — it determines how a page looks.

三种添加 CSS 的方式Three Ways to Add CSS

① 行内样式① Inline Styles

HTML 
<p style="color: red;">
  红色文字
</p>

直接写在标签的 style 属性中。简单但不推荐大量使用。Written directly in the tag's style attribute. Simple but not recommended for heavy use.

② 内部样式表② Internal Stylesheet

HTML 
<head>
  <style>
    p { color: blue; }
  </style>
</head>

写在 <head> 的 <style> 标签中。适合单页面。Placed in a <style> tag inside <head>. Good for single pages.

③ 外部样式表 ✅③ External Stylesheet ✅

HTML 
<head>
  <link rel="stylesheet"
    href="style.css">
</head>

独立的 .css 文件，多个页面可以共享。推荐使用！A separate .css file shared across multiple pages. Recommended!

CSS 选择器CSS Selectors

选择器用来"选中"你要美化的 HTML 元素。最常用的三种选择器： Selectors "target" the HTML elements you want to style. The three most common selectors:

CSS 
/* 元素选择器 - 选中所有 p 标签 */
p {
  color: #333;
}

/* 类选择器 - 选中 class="highlight" 的元素 */
.highlight {
  background-color: yellow;
}

/* ID 选择器 - 选中 id="header" 的元素 */
#header {
  font-size: 24px;
}

💡 优先级：行内样式 > ID 选择器 > 类选择器 > 元素选择器。一般推荐多用类选择器，它灵活又可复用。 💡 Specificity: Inline styles > ID selectors > Class selectors > Element selectors. In practice, class selectors are most recommended — they're flexible and reusable.

3.3 颜色与字体 3.3 Colors & Fonts

颜色Colors

CSS 支持多种颜色格式。以下是最常用的三种： CSS supports multiple color formats. Here are the three most common:

CSS 
/* 颜色名称 */
h1 { color: tomato; }

/* 十六进制 (Hex) */
h2 { color: #3498db; }

/* RGB 函数 */
h3 { color: rgb(46, 204, 113); }

/* 背景颜色 */
body {
  background-color: #f5f5f5;
  color: #333333;
}

💡 小技巧：十六进制颜色 #RRGGBB 中，每两位分别代表红(R)、绿(G)、蓝(B)的强度，从 00（最暗）到 FF（最亮）。例如 #FF0000 是纯红色。 💡 Tip: In hex color #RRGGBB, each pair represents Red, Green, Blue intensity from 00 (darkest) to FF (brightest). For example, #FF0000 is pure red.

字体Fonts

字体能极大地影响网页的风格和可读性。以下是最常用的字体属性： Fonts dramatically affect a page's style and readability. Here are the most common font properties:

CSS 
body {
  /* 字体族：优先使用第一个，找不到就用下一个 */
  font-family: "Noto Sans SC", Arial, sans-serif;

  /* 字体大小 */
  font-size: 16px;

  /* 行高：推荐 1.5-1.8 倍，提高可读性 */
  line-height: 1.6;
}

h1 {
  /* 字体粗细：100-900，或 normal/bold */
  font-weight: 700;
  font-size: 2rem;
}

🌐 Google Fonts：想用漂亮的免费字体？访问 fonts.google.com，选择字体后，把提供的 <link> 标签复制到你的 <head> 中即可使用！ 🌐 Google Fonts: Want beautiful free fonts? Visit fonts.google.com, pick a font, and copy the provided <link> tag into your <head>!

3.4 盒模型 3.4 The Box Model

在 CSS 中，每个 HTML 元素都被看作一个"盒子"。理解盒模型是掌握 CSS 布局的关键。盒模型从外到内分为四层： In CSS, every HTML element is treated as a "box." Understanding the box model is key to mastering CSS layout. From outside to inside, there are four layers:

1️⃣

Margin（外边距）Margin (Outer Spacing)

盒子与其他元素之间的距离。是透明的，不会显示背景色。用来控制元素之间的间隔。Space between the box and other elements. It's transparent — no background color. Controls spacing between elements.

2️⃣

Border（边框）Border

围绕内容和内边距的线条。可以设置宽度、样式（实线、虚线等）和颜色。A line around the content and padding. You can set its width, style (solid, dashed, etc.), and color.

3️⃣

Padding（内边距）Padding (Inner Spacing)

内容与边框之间的空间。会显示元素的背景色。让内容不会紧贴边框。Space between content and border. Shows the element's background color. Keeps content from touching the border.

4️⃣

Content（内容）Content

实际的文字、图片等内容区域。通过 width 和 height 设置大小。The actual text, images, etc. Set its size with width and height.

CSS 
.my-box {
  /* 内容大小 */
  width: 300px;
  height: 200px;

  /* 内边距：内容与边框的距离 */
  padding: 20px;

  /* 边框：2px 粗的实线，颜色为深灰 */
  border: 2px solid #333;

  /* 外边距：与其他元素的距离 */
  margin: 16px;

  /* 推荐：让 width/height 包含 padding 和 border */
  box-sizing: border-box;
}

⚠️ 重要：默认情况下，width 只包含内容区域，加上 padding 和 border 后，元素的实际宽度会更大。建议在全局添加 box-sizing: border-box;，这样 width 就会包含 padding 和 border。 ⚠️ Important: By default, width only includes the content area — adding padding and border makes the element wider. Add box-sizing: border-box; globally so width includes padding and border.

3.5 动手实践 3.5 Hands-on Practice

让我们为第 2 课创建的个人主页添加 CSS 样式！创建一个 style.css 文件，然后在 HTML 中通过 <link> 引入。 Let's add CSS styles to the personal page from Lesson 2! Create a style.css file and link it in your HTML with <link>.

CSS 
/* 全局重置 */
* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

/* 页面基础样式 */
body {
  font-family: "Noto Sans SC", Arial, sans-serif;
  font-size: 16px;
  line-height: 1.6;
  color: #333;
  background-color: #f9f9f9;
}

/* 导航栏 */
nav {
  background-color: #2c3e50;
  padding: 12px 24px;
}

nav a {
  color: white;
  text-decoration: none;
  margin-right: 16px;
}

nav a:hover {
  color: #3498db;
}

/* 标题 */
h1 {
  color: #2c3e50;
  font-size: 2rem;
  margin: 24px 0 16px;
}

h2 {
  color: #34495e;
  font-size: 1.5rem;
  margin: 20px 0 12px;
  border-bottom: 2px solid #3498db;
  padding-bottom: 8px;
}

/* 段落 */
p {
  margin-bottom: 12px;
  max-width: 700px;
}

/* 表单样式 */
form {
  max-width: 400px;
  padding: 20px;
  background: white;
  border: 1px solid #ddd;
  border-radius: 8px;
}

input, textarea {
  width: 100%;
  padding: 8px 12px;
  margin: 8px 0 16px;
  border: 1px solid #ccc;
  border-radius: 4px;
  font-size: 14px;
}

button {
  background-color: #3498db;
  color: white;
  padding: 10px 24px;
  border: none;
  border-radius: 4px;
  cursor: pointer;
  font-size: 16px;
}

button:hover {
  background-color: #2980b9;
}

💡 试一试：把以上 CSS 保存为 style.css，并在你的 HTML <head> 中加入 <link rel="stylesheet" href="style.css">。刷新浏览器，看看变化有多大！ 💡 Try it: Save the CSS above as style.css and add <link rel="stylesheet" href="style.css"> in your HTML <head>. Refresh the browser and see the difference!

CSS 
/* 挑战 B 参考：卡片组件 */

课堂测验Lesson Quiz

1. CSS 的全称是什么？1. What does CSS stand for?

CSS 全称为 Cascading Style Sheets（层叠样式表），用于控制网页的视觉呈现。CSS stands for Cascading Style Sheets, used to control the visual presentation of webpages.

2. 哪种选择器用于选中 class="intro" 的元素？2. Which selector targets elements with class="intro"?

类选择器以点号 . 开头，所以 .intro 选中所有 class="intro" 的元素。#intro 是 ID 选择器。Class selectors start with a dot ., so .intro targets all elements with class="intro". #intro is an ID selector.

3. 盒模型中，padding 是哪个部分？3. In the box model, what is padding?

Padding（内边距）是内容区域与边框之间的空间。Margin（外边距）才是元素与其他元素之间的距离。Padding is the space between the content area and the border. Margin is the space between the element and other elements.

4. 推荐使用哪种方式添加 CSS？4. Which is the recommended way to add CSS?

外部样式表是最佳实践：代码组织清晰、可以跨页面复用、易于维护。External stylesheets are best practice: clean code organization, reusable across pages, and easy to maintain.

5. #FF0000 代表什么颜色？5. What color does #FF0000 represent?

#FF0000 中，红色(FF)为最大值，绿色(00)和蓝色(00)为零，所以是纯红色。In #FF0000, red (FF) is at maximum while green (00) and blue (00) are zero — so it's pure red.
