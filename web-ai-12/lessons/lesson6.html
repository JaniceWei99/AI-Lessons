6.2 DOM 是什么？6.2 What Is the DOM?

DOM（Document Object Model，文档对象模型）是浏览器把 HTML 文档解析后生成的树形结构。每一个 HTML 标签都变成了树上的一个"节点"，JavaScript 可以通过 DOM 来读取和修改这些节点。The DOM (Document Object Model) is a tree structure generated when the browser parses an HTML document. Each HTML tag becomes a "node" on the tree, and JavaScript can read and modify these nodes through the DOM.

想象一下：DOM 就像一棵倒过来的大树。<html> 是树根，<head> 和 <body> 是两大树枝，每个标签都是更小的树枝或树叶。JavaScript 就是园丁，可以修剪、添加或移除树上的任何部分。Think of it this way: The DOM is like an upside-down tree. <html> is the root, <head> and <body> are two main branches, and each tag is a smaller branch or leaf. JavaScript is the gardener who can trim, add, or remove any part of the tree.

查找元素的三种方式Three Ways to Find Elements

JavaScript 
// 1. 通过 ID 查找（返回单个元素）
const title = document.getElementById("main-title");

// 2. 通过 CSS 选择器查找第一个匹配项
const firstCard = document.querySelector(".card");

// 3. 通过 CSS 选择器查找所有匹配项（返回列表）
const allCards = document.querySelectorAll(".card");

// 遍历所有匹配的元素
allCards.forEach(card => {
  console.log(card.textContent);
});

getElementByIdgetElementById

通过 id 属性查找，返回单个元素。最快的查找方式。Finds by id attribute, returns a single element. The fastest lookup method.

querySelectorquerySelector

用 CSS 选择器查找，返回第一个匹配的元素。非常灵活。Uses CSS selectors to find the first matching element. Very flexible.

querySelectorAllquerySelectorAll

用 CSS 选择器查找所有匹配的元素，返回一个 NodeList 列表。Uses CSS selectors to find all matching elements, returns a NodeList.

6.3 修改 DOM6.3 Modifying the DOM

修改文本内容Modifying Text Content

JavaScript 
const heading = document.getElementById("title");

// textContent — 只设置纯文本
heading.textContent = "New Title";

// innerHTML — 可以包含 HTML 标签（注意安全！）
heading.innerHTML = "New <em>Title</em>";

安全警告：使用 innerHTML 时要小心！如果内容来自用户输入，可能导致 XSS（跨站脚本）攻击。对于纯文本内容，优先使用 textContent。Security warning: Be careful with innerHTML! If the content comes from user input, it can lead to XSS (Cross-Site Scripting) attacks. For plain text, prefer textContent.

修改样式Modifying Styles

JavaScript 
const box = document.querySelector(".box");

// 直接修改 style 属性（注意用驼峰命名法）
box.style.backgroundColor = "#4f46e5";
box.style.color = "white";
box.style.borderRadius = "12px";
box.style.padding = "20px";

操作 CSS 类名Manipulating CSS Classes

比直接修改 style 更好的方式是操作类名，让样式保持在 CSS 文件中：A better approach than modifying styles directly is to manipulate class names, keeping styles in your CSS file:

JavaScript 
const element = document.querySelector(".card");

// 添加类名
element.classList.add("active");

// 移除类名
element.classList.remove("hidden");

// 切换类名（有则删、无则加）
element.classList.toggle("dark-mode");

// 检查是否包含某个类名
if (element.classList.contains("active")) {
  console.log("Element is active!");
}

6.4 表单交互6.4 Form Interactions

读取输入框的值Reading Input Values

HTML 
<input type="text" id="nameInput" placeholder="Enter your name">
<button id="submitBtn">Submit</button>
<p id="greeting"></p>
JavaScript 
const input = document.getElementById("nameInput");
const btn = document.getElementById("submitBtn");
const greeting = document.getElementById("greeting");

btn.addEventListener("click", () => {
  const name = input.value.trim();  // .trim() 去掉首尾空格
  if (name) {
    greeting.textContent = `Hello, ${name}!`;
  } else {
    greeting.textContent = "Please enter your name.";
  }
});

表单验证与阻止默认行为Form Validation & Preventing Default

表单的默认行为是提交时刷新页面。使用 event.preventDefault() 可以阻止这个行为，让我们用 JS 来处理数据。A form's default behavior is to refresh the page on submit. Use event.preventDefault() to stop this, allowing us to handle data with JS.

HTML 
<form id="signupForm">
  <input type="text" id="username" placeholder="Username" required>
  <input type="email" id="email" placeholder="Email" required>
  <input type="password" id="password" placeholder="Password" required>
  <button type="submit">Sign Up</button>
  <p id="formMessage"></p>
</form>
JavaScript 
const form = document.getElementById("signupForm");
const message = document.getElementById("formMessage");

form.addEventListener("submit", (event) => {
  event.preventDefault();  // 阻止页面刷新

  const username = document.getElementById("username").value.trim();
  const email = document.getElementById("email").value.trim();
  const password = document.getElementById("password").value;

  // 简单验证
  if (password.length < 6) {
    message.textContent = "Password must be at least 6 characters!";
    message.style.color = "red";
    return;
  }

  message.textContent = `Welcome, ${username}! Account created.`;
  message.style.color = "green";
  form.reset();  // 清空表单
});

event 对象：事件处理函数可以接收一个 event 参数（通常简写为 e），它包含了事件的所有信息。event.preventDefault() 是最常用的方法之一。The event object: Event handler functions can receive an event parameter (often abbreviated as e), which contains all event information. event.preventDefault() is one of the most commonly used methods.

6.5 动手实践：待办事项列表6.5 Hands-On: To-Do List App

让我们综合所有知识，构建一个完整的待办事项应用！可以添加任务、标记完成和删除任务。Let's combine everything we've learned to build a complete to-do list app! You can add tasks, mark them complete, and delete them.

HTML 
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Todo List</title>
  <style>
    * { margin: 0; padding: 0; box-sizing: border-box; }
    body {
      font-family: sans-serif;
      background: #f0f4f8;
      display: flex;
      justify-content: center;
      padding: 40px 20px;
    }
    .app {
      width: 100%;
      max-width: 500px;
      background: white;
      border-radius: 16px;
      padding: 30px;
      box-shadow: 0 4px 20px rgba(0,0,0,0.1);
    }
    h1 { text-align: center; margin-bottom: 20px; }
    .input-row {
      display: flex;
      gap: 8px;
      margin-bottom: 20px;
    }
    .input-row input {
      flex: 1;
      padding: 10px 14px;
      border: 2px solid #e2e8f0;
      border-radius: 8px;
      font-size: 16px;
    }
    .input-row button {
      padding: 10px 20px;
      background: #4f46e5;
      color: white;
      border: none;
      border-radius: 8px;
      cursor: pointer;
      font-size: 16px;
    }
    ul { list-style: none; }
    li {
      display: flex;
      align-items: center;
      justify-content: space-between;
      padding: 12px;
      border-bottom: 1px solid #e2e8f0;
    }
    li.done span { text-decoration: line-through; color: #94a3b8; }
    .delete-btn {
      background: #ef4444;
      color: white;
      border: none;
      padding: 4px 12px;
      border-radius: 6px;
      cursor: pointer;
    }
  </style>
</head>
<body>
  <div class="app">
    <h1>My Todo List</h1>
    <div class="input-row">
      <input type="text" id="todoInput" placeholder="Add a task...">
      <button id="addBtn">Add</button>
    </div>
    <ul id="todoList"></ul>
  </div>

  <script>
    const input = document.getElementById("todoInput");
    const addBtn = document.getElementById("addBtn");
    const list = document.getElementById("todoList");

    function addTask() {
      const text = input.value.trim();
      if (!text) return;

      // 创建列表项
      const li = document.createElement("li");

      // 任务文本（点击切换完成状态）
      const span = document.createElement("span");
      span.textContent = text;
      span.addEventListener("click", () => {
        li.classList.toggle("done");
      });

      // 删除按钮
      const delBtn = document.createElement("button");
      delBtn.textContent = "Delete";
      delBtn.classList.add("delete-btn");
      delBtn.addEventListener("click", () => {
        li.remove();
      });

      li.appendChild(span);
      li.appendChild(delBtn);
      list.appendChild(li);

      input.value = "";
      input.focus();
    }

    addBtn.addEventListener("click", addTask);

    // 按回车键也能添加
    input.addEventListener("keydown", (e) => {
      if (e.key === "Enter") addTask();
    });
  </script>
</body>
</html>

挑战 B：本地存储Challenge B: Local Storage

使用 localStorage 让数据持久化。页面刷新后，之前添加的任务不会丢失。提示：用 JSON.stringify() 存储，JSON.parse() 读取。Use localStorage to persist data. After refreshing the page, previously added tasks remain. Hint: Use JSON.stringify() to save and JSON.parse() to load.

localStorage 入门：localStorage Primer:

JavaScript 
// 保存数据
const tasks = ["Buy milk", "Study JS", "Build website"];
localStorage.setItem("todos", JSON.stringify(tasks));

// 读取数据
const saved = JSON.parse(localStorage.getItem("todos"));
console.log(saved);  // ["Buy milk", "Study JS", "Build website"]

// 删除数据
localStorage.removeItem("todos");

课堂测验Lesson Quiz

1. DOM 是什么？1. What is the DOM?

DOM（Document Object Model）是浏览器将 HTML 文档解析后生成的树形结构，JavaScript 可以通过它来访问和操作网页元素。The DOM (Document Object Model) is a tree structure generated when the browser parses an HTML document. JavaScript can access and manipulate page elements through it.

2. 以下哪个方法可以查找页面中所有 class 为 "item" 的元素？2. Which method finds all elements with class "item" on the page?

querySelectorAll 返回所有匹配的元素列表。querySelector 只返回第一个匹配项，getElementById 只能通过 ID 查找，findAll 不存在。querySelectorAll returns all matching elements. querySelector returns only the first match, getElementById only finds by ID, and findAll does not exist.

3. classList.toggle("active") 的作用是什么？3. What does classList.toggle("active") do?

toggle 的含义是"切换"——如果元素有该类名就移除，没有就添加。这在实现开关效果时非常实用。toggle means "switch" — if the element has the class, it removes it; if not, it adds it. This is very useful for implementing on/off effects.

4. event.preventDefault() 在表单中的作用是什么？4. What does event.preventDefault() do in a form?

event.preventDefault() 阻止事件的默认行为。对于表单提交事件，默认行为是刷新页面并发送数据，调用此方法后可以用 JavaScript 自行处理数据。event.preventDefault() stops the event's default behavior. For form submit events, the default is to refresh the page and send data. Calling this method lets JavaScript handle the data instead.

5. 如何将一个数组保存到 localStorage？5. How do you save an array to localStorage?

localStorage 只能存储字符串，所以要先用 JSON.stringify() 将数组转换为 JSON 字符串，再存储。读取时用 JSON.parse() 转回数组。localStorage can only store strings, so you must first convert the array to a JSON string using JSON.stringify(). When reading, use JSON.parse() to convert it back to an array.
