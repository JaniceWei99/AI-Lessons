# 第1课：HTML 基础（上）— 你的第一个网页

## 教学目标

- 理解 HTML 是什么，以及它在网页中的角色（网页的"骨架"）
- 掌握 HTML 基本结构：`<!DOCTYPE html>`、`<html>`、`<head>`、`<body>`
- 学会使用标题标签 `<h1>` ~ `<h6>`、段落标签 `<p>`、图片标签 `<img>`
- 理解标签、元素、属性的概念
- 独立完成一个个人简介页面

## 课时建议

**总时长：90 分钟**

## 教学流程

### 一、导入环节（10 分钟）

> **教师讲稿：**
>
> 同学们好！欢迎来到 AI 网站工坊的第一节课。今天我们要做一件很酷的事——创建你人生中的第一个网页！
>
> 在开始之前，我想问大家一个问题：你们每天都在浏览网页，但有没有想过，网页是怎么做出来的？
>
> 其实，每一个网页背后都有三种"语言"在合作：HTML、CSS 和 JavaScript。我喜欢用"人"来比喻——HTML 是骨架，决定了网页上有什么内容；CSS 是皮肤和衣服，让网页变得好看；JavaScript 是大脑，让网页能动、能思考。今天我们从骨架开始！

**教学提示：** 打开任意网站（如百度），右键 → 查看网页源代码，让学生直观感受 HTML 代码。

### 二、核心概念讲解（25 分钟）

#### 2.1 什么是标签和元素（8 分钟）

> HTML 的全称是 HyperText Markup Language——超文本标记语言。它用"标签"来告诉浏览器：这是标题、这是段落、这是图片。
>
> 标签长这样：`<h1>你好世界</h1>`。注意，大多数标签是成对出现的——有开始标签 `<h1>` 和结束标签 `</h1>`，中间夹着的是内容。一个完整的开始标签 + 内容 + 结束标签，我们叫它一个"元素"。
>
> 还有一种特殊的"自闭合标签"，比如图片 `<img>`，它不需要结束标签，因为它不包裹文字内容。

**教学提示：** 在黑板或投影上画出标签结构图——开始标签、内容、结束标签，用不同颜色标注。

#### 2.2 HTML 基本结构（8 分钟）

> 每个 HTML 文件都有一个固定的"骨架"结构，就像写信有格式一样：

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <title>我的第一个网页</title>
</head>
<body>
  <!-- 网页的可见内容写在这里 -->
</body>
</html>
```

> `<!DOCTYPE html>` 告诉浏览器：这是 HTML5 文档。`<head>` 里放的是"看不见"的信息，比如网页标题和编码方式。`<body>` 里放的才是用户看得到的内容。

**教学提示：** 现场演示 —— 让学生打开浏览器开发者工具（F12），切到 Elements 面板，展开看真实网页的 HTML 结构。

#### 2.3 标题与段落（5 分钟）

> HTML 有六级标题，从 `<h1>` 到 `<h6>`，`<h1>` 最大最重要，`<h6>` 最小。一个页面通常只有一个 `<h1>`，就像文章只有一个大标题。段落用 `<p>` 标签，浏览器会自动在段落之间加空行。

**现场演示：** 依次输入 `<h1>` 到 `<h6>` 和 `<p>` 标签，保存后刷新浏览器，让学生观察大小变化。

#### 2.4 属性与图片标签（4 分钟）

> 标签可以携带额外信息，这些信息叫"属性"。比如图片标签：
>
> `<img src="photo.jpg" alt="我的照片">`
>
> `src` 属性告诉浏览器图片在哪里，`alt` 属性是替代文本——当图片加载失败时显示的文字，也帮助视障用户理解图片内容。`alt` 属性非常重要，永远不要忘记写它！

### 三、编码指导 — 动手实践（40 分钟）

#### 3.1 教师带码（15 分钟）

> 现在我们一起来创建一个个人简介页面。请跟着我一步一步做：

**教学提示：** 打开 VS Code（或在线编辑器），新建 `index.html`，逐行输入并讲解。每写几行就保存、刷新浏览器，让学生看到实时效果。

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <title>关于我</title>
</head>
<body>
  <h1>大家好，我是小明</h1>
  <h2>关于我</h2>
  <p>我今年13岁，喜欢编程和打篮球。</p>
  <h2>我的爱好</h2>
  <p>我最喜欢的科目是数学和计算机科学。</p>
  <img src="https://via.placeholder.com/300x200" alt="我的照片">
</body>
</html>
```

#### 3.2 学生自主编码（25 分钟）

> 现在轮到你们了！请创建你自己的个人简介页面，要求包含：
> - 至少使用 `<h1>` 和一个 `<h2>`
> - 至少写两个段落 `<p>` 介绍自己
> - 插入至少一张图片（可以用占位图片链接）
> - **挑战：** 试试用 `<h3>` 添加更多小标题，多写几个段落

**教学提示：** 巡视教室，关注以下常见问题。鼓励学生互相交流。

### 四、常见问题与排错（5 分钟）

| 常见错误 | 原因 | 解决方法 |
|---------|------|---------|
| 页面显示乱码 | 缺少 `<meta charset="UTF-8">` | 在 `<head>` 中添加字符编码声明 |
| 标签没生效 | 忘记写结束标签 | 检查每个开始标签是否有对应的 `</...>` |
| 图片不显示 | `src` 路径错误 | 检查文件名拼写和路径是否正确 |
| 页面空白 | 内容写在 `<head>` 里了 | 确保可见内容在 `<body>` 标签内 |

**教学提示：** 教学生使用浏览器开发者工具（F12）的 Elements 面板检查标签结构，这是 debug 的第一步。

### 五、总结与回顾（5 分钟）

> 今天我们学了什么？
> 1. HTML 是网页的骨架
> 2. 标签是 HTML 的基本单位，大多数成对出现
> 3. 每个 HTML 文件都有 `<!DOCTYPE>`、`<html>`、`<head>`、`<body>` 的固定结构
> 4. `<h1>`~`<h6>` 是标题，`<p>` 是段落，`<img>` 是图片
> 5. 属性（如 `src`、`alt`）给标签添加额外信息

### 六、课后延伸（5 分钟）

- **基础作业：** 在个人简介页面中添加 3 个以上段落，使用至少 3 种不同的标题级别
- **进阶挑战：** 尝试搜索 `<strong>`、`<em>`、`<br>`、`<hr>` 标签的用途，在页面中使用它们
- **预习提示：** 下节课我们将学习超链接、列表和表单——网页中最常用的交互元素！

---

> **备课提醒：** 课前确保所有学生的编辑器和浏览器环境已配置好。推荐使用 VS Code + Live Server 插件，实现保存后自动刷新浏览器。
