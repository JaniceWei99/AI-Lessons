# 第5课：JavaScript 入门（上）— 变量、函数与事件

## 教学目标

- 理解 JavaScript 的角色：网页的"大脑"
- 掌握引入 JS 的方式（内联脚本、外部文件）和 `console.log` 的使用
- 学会声明变量（`let`、`const`）和基本数据类型（字符串、数字、布尔值）
- 掌握模板字符串、基本运算符
- 理解函数的定义与调用（声明式函数、箭头函数）
- 学会事件处理（`onclick`、`addEventListener`）
- 完成一个点击计数器应用

## 课时建议

**总时长：90 分钟**

## 教学流程

### 一、导入环节（8 分钟）

> **教师讲稿：**
>
> 同学们，前四节课我们用 HTML 搭好了骨架，用 CSS 穿上了漂亮的衣服。但目前我们的网页像一幅画——好看，但不能动。今天开始，我们给网页装上"大脑"——JavaScript！
>
> JavaScript 是世界上使用最广泛的编程语言之一。有了它，按钮可以被点击、表单可以被验证、内容可以动态更新。以后我们还会用它调用 AI 的接口！
>
> 我先给大家看个效果：（演示一个简单的点击计数器）每次点按钮，数字就加1——这就是 JavaScript 的魔力。今天我们就来实现它。

**教学提示：** 让学生打开浏览器控制台（F12 → Console），输入 `alert("Hello!")` 并回车，看到弹窗——这是他们写的第一行 JavaScript！

### 二、核心概念讲解（30 分钟）

#### 2.1 引入 JavaScript（3 分钟）

```html
<!-- 方式一：内联脚本（写在 HTML 底部） -->
<script>
  console.log("Hello JavaScript!");
</script>

<!-- 方式二：外部文件（推荐！） -->
<script src="script.js"></script>
```

> `<script>` 标签放在 `</body>` 前面——这样 HTML 先加载完，JS 再执行，不会出现找不到元素的问题。`console.log()` 是你的"调试好朋友"——它在控制台打印信息，帮你检查代码是否按预期运行。

**教学提示：** 让学生打开 DevTools 的 Console 面板，现场输入 `console.log("我的第一行JS")` 看效果。

#### 2.2 变量与数据类型（8 分钟）

> 变量就像一个有名字的盒子，你可以往里面放东西：

```javascript
// let —— 可以改变的变量
let score = 0;
score = 10; // 可以重新赋值

// const —— 不可改变的常量
const name = "小明";
// name = "小红"; // 报错！const 不能改

// 三种基本数据类型
let userName = "小明";       // 字符串（用引号包裹）
let age = 13;                // 数字（不用引号）
let isStudent = true;        // 布尔值（true 或 false）
```

> **重要规则：优先用 `const`，只有确定需要改变时才用 `let`。永远不要用 `var`（旧语法，有坑）。**

#### 2.3 模板字符串与运算符（4 分钟）

```javascript
// 模板字符串 —— 用反引号 `` 包裹，${} 插入变量
const name = "小明";
const age = 13;
console.log(`我叫${name}，今年${age}岁`);

// 常用运算符
let a = 10 + 3;   // 13  加
let b = 10 - 3;   // 7   减
let c = 10 * 3;   // 30  乘
let d = 10 / 3;   // 3.33 除
let e = 10 % 3;   // 1   取余
```

> 注意反引号不是普通的单引号！它在键盘左上角，Tab 键上方。模板字符串比用 `+` 拼接字符串方便多了。

#### 2.4 函数（8 分钟）

> 函数就是"可以反复使用的代码块"。就像你在手机上设了一个闹钟——定义好之后，每到时间就自动执行。

```javascript
// 声明式函数
function greet(name) {
  return `你好，${name}！`;
}
console.log(greet("小明")); // "你好，小明！"

// 箭头函数（ES6 简写方式）
const add = (a, b) => {
  return a + b;
};
console.log(add(3, 5)); // 8

// 如果只有一行，可以更简洁
const double = (n) => n * 2;
```

> 函数有三个要素：名字、参数（输入）、返回值（输出）。`return` 关键字把结果"送出去"。

#### 2.5 事件处理（7 分钟）

> 事件是用户和网页交互的方式——点击、输入、鼠标移动都是事件。

```javascript
// 方式一（不推荐）：HTML 中直接写
// <button onclick="handleClick()">点我</button>

// 方式二（推荐）：addEventListener
const button = document.querySelector("#myBtn");
button.addEventListener("click", function() {
  console.log("按钮被点击了！");
});

// 常用事件
// "click"    —— 点击
// "input"    —— 输入内容变化
// "submit"   —— 表单提交
// "keydown"  —— 按下键盘
// "mouseover" —— 鼠标移入
```

> `addEventListener` 是更专业的方式，因为它可以给同一个元素绑定多个事件，而 `onclick` 只能绑一个。

### 三、编码指导 — 动手实践（35 分钟）

#### 3.1 教师带码 — 点击计数器（15 分钟）

> 我们一起做一个点击计数器！

**HTML 部分：**
```html
<div class="counter-app">
  <h1>点击计数器</h1>
  <p id="count-display">0</p>
  <button id="add-btn">+1</button>
  <button id="subtract-btn">-1</button>
  <button id="reset-btn">重置</button>
</div>
```

**JS 部分（逐步构建）：**
```javascript
// 第一步：获取元素
const display = document.querySelector("#count-display");
const addBtn = document.querySelector("#add-btn");
const subtractBtn = document.querySelector("#subtract-btn");
const resetBtn = document.querySelector("#reset-btn");

// 第二步：声明变量
let count = 0;

// 第三步：定义更新显示的函数
const updateDisplay = () => {
  display.textContent = count;
};

// 第四步：绑定事件
addBtn.addEventListener("click", () => {
  count++;
  updateDisplay();
});

subtractBtn.addEventListener("click", () => {
  count--;
  updateDisplay();
});

resetBtn.addEventListener("click", () => {
  count = 0;
  updateDisplay();
});
```

**教学提示：** 逐步编写，每完成一步就保存测试。先只做 +1 按钮，确认生效后再加 -1 和重置。边写边在控制台用 `console.log(count)` 验证。

#### 3.2 学生自主编码（20 分钟）

> 请独立完成点击计数器，然后尝试以下扩展：
> 1. 当计数为正数时显示绿色，负数时显示红色
> 2. 添加一个 +5 和 -5 的按钮
> - **挑战：** 设置计数上限（如100）和下限（如-100），到达极限时弹出提示

### 四、常见问题与排错（7 分钟）

| 常见错误 | 原因 | 解决方法 |
|---------|------|---------|
| `Cannot read property of null` | `querySelector` 找不到元素 | 检查选择器拼写，确保 JS 在 HTML 元素之后加载 |
| 点击无反应 | 事件没绑定成功或函数名拼错 | 在事件回调函数中加 `console.log` 测试 |
| 变量值不变 | 用了 `const` 声明需要改变的变量 | 需要改变的变量用 `let` |
| 字符串拼接结果奇怪 | `"1" + 1` 得到 `"11"` 而不是 `2` | 注意数据类型，字符串和数字相加会拼接 |
| 反引号打不出来 | 不知道按键位置 | 键盘左上角，Tab 键上方，需要切换到英文输入法 |

**教学提示：** 教学生使用 `console.log` 进行"断点式"调试——在关键位置打印变量值，追踪程序执行流程。也可以在 DevTools 的 Sources 面板设置断点，逐行执行。

### 五、总结与回顾（5 分钟）

> 今天的 JavaScript 基础：
> 1. **变量**：`let`（可变）和 `const`（不可变），优先用 `const`
> 2. **数据类型**：字符串、数字、布尔值
> 3. **函数**：可复用的代码块，建议使用箭头函数
> 4. **事件**：用 `addEventListener` 响应用户操作
> 5. **调试**：`console.log` 是你最好的朋友！

### 六、课后延伸（5 分钟）

- **基础作业：** 给计数器添加颜色变化功能——正数绿色、零黑色、负数红色
- **进阶挑战：** 制作一个"猜数字"小游戏——程序随机生成一个1-100的数字，用户输入猜测，程序提示"太大"或"太小"
- **预习提示：** 下节课学习 DOM 操作——用 JavaScript 动态增删网页元素，做一个待办事项应用！

---

> **备课提醒：** JavaScript 是很多学生第一次接触"真正的编程"，可能会感到有难度。节奏要慢一些，多用 `console.log` 演示每一步的结果。鼓励学生在控制台里"玩"——随意输入表达式看结果，建立对编程的直觉。
