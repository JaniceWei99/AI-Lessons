# 第6课：JavaScript 入门（下）— DOM 操作与表单交互

## 教学目标

- 理解 DOM 树结构及其与 HTML 的关系
- 掌握查找元素的方法：`getElementById`、`querySelector`、`querySelectorAll`
- 学会修改文本内容（`textContent`/`innerHTML`）和样式
- 掌握类名操作：`classList.add/remove/toggle/contains`
- 学会动态创建和删除 DOM 元素
- 理解表单交互：读取值、验证、`preventDefault`、`form.reset()`
- 完成一个 To-Do List（待办事项）应用

## 课时建议

**总时长：90 分钟**

## 教学流程

### 一、导入环节（7 分钟）

> **教师讲稿：**
>
> 上节课我们学会了 JavaScript 的基本语法——变量、函数和事件。今天我们要学一个更强大的能力：**用 JS 操控网页上的所有元素**。
>
> 浏览器把 HTML 解析成一棵"树"——叫做 DOM（Document Object Model，文档对象模型）。你可以把 DOM 想象成一棵家族树：`<html>` 是老祖宗，`<head>` 和 `<body>` 是两个孩子，`<body>` 下面又有各种子孙后代。JavaScript 可以沿着这棵树，找到任何元素，然后修改它、删除它，甚至创建新的元素！
>
> 今天的目标是做一个 To-Do List 待办事项应用——你可以输入任务、添加到列表、标记完成、删除任务。这用到了今天学的所有知识。

**教学提示：** 在 DevTools 的 Elements 面板中展开一个真实网页的 DOM 树，让学生看到层级关系。

### 二、核心概念讲解（28 分钟）

#### 2.1 查找 DOM 元素（6 分钟）

```javascript
// 按 ID 查找（返回单个元素）
const title = document.getElementById("main-title");

// CSS 选择器查找（返回第一个匹配的元素，最常用！）
const btn = document.querySelector(".submit-btn");
const firstCard = document.querySelector(".card");

// 查找所有匹配元素（返回 NodeList）
const allCards = document.querySelectorAll(".card");
allCards.forEach(card => {
  console.log(card.textContent);
});
```

> 推荐统一使用 `querySelector` 和 `querySelectorAll`——它们用 CSS 选择器语法，和我们学 CSS 时一样：`.` 是类名，`#` 是 ID。

#### 2.2 修改内容和样式（6 分钟）

```javascript
const title = document.querySelector("#main-title");

// 修改文字内容
title.textContent = "新标题";       // 纯文本（安全！）
title.innerHTML = "<em>新标题</em>"; // 可以包含 HTML 标签

// 修改样式
title.style.color = "blue";
title.style.fontSize = "24px";      // 注意：CSS 的 font-size 变成 fontSize
title.style.backgroundColor = "#f0f0f0"; // 驼峰命名法！
```

> `textContent` 和 `innerHTML` 的区别：`textContent` 只处理纯文本，更安全；`innerHTML` 可以插入 HTML 标签，但要小心用户输入的恶意代码（XSS 攻击）。在这个课程中，处理用户输入时优先用 `textContent`。

#### 2.3 类名操作（5 分钟）

```javascript
const element = document.querySelector(".card");

element.classList.add("active");     // 添加类名
element.classList.remove("active");  // 移除类名
element.classList.toggle("active");  // 切换：有则删，无则加
element.classList.contains("active"); // 检查是否包含，返回 true/false
```

> 为什么要用类名操作而不是直接改 `style`？因为把样式写在 CSS 中，用 JS 切换类名，这样代码更干净、更好维护。比如实现暗色模式，只需要给 `<body>` 切换一个 `dark-mode` 类名就行了。

**现场演示：** 用 `classList.toggle` 实现一个按钮切换暗色/亮色主题的效果。

#### 2.4 创建与删除元素（5 分钟）

```javascript
// 创建新元素
const newItem = document.createElement("li");
newItem.textContent = "新的待办事项";
newItem.classList.add("todo-item");

// 添加到页面
const list = document.querySelector("#todo-list");
list.appendChild(newItem);          // 添加到末尾

// 删除元素
newItem.remove();                   // 直接删除自身
// 或者
list.removeChild(newItem);          // 从父元素中删除子元素
```

#### 2.5 表单交互（6 分钟）

```javascript
const form = document.querySelector("#todo-form");
const input = document.querySelector("#todo-input");

form.addEventListener("submit", (event) => {
  event.preventDefault();  // 阻止表单默认提交（页面刷新）

  const value = input.value.trim();  // 获取输入值并去掉首尾空格

  // 简单验证
  if (value === "") {
    alert("请输入内容！");
    return;  // 提前返回，不执行后面的代码
  }

  console.log("用户输入了：", value);
  input.value = "";  // 清空输入框
  // 或者用 form.reset(); 重置整个表单
});
```

> `preventDefault()` 非常重要！表单的默认行为是提交后刷新页面，但我们要用 JavaScript 来处理数据，不需要刷新。

### 三、编码指导 — 动手实践（40 分钟）

#### 3.1 教师带码 — To-Do List（18 分钟）

**教学提示：** 这是本课的核心项目，分步骤构建，每步保存测试。

**HTML：**
```html
<div class="todo-app">
  <h1>我的待办事项</h1>
  <form id="todo-form">
    <input type="text" id="todo-input" placeholder="输入新任务...">
    <button type="submit">添加</button>
  </form>
  <ul id="todo-list"></ul>
</div>
```

**JS（逐步构建）：**

```javascript
// 第一步：获取元素
const form = document.querySelector("#todo-form");
const input = document.querySelector("#todo-input");
const todoList = document.querySelector("#todo-list");

// 第二步：监听表单提交
form.addEventListener("submit", (e) => {
  e.preventDefault();

  const taskText = input.value.trim();
  if (taskText === "") return;

  // 第三步：创建新的待办项
  const li = document.createElement("li");
  li.classList.add("todo-item");

  // 任务文字
  const span = document.createElement("span");
  span.textContent = taskText;

  // 完成按钮
  const completeBtn = document.createElement("button");
  completeBtn.textContent = "✓";
  completeBtn.classList.add("complete-btn");
  completeBtn.addEventListener("click", () => {
    li.classList.toggle("completed");
  });

  // 删除按钮
  const deleteBtn = document.createElement("button");
  deleteBtn.textContent = "✕";
  deleteBtn.classList.add("delete-btn");
  deleteBtn.addEventListener("click", () => {
    li.remove();
  });

  // 第四步：组装并添加到列表
  li.appendChild(span);
  li.appendChild(completeBtn);
  li.appendChild(deleteBtn);
  todoList.appendChild(li);

  // 第五步：清空输入框
  input.value = "";
  input.focus();
});
```

**配套 CSS：**
```css
.todo-item { display: flex; align-items: center; gap: 10px; padding: 8px; }
.todo-item.completed span { text-decoration: line-through; color: #999; }
.complete-btn { color: green; }
.delete-btn { color: red; }
```

**教学提示：** 先做最简版——只有添加功能，确认生效后再加完成和删除按钮。在每个步骤后，在 Console 中打印验证（如 `console.log("添加了：", taskText)`）。

### 四、常见问题与排错（5 分钟）

| 常见错误 | 原因 | 解决方法 |
|---------|------|---------|
| 表单提交后页面刷新 | 忘了 `event.preventDefault()` | 在事件回调的第一行调用 |
| `input.value` 为空 | 在表单提交后才获取的值 | 确保在 `preventDefault` 之后、`reset` 之前获取 |
| 删除按钮无效果 | 事件绑定在创建元素之前 | 在 `createElement` 之后立即绑定事件 |
| `textContent` 没变化 | 选择器写错了，获取的元素为 null | 用 `console.log` 检查元素是否获取成功 |
| CSS 驼峰命名搞混 | `font-size` 在 JS 中要写成 `fontSize` | CSS 中的短横线 → JS 中改为驼峰 |

**教学提示：** 教学生使用 DevTools 的 Elements 面板实时观察 DOM 变化——添加任务时可以看到新的 `<li>` 出现在 DOM 树中。

### 五、总结与回顾（5 分钟）

> JavaScript 两节课的核心能力：
> 1. **查找元素**：`querySelector` / `querySelectorAll`
> 2. **修改内容**：`textContent`（安全）、`innerHTML`（谨慎）
> 3. **操作类名**：`classList.add/remove/toggle` 控制样式
> 4. **创建/删除**：`createElement` + `appendChild` / `remove`
> 5. **表单交互**：`preventDefault` + `input.value` + `form.reset()`
>
> 至此，HTML + CSS + JavaScript 三件套就全部入门了！下节课开始真正激动人心的部分——调用网络上的 API！
