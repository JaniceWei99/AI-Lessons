# 第4课：CSS 美化（下）— Flexbox 布局与响应式设计

## 教学目标

- 理解 Flexbox 布局的核心概念（主轴、交叉轴）
- 掌握常用 Flex 属性：`display:flex`、`flex-direction`、`justify-content`、`align-items`、`gap`、`flex-wrap`
- 能用 Flexbox 实现导航栏、卡片网格、居中布局等常见场景
- 理解响应式设计的意义，掌握媒体查询（media queries）的用法
- 了解移动优先（mobile-first）设计理念
- 完成一个响应式个人作品集页面

## 课时建议

**总时长：90 分钟**

## 教学流程

### 一、导入环节（8 分钟）

> **教师讲稿：**
>
> 上节课我们学了颜色、字体和盒模型。但有一个问题一直困扰着初学者——怎么把元素放到我想要的位置？比如，我想让导航栏水平排列，想让卡片并排显示，想让内容完美居中。过去这是 CSS 最痛苦的部分，但 Flexbox（弹性盒子）的出现改变了一切！
>
> 另外，现在大部分人用手机上网。如果你的网页只在电脑上好看，在手机上却乱掉了，那可不行。今天后半节课，我们还要学习"响应式设计"——让网页自动适应不同屏幕大小。

**教学提示：** 用手机打开一个没有做响应式的网页（如某些老旧政府网站），再打开一个响应式网页（如 apple.com），让学生对比体验。

### 二、核心概念讲解（30 分钟）

#### 2.1 Flexbox 基础（15 分钟）

> Flexbox 的核心思想：给父容器加上 `display: flex`，它的子元素就会自动按一行排列。

```css
.container {
  display: flex;           /* 开启弹性布局 */
  flex-direction: row;     /* 主轴方向：row(水平) | column(垂直) */
  justify-content: center; /* 主轴对齐方式 */
  align-items: center;     /* 交叉轴对齐方式 */
  gap: 20px;               /* 子元素之间的间距 */
  flex-wrap: wrap;          /* 空间不够时换行 */
}
```

> 把 Flexbox 想象成一条"传送带"：`flex-direction` 决定传送带的方向（水平或垂直），`justify-content` 决定物品在传送带上怎么分布（靠左、居中、两端分散），`align-items` 决定物品在传送带的垂直方向怎么对齐。
>
> `justify-content` 的常用值：
> - `flex-start`（默认，靠左）
> - `center`（居中）
> - `space-between`（两端对齐，中间等距）
> - `space-around`（每个元素两边等距）
>
> 记住一个万能居中技巧：

```css
.center-everything {
  display: flex;
  justify-content: center;
  align-items: center;
}
```

> 这三行代码可以让任何内容水平垂直都居中——以前需要很多 hack 才能做到的事，现在三行搞定！

**现场演示：** 逐步修改 `justify-content` 和 `align-items` 的值，让学生实时观察布局变化。推荐使用 Flexbox Froggy（flexboxfroggy.com）让学生互动练习。

#### 2.2 实用 Flexbox 布局（5 分钟）

```css
/* 导航栏 */
.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 10px 20px;
  background: #2c3e50;
}

/* 卡片网格 */
.card-grid {
  display: flex;
  flex-wrap: wrap;
  gap: 20px;
}
.card-grid .card {
  flex: 1 1 300px; /* 最小 300px，自动拉伸填满空间 */
}
```

#### 2.3 响应式设计与媒体查询（10 分钟）

> 响应式设计的目标是：**一套代码，适配所有屏幕。**
>
> 首先确保 HTML 中有 viewport meta 标签（我们第1课已经加了）：
> ```html
> <meta name="viewport" content="width=device-width, initial-scale=1.0">
> ```
>
> 然后使用**媒体查询（Media Queries）**在不同屏幕宽度下应用不同样式：

```css
/* 默认样式（手机优先） */
.card-grid {
  display: flex;
  flex-direction: column;
  gap: 16px;
}

/* 平板（768px 以上） */
@media (min-width: 768px) {
  .card-grid {
    flex-direction: row;
    flex-wrap: wrap;
  }
  .card { flex: 1 1 45%; }
}

/* 桌面（1024px 以上） */
@media (min-width: 1024px) {
  .card { flex: 1 1 30%; }
}
```

> 常用断点参考：480px（大手机）、768px（平板）、1024px（小桌面）、1200px（大桌面）。
>
> **移动优先（Mobile First）** 的思路是：先写手机上的样式，然后用 `min-width` 逐步增强。这样做的好处是，手机用户不需要加载多余的样式。

**教学提示：** 在 DevTools 中打开设备模拟器（Ctrl+Shift+M），切换不同设备查看响应式效果。演示拖拽浏览器窗口宽度时布局动态变化的过程。

### 三、编码指导 — 动手实践（35 分钟）

#### 3.1 教师带码（12 分钟）

> 我们来创建一个响应式个人作品集页面：

```html
<nav class="navbar">
  <div class="logo">我的作品集</div>
  <div class="nav-links">
    <a href="#about">关于</a>
    <a href="#projects">项目</a>
    <a href="#contact">联系</a>
  </div>
</nav>

<main class="container">
  <section id="projects">
    <h2>我的项目</h2>
    <div class="card-grid">
      <div class="card">
        <h3>项目一</h3>
        <p>HTML 个人简介页面</p>
      </div>
      <div class="card">
        <h3>项目二</h3>
        <p>CSS 美化页面</p>
      </div>
      <div class="card">
        <h3>项目三</h3>
        <p>即将到来...</p>
      </div>
    </div>
  </section>
</main>
```

**教学提示：** 先写手机版样式，再逐步添加媒体查询。每添加一个断点，用 DevTools 设备模拟器演示效果。

#### 3.2 学生自主编码（23 分钟）

> 请完成你的响应式作品集页面：
> 1. 用 Flexbox 创建水平导航栏
> 2. 用卡片网格展示至少 3 个项目
> 3. 添加至少一个媒体查询，让页面在手机和电脑上都好看
> - **挑战：** 用 `flex-direction: column` 让导航栏在手机上变成垂直排列

### 四、常见问题与排错（7 分钟）

| 常见错误 | 原因 | 解决方法 |
|---------|------|---------|
| `display: flex` 没效果 | 加在了子元素上而非父容器 | Flex 属性要加在**父容器**上 |
| 元素不换行 | 忘了加 `flex-wrap: wrap` | 默认是 `nowrap`，必须显式开启 |
| 媒体查询没生效 | 缺少 viewport meta 标签 | 确保 `<head>` 中有 viewport 设置 |
| 媒体查询顺序错乱 | `max-width` 和 `min-width` 混用 | 移动优先统一用 `min-width`，从小到大写 |
| 居中不生效 | 容器没有设置高度 | 垂直居中需要容器有明确的高度 |

**教学提示：** 在 DevTools 中选中 flex 容器时，Chrome 会在元素旁显示一个 `flex` 标签，点击可以可视化查看 Flex 布局——这是调试 Flexbox 的利器！

### 五、总结与回顾（5 分钟）

> 今天两个大知识点：
> 1. **Flexbox** —— 用 `display: flex` + `justify-content` + `align-items` 解决 90% 的布局问题
> 2. **响应式设计** —— 用 `@media (min-width: ...)` 让网页适配不同屏幕
> 3. 万能居中：`display: flex; justify-content: center; align-items: center;`
> 4. 移动优先：先写小屏样式，再用 `min-width` 逐步增强
>
> 到这里，HTML + CSS 的基础就全部学完了！下节课开始进入 JavaScript——让网页"活"起来！

### 六、课后延伸（5 分钟）

- **基础作业：** 完善作品集页面，确保在手机和桌面上都能正常显示
- **进阶挑战：** 去 Flexbox Froggy（flexboxfroggy.com）完成所有关卡
- **预习提示：** 下节课开始学习 JavaScript——编程的基本功：变量、函数和事件
