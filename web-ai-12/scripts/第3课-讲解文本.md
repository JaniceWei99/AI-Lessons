# 第3课：CSS 美化（上）— 颜色、字体与盒模型

## 教学目标

- 理解 CSS 的角色：网页的"皮肤和衣服"
- 掌握三种引入 CSS 的方式（内联、内部、外部）
- 学会使用选择器（元素、类、ID）及优先级概念
- 掌握颜色表示方法（命名色、十六进制、RGB）
- 理解字体属性（font-family、font-size、font-weight、line-height）
- 深入理解盒模型（margin/border/padding/content）和 `box-sizing: border-box`

## 课时建议

**总时长：90 分钟**

## 教学流程

### 一、导入环节（8 分钟）

> **教师讲稿：**
>
> 同学们，过去两节课我们用 HTML 搭好了网页的骨架。但说实话，现在的页面看起来……是不是有点"朴素"？今天我们要学习 CSS，它的全称是 Cascading Style Sheets——层叠样式表。如果 HTML 是房子的砖墙，CSS 就是装修——刷墙漆、铺地板、挂窗帘。
>
> 我来给大家看一个对比：这是没有 CSS 的页面（展示纯 HTML），这是加了 CSS 之后的同一个页面（展示美化后的效果）。差别大不大？

**教学提示：** 准备好一个前后对比的 demo——同一个 HTML 文件，分别展示有无 CSS 的效果。可以使用 CSS Zen Garden（csszengarden.com）让学生感受"同样的 HTML，不同的 CSS"带来的惊艳效果。

### 二、核心概念讲解（30 分钟）

#### 2.1 三种引入 CSS 的方式（5 分钟）

> CSS 可以通过三种方式添加到 HTML 中：

```html
<!-- 1. 内联样式：直接写在标签上 -->
<h1 style="color: red;">你好</h1>

<!-- 2. 内部样式：写在 <head> 里的 <style> 标签中 -->
<style>
  h1 { color: red; }
</style>

<!-- 3. 外部样式：最推荐！单独的 .css 文件 -->
<link rel="stylesheet" href="style.css">
```

> 我们在课程中主要使用外部样式表，因为它让 HTML 和 CSS 分离，代码更整洁、更好维护。

#### 2.2 选择器与优先级（8 分钟）

> CSS 的核心语法是：**选择器 { 属性: 值; }**

```css
/* 元素选择器 —— 选中所有该标签 */
p { color: blue; }

/* 类选择器 —— 选中所有带该 class 的元素（最常用！） */
.highlight { background-color: yellow; }

/* ID 选择器 —— 选中唯一的特定元素 */
#main-title { font-size: 32px; }
```

> 优先级从低到高是：元素选择器 < 类选择器 < ID 选择器 < 内联样式。
>
> 记住口诀：**类选择器是你最好的朋友**——实际开发中 90% 的情况都用类选择器。ID 选择器虽然优先级高，但一个 ID 只能用一次，不灵活。

**现场演示：** 给同一个元素同时加上元素样式、类样式和 ID 样式，让学生观察哪个生效——直观理解优先级。

#### 2.3 颜色（5 分钟）

> CSS 中有多种表示颜色的方法：

```css
color: red;                  /* 命名颜色，共 140+ 种 */
color: #FF5733;              /* 十六进制，最常用 */
color: rgb(255, 87, 51);     /* RGB 值 */
background-color: #f0f0f0;   /* 背景色 */
```

> 十六进制颜色中，`#FF0000` 是纯红，`#00FF00` 是纯绿，`#0000FF` 是纯蓝。两两组合就能产生千万种颜色。

**教学提示：** 让学生打开浏览器开发者工具，点击颜色值旁边的小色块，使用取色器实时调色——这是前端开发中非常实用的技巧。

#### 2.4 字体属性（5 分钟）

```css
body {
  font-family: "Noto Sans SC", "Microsoft YaHei", sans-serif;
  font-size: 16px;
  font-weight: 400;       /* 400=正常, 700=粗体 */
  line-height: 1.6;       /* 行高：字号的 1.6 倍，提升阅读体验 */
}
```

> `font-family` 可以设置多个字体作为后备方案——如果第一个字体用户电脑没有，就用第二个，最后用通用字体族 `sans-serif`。想用更多漂亮字体？可以用 Google Fonts 免费加载！

#### 2.5 盒模型（7 分钟）

> **这是 CSS 中最重要的概念之一！** 每个 HTML 元素都是一个"盒子"，由四层组成：
>
> 从外到内：**margin（外边距）→ border（边框）→ padding（内边距）→ content（内容）**
>
> 想象一个快递包裹：content 是里面的商品，padding 是包裹里的泡沫，border 是纸箱，margin 是纸箱和其他包裹之间的距离。

```css
.card {
  margin: 20px;           /* 盒子外部的间距 */
  border: 2px solid #333; /* 边框 */
  padding: 15px;          /* 盒子内部的间距 */
  width: 300px;
}
```

> 默认情况下，`width` 只计算 content 的宽度，加上 padding 和 border 后实际宽度会更大。这很坑！所以我们加一行万能代码：

```css
* { box-sizing: border-box; }
```

> 加了这行后，`width` 就包含了 padding 和 border，计算尺寸变得简单多了。**请同学们养成习惯，每个项目开头都加上这行。**

**教学提示：** 让学生打开 DevTools → Elements → 选中任意元素，查看右侧的盒模型图。鼠标悬停在网页元素上时，会直接高亮显示 margin（橙色）、padding（绿色）和 content（蓝色）。

### 三、编码指导 — 动手实践（35 分钟）

#### 3.1 教师带码（12 分钟）

> 我们来给之前做的个人简介页面"换装"：

**教学提示：** 创建外部 `style.css` 文件，逐步添加样式。每加一个属性就保存刷新，让学生看到变化。

```css
* { box-sizing: border-box; margin: 0; padding: 0; }

body {
  font-family: "Noto Sans SC", sans-serif;
  font-size: 16px;
  line-height: 1.6;
  color: #333;
  background-color: #f5f5f5;
  padding: 20px;
}

h1 { color: #2c3e50; font-size: 28px; margin-bottom: 10px; }
h2 { color: #3498db; margin: 20px 0 10px; }
p  { margin-bottom: 12px; }

.card {
  background: white;
  padding: 24px;
  margin-bottom: 20px;
  border: 1px solid #ddd;
  border-radius: 8px;
}
```

#### 3.2 学生自主编码（23 分钟）

> 请给你的个人主页添加 CSS 样式：
> 1. 创建外部 `style.css` 文件并正确链接
> 2. 设置 `body` 的字体、背景色和文字颜色
> 3. 给标题设置不同的颜色和大小
> 4. 用 `padding` 和 `margin` 调整间距
> 5. **挑战：** 尝试用 Google Fonts 引入一种你喜欢的字体

### 四、常见问题与排错（7 分钟）

| 常见错误 | 原因 | 解决方法 |
|---------|------|---------|
| CSS 没有生效 | `<link>` 标签路径错误或拼写错误 | 检查 `href` 路径，确保文件名一致 |
| 样式被覆盖了 | 优先级冲突 | 用 DevTools 检查哪个样式被划掉了 |
| 宽度计算不对 | 没有用 `box-sizing: border-box` | 在 CSS 开头加上全局 `border-box` |
| 字体没有变化 | 字体名称拼写错误或无引号 | 多词字体名必须用引号包裹 |
| 颜色值无效 | 十六进制少了 `#` 号 | `#FF5733` 不能写成 `FF5733` |

**教学提示：** 重点演示 DevTools 中"样式被划掉"的情况——让学生学会查看 Computed（计算后）面板来理解最终生效的样式。

### 五、总结与回顾（5 分钟）

> 今天的核心知识点：
> 1. CSS 让网页从"素颜"变"精装修"
> 2. 推荐使用**外部样式表** + **类选择器**
> 3. 颜色三种写法：命名、十六进制（最常用）、RGB
> 4. 盒模型四层：margin → border → padding → content
> 5. **必加代码：** `* { box-sizing: border-box; }`

### 六、课后延伸（5 分钟）

- **基础作业：** 让你的个人页面至少使用 5 种不同的 CSS 属性
- **进阶挑战：** 访问 Google Fonts 选一种中文字体，应用到你的页面
- **预习提示：** 下节课学习 Flexbox 布局——让元素排列不再是噩梦！
