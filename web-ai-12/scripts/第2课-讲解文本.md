# 第2课：HTML 基础（下）— 链接、列表与表单

## 教学目标

- 掌握超链接 `<a>` 标签的用法（外部链接、内部链接、锚点链接、`target="_blank"`）
- 学会使用有序列表 `<ol>` 和无序列表 `<ul>`，理解列表嵌套
- 了解表单基础：`<input>` 各种类型、`<textarea>`、`<select>`、`<button>`
- 理解 `<label>` 与 `for`/`id` 的关联机制
- 完成一个包含导航、爱好列表和联系表单的个人主页

## 课时建议

**总时长：90 分钟**

## 教学流程

### 一、导入环节（8 分钟）

> **教师讲稿：**
>
> 上节课我们用 HTML 创建了个人简介页，但那个页面是"孤立的"——不能跳转到其他页面，也不能让用户输入信息。今天我们要给网页加上三个超级重要的功能：链接、列表和表单。
>
> 想象一下，如果网页没有链接，互联网就不存在了——你无法从一个页面跳到另一个页面。"超文本"中的"超"字，说的就是链接！

**教学提示：** 在浏览器中打开维基百科任意页面，演示点击链接跳转的效果，让学生体会"超链接"的含义。

### 二、核心概念讲解（25 分钟）

#### 2.1 超链接 `<a>` 标签（10 分钟）

> 链接用 `<a>` 标签（anchor，锚点的意思）：
>
> ```html
> <a href="https://www.baidu.com">打开百度</a>
> <a href="about.html">关于我</a>
> <a href="#hobbies">跳到爱好部分</a>
> ```
>
> `href` 属性指定目标地址。注意三种链接类型：外部链接（完整 URL）、内部链接（相对路径）、锚点链接（`#id` 跳到页面内某个位置）。
>
> 如果想让链接在新标签页打开，加上 `target="_blank"`：
> ```html
> <a href="https://www.baidu.com" target="_blank">新标签页打开百度</a>
> ```

**现场演示：** 创建两个 HTML 文件，演示它们之间如何互相链接。再演示锚点链接在长页面中的跳转效果。

#### 2.2 列表（7 分钟）

> 列表分两种——无序列表用圆点，有序列表用数字：
>
> ```html
> <!-- 无序列表 -->
> <ul>
>   <li>篮球</li>
>   <li>编程</li>
>   <li>画画</li>
> </ul>
>
> <!-- 有序列表 -->
> <ol>
>   <li>起床</li>
>   <li>刷牙</li>
>   <li>吃早餐</li>
> </ol>
> ```
>
> 列表还能嵌套！比如在"编程"下面再列出"Python"、"HTML"等子项。网站的导航栏本质上就是一个列表。

**教学提示：** 让学生打开任意网站的导航栏，用开发者工具查看，发现它们通常就是 `<ul>` + `<li>` + `<a>` 的组合。

#### 2.3 表单基础（8 分钟）

> 表单是用户和网站之间"对话"的窗口——登录、注册、搜索、留言，都需要表单。

```html
<form>
  <label for="name">姓名：</label>
  <input type="text" id="name" placeholder="请输入姓名">

  <label for="email">邮箱：</label>
  <input type="email" id="email" placeholder="example@mail.com">

  <label for="pwd">密码：</label>
  <input type="password" id="pwd">

  <label for="age">年龄：</label>
  <input type="number" id="age" min="1" max="120">

  <label for="msg">留言：</label>
  <textarea id="msg" rows="4"></textarea>

  <label for="city">城市：</label>
  <select id="city">
    <option value="beijing">北京</option>
    <option value="shanghai">上海</option>
    <option value="shenzhen">深圳</option>
  </select>

  <button type="submit">提交</button>
</form>
```

> 重点讲解 `<label>` 的 `for` 属性要和 `<input>` 的 `id` 属性一致——这样点击文字也能激活输入框，对用户体验和无障碍访问都很重要。
>
> `type="email"` 会自动验证邮箱格式，`type="password"` 会隐藏输入内容——浏览器帮我们做了很多事！

**现场演示：** 逐个演示不同 `input type` 的效果，特别是 `email` 提交时的自动验证。

### 三、编码指导 — 动手实践（40 分钟）

#### 3.1 教师带码（15 分钟）

> 我们来升级上节课的个人主页，加入导航、爱好列表和联系表单：

**教学提示：** 逐步构建，每完成一个模块就保存刷新。用 Live Server 插件可实时预览。

```html
<body>
  <!-- 导航 -->
  <nav>
    <a href="#about">关于我</a> |
    <a href="#hobbies">我的爱好</a> |
    <a href="#contact">联系我</a>
  </nav>

  <h1>小明的个人主页</h1>

  <section id="about">
    <h2>关于我</h2>
    <p>我是一名热爱编程的中学生...</p>
  </section>

  <section id="hobbies">
    <h2>我的爱好</h2>
    <ul>
      <li>编程
        <ul>
          <li>HTML/CSS</li>
          <li>Python</li>
        </ul>
      </li>
      <li>篮球</li>
      <li>阅读</li>
    </ul>
  </section>

  <section id="contact">
    <h2>联系我</h2>
    <form>
      <label for="visitor-name">你的名字：</label>
      <input type="text" id="visitor-name" required><br><br>
      <label for="visitor-email">你的邮箱：</label>
      <input type="email" id="visitor-email" required><br><br>
      <label for="message">留言：</label><br>
      <textarea id="message" rows="4" cols="40"></textarea><br><br>
      <button type="submit">发送</button>
    </form>
  </section>
</body>
```

#### 3.2 学生自主编码（25 分钟）

> 请完成以下任务：
> 1. 在个人主页中添加导航栏，至少包含 3 个锚点链接
> 2. 用无序列表列出你的爱好，至少一个嵌套子列表
> 3. 创建一个联系表单，包含姓名、邮箱、留言和提交按钮
> - **挑战 1：** 创建 3 个独立的 HTML 页面，用链接互相连接
> - **挑战 2：** 制作一个调查问卷表单，使用 `<select>`、`<input type="number">` 等

### 四、常见问题与排错（7 分钟）

| 常见错误 | 原因 | 解决方法 |
|---------|------|---------|
| 链接点击没反应 | `href` 值为空或拼写错误 | 检查 URL 或文件路径是否正确 |
| 锚点跳转不工作 | `href="#xxx"` 和目标元素 `id="xxx"` 不匹配 | 确保两者完全一致，注意大小写 |
| 列表没有缩进/圆点 | `<li>` 没有放在 `<ul>` 或 `<ol>` 内 | `<li>` 必须是列表标签的直接子元素 |
| 点击标签文字不能聚焦输入框 | `label` 的 `for` 和 `input` 的 `id` 不匹配 | 检查两者的值是否一致 |
| 表单提交后页面刷新 | 这是默认行为，后面学 JS 会解决 | 暂时忽略，第6课会讲 `preventDefault` |

**教学提示：** 教学生右键检查元素，查看嵌套结构是否正确。列表嵌套错误是初学者最常犯的错误之一。

### 五、总结与回顾（5 分钟）

> 两节 HTML 课下来，我们已经掌握了网页的核心骨架：
> 1. **链接** `<a>` —— 把网页连接成"网"
> 2. **列表** `<ul>`/`<ol>` —— 结构化地展示信息
> 3. **表单** `<form>` —— 让用户和网页互动
> 4. HTML 就像是在写一封"结构化的信"，每个标签都有自己的意义

### 六、课后延伸（5 分钟）

- **基础作业：** 完善个人主页，确保所有链接、列表、表单都能正常工作
- **进阶挑战：** 创建一个包含 3 个页面的迷你网站（首页、爱好页、联系页），页面之间可以互相跳转
- **预习提示：** 下节课进入 CSS 的世界——我们要让这些"素颜"的网页变得漂亮起来！

---

> **备课提醒：** 确保学生理解"文件路径"的概念——相对路径和绝对路径。创建多页面网站时，文件必须在同一个文件夹或正确的相对位置。
