## 1. 课程简介

「AI 探索之旅」是一套面向 10–16 岁青少年的人工智能入门课程。提供五个版本，从零开始带你了解 AI 的方方面面。

#### 精简版 (standard-5) — 5 课时

| 课号 | 标题                | 你将学到                                       |
|:----:|:-------------------|:----------------------------------------------|
|  1   | 走进 AI 的世界      | AI 的定义、发展历史、日常应用                      |
|  2   | 机器是怎么"学习"的  | 数据、监督学习、决策树、线性回归                    |
|  3   | 神经网络与深度学习   | 人工神经元、网络结构、CNN、训练过程                 |
|  4   | 大语言模型与 AIGC   | Token、Transformer、Prompt 技巧、AI 生成内容      |
|  5   | AI 的未来与责任     | AI 偏见、深度伪造、隐私保护、未来能力               |

#### 完整版 (standard-15) — 15 课时

涵盖 AI 基础到进阶的所有核心主题，适合课时充裕的教学场景。

#### 实验版 (lab-10) — 10 个动手实验

以动手实践为主，每课包含编程实验和互动练习。

#### AI App 创造营 (app-inventor-10) — 10 课时

使用 MIT App Inventor + AI API（ChatGPT / Gemini），制作真正能安装到手机上的智能 App。适合 10–14 岁零基础学生。

#### AI 网站工坊 (web-ai-12) — 12 课时

学习 HTML / CSS / JavaScript + AI API，做出有 AI 聊天助手和图像生成功能的真网站，部署到 GitHub Pages 上线可访问。适合 12–16 岁学生。

> 五个版本均可从门户首页选择进入。

**建议学习顺序**: 选择适合的版本后，按课号顺序依次学习。精简版每课约 45 分钟，创造营和网站工坊每课约 1.5 小时。

## 2. 打开课程

### 在线访问

如果课程已部署到服务器，在浏览器中输入老师提供的网址即可。

### 本地打开

1. 在电脑上找到 `AI-Class` 文件夹
2. 双击 `index.html`
3. 浏览器将自动打开课程首页

> 支持的浏览器：Chrome、Edge、Firefox、Safari（推荐使用 Chrome 或 Edge）。

## 4. 语言切换

课程支持**中文**和 **English** 两种语言。

| 操作                            | 说明                           |
|:-------------------------------|:-------------------------------|
| 点击导航栏右上角 **EN** 按钮      | 切换到英文                      |

## 6. 课程学习流程

### 推荐学习步骤

```
1. 从首页点击"开始探索"或选择第 1 课
2. 从上到下阅读课程内容
3. 遇到交互组件时动手参与
4. 阅读完内容后完成课末测验
5. 点击底部"下一课"进入下一课
6. 完成第 5 课的结业测验
```

### 自由学习

你也可以：
- 通过顶部导航栏跳到任意课程
- 重复访问已完成的课程
- 重新作答测验（刷新页面重置）

## 7. 交互组件使用说明

### 7.1 交互式时间线（第 1 课）

| 操作         | 效果                                   |
|:------------|:---------------------------------------|
| 向下滚动     | 时间线事件卡片依次淡入                    |
| 点击事件卡片 | 展开显示该事件的详细信息                  |
| 再次点击     | 收起详细信息                              |

### 7.2 AI 还是人类？（第 1 课）

一个猜谜互动：判断展示的内容是 AI 生成还是人类创作。

| 操作                   | 效果                                 |
|:----------------------|:-------------------------------------|
| 阅读展示内容            | 思考这是 AI 还是人类的作品             |

通过回答是/否问题来分类动物，体验机器学习中的决策树。

| 操作            | 效果                                 |

| 点击"是"或"否"   | 跳到下一个问题节点                    |
| 到达叶节点       | 显示分类结果和完整路径                 |
| 点击"重新开始"   | 重置决策树，可以重新玩                 |

### 7.4 可展开卡片（第 3、4、5 课）

| 操作         | 效果                    |
|:------------|:-----------------------|

| 再次点击     | 收起内容                |

### 7.5 Prompt 挑战（第 4 课）

| 操作                  | 效果                              |
|:---------------------|:---------------------------------|

| 点击"查看参考答案"      | 展开显示优秀 Prompt 示例和解释     |
| 点击"收起参考答案"      | 收起参考内容                       |

## 8. 测验系统

每课末尾都有一个小测验。

### 基本规则

| 规则           | 说明                                      |
|:--------------|:-----------------------------------------|
| 题型           | 单选题 (A/B/C/D)                          |
| 题数           | 每课 3–10 题不等                            |
| 作答次数        | 每题只能作答 1 次                           |
| 即时反馈        | 选中后立刻显示对错                           |
| 答案解释        | 错误时显示正确答案及解释                     |
| 部分提交        | 答第一题后出现"提交测验"按钮，可随时提交       |
| 正确率统计      | 全部答完或提交后显示得分和正确率百分比          |
| 智能反馈        | 全对→鼓励、低于60%→建议重学、其他→继续学习     |
| 漏答提示        | 未作答的题目按错误计分，并显示橙色提醒          |

### 操作流程

```
1. 阅读题目
2. 点击你认为正确的选项
3. 正确 → 选项变绿色
   错误 → 你的选项变红色，正确答案用绿色标出
4. 查看题目下方的解释
5. 继续下一题
6. 全部完成 → 自动显示结果
   或点击"提交测验" → 提交已答题目（未答算错）
7. 查看正确率和反馈建议
```

### 进度保存

- 完成测验后，分数自动保存
- 首页的课程卡片上会显示完成标记和分数
- 进度保存在浏览器本地，清除浏览器数据会丢失

## 9. 学习进度

| 功能               | 说明                                          |
|:------------------|:---------------------------------------------|
| 自动记录           | 完成课程测验后自动标记该课为"已完成"              |
| 首页显示           | 已完成的课程卡片上显示 "✓ 已完成 (得分)"          |
| 存储位置           | 浏览器 localStorage（本地存储）                  |
| 跨设备同步         | 不支持 — 进度仅保存在当前浏览器                   |
| 重置进度           | 清除浏览器 localStorage 即可重置所有进度          |

### 手动重置进度

在浏览器控制台 (F12 → Console) 中输入：

```javascript
localStorage.removeItem('ai-class-progress');
location.reload();
```

## 10. 打印 / 导出

课程支持打印为纸质讲义：

1. 在课程页面按 `Ctrl + P` (Windows) 或 `Cmd + P` (Mac)
2. 打印预览会自动：
   - 隐藏导航栏、交互按钮
   - 展开所有折叠内容
   - 显示测验正确答案
   - 使用白色背景 + 黑色文字
3. 选择打印或"另存为 PDF"

## 11. 教师讲解文本

项目为全部 52 课提供了教师讲解文本（讲稿），以 Markdown 文件形式放在各版本的 `scripts/` 目录中，供授课教师备课参考。

### 文件位置

| 版本 | 路径 | 文件数 |
|:----|:-----|:------:|
| 精简版 | `standard-5/scripts/第X课-讲解文本.md` | 5 |
| 完整版 | `standard-15/scripts/第X课-讲解文本.md` | 15 |
| 实验版 | `lab-10/scripts/第X课-讲解文本.md` | 10 |
| 创造营 | `app-inventor-10/scripts/第X课-讲解文本.md` | 10 |
| 网站工坊 | `web-ai-12/scripts/第X课-讲解文本.md` | 12 |

### 讲稿内容

每份讲稿包含以下部分：

| 章节 | 说明 |
|:----|:----|
| 课程标题 | 中英文课名 |
| 教学目标 | 本课学生应掌握的 3-7 个要点 |
| 课时建议 | 建议的总时长 |
| 教学流程 | 分段时间表，各环节的时间分配 |
| 导入环节 | 开场白、引入方式、破冰活动 |
| 核心概念讲解 | 每个知识点怎么讲，配有类比、故事、例子 |
| 互动/实操环节 | 怎么带学生做活动或实验，包含引导问题 |
| 常见提问 | 学生可能的问题及建议回答 |
| 总结与回顾 | 收尾方式与下节课预告 |
| 课后延伸 | 可选作业和进阶挑战 |

### 使用建议

1. 课前阅读对应课号的讲解文本，了解教学流程与时间分配
2. 根据实际班级情况调整时间和互动方式
3. 授课时配合网页内容展示，讲稿中标有 "展示网页 X.X 节" 的提示
4. 讲稿中的 "此时可以问学生..." 为建议的课堂互动点
5. 可根据需要增删讲稿中的例子和类比

## 12. 常见问题 FAQ

### Q: 页面打开是空白的？

A: 请确认使用现代浏览器（Chrome / Edge / Firefox / Safari）。不支持 IE 浏览器。

### Q: 切换语言后部分内容没变？

A: 所有文本内容都支持双语切换。如果发现遗漏，请联系教师反馈。

### Q: 测验可以重新答吗？

A: 刷新页面即可重新作答。但已保存的最高分不会被覆盖。

### Q: 换了电脑后进度不见了？

A: 进度保存在浏览器本地存储中，不会跨设备同步。请在同一台电脑、同一个浏览器中学习。

### Q: 没有网络也能用吗？

A: 可以。将整个 `AI-Class` 文件夹拷贝到电脑上，双击 `index.html` 即可离线使用。唯一的区别是中文字体可能使用系统默认字体。

### Q: 手机上可以学吗？

A: 可以。课程完全支持手机浏览器，界面会自动适配小屏幕。
