### Session 1 — 2026-03-25 22:00 +0800

#### 主题: 项目规划与规格制定

**讨论要点**:
- 确定课程目标受众: 13–15 岁初中生
- 确定课程总量: 5 课时，每课 45 分钟
- 确定技术方案: 纯前端 (HTML + CSS + JS)，零构建依赖
- 确定交付形式: 网页，支持离线使用

**决策记录**:

| 决策项         | 决定                        | 理由                                          |
|:--------------|:---------------------------|:----------------------------------------------|
| 技术栈         | 纯 HTML/CSS/JS              | 零依赖，便于离线使用，降低维护成本                |
| UI 框架        | 不使用                      | 项目体量不大，避免框架学习成本和体积                |
| CSS 方案       | 自定义 CSS 变量              | 完全可控，支持主题切换                            |
| 数据库         | 不使用，用 localStorage      | 无需后端，简化部署                                |
| 语言支持       | 中英双语                     | 目标用户包含国内和海外中学生                       |

**产出**:
- `SPEC.md` v1.0 — 初始规格文档

### Session 2 — 2026-03-25 22:30 +0800

#### 主题: 规格细化与确认

**讨论要点**:
- 细化 5 课的章节内容和交互组件
- 确认双语实现方案: 同页内联，CSS 切换可见性
- 确认暗色模式实现方案: CSS 变量 + `data-theme`
- 确认进度保存方案: localStorage 结构

**决策记录**:

| 决策项         | 决定                                | 理由                                     |
|:--------------|:------------------------------------|:----------------------------------------|
| 双语方案       | 同页 `<span class="zh/en">` + CSS   | 切换即时，无需重新加载页面                  |
| 暗色模式       | 跟随系统偏好 + 手动切换               | 兼顾自动和用户手动偏好                      |
| 进度存储       | localStorage JSON                   | 简单可靠，无需后端                          |
| 打印样式       | `@media print` 隐藏交互，显示答案     | 方便教师打印为纸质讲义                      |
| 代码高亮       | 自定义 CSS 类                        | 代码量少，不值得引入 Prism.js               |

## 决策汇总

### 关键技术决策

| #  | 决策                          | 选择             | 备选方案                  | 决策理由                     |
|:--:|:------------------------------|:----------------|:------------------------|:-----------------------------|
| 1  | 前端框架                       | 不使用           | React / Vue             | 零构建，降低复杂度              |
| 2  | CSS 框架                       | 不使用           | Bootstrap / Tailwind    | 完全可控，体积更小              |
| 3  | 双语实现                       | CSS 可见性切换    | 多页面 / i18n 库          | 即时切换，零请求               |
| 4  | 暗色模式                       | CSS 变量切换     | CSS filter invert       | 精确控制各组件颜色              |
| 5  | 数据存储                       | localStorage    | IndexedDB / 后端 API     | 简单，无需后端                 |
| 6  | 代码高亮                       | 手动 CSS 类     | Prism.js / Highlight.js  | 代码量少，不值得引入库          |
| 7  | 滚动动画                       | IntersectionObserver | scroll 事件监听      | 性能更好                      |
| 8  | 测试框架                       | Python + BS4    | Playwright / Puppeteer   | 轻量，无需浏览器环境            |
| 9  | 测验结果渲染                    | JS 动态生成     | HTML 静态模板             | 无需修改 HTML 文件，自动适配所有页面 |
