神经网络与深度学习 Neural Networks & Deep Learning

探索大脑启发的AI架构 Explore brain-inspired AI architectures

3.1 从大脑到人工神经元 3.1 From Brain to Artificial Neurons

人脑有约860亿个神经元，它们通过突触相互连接，构成了思考的基础。科学家从中获得灵感，创造了人工神经元。 The brain has ~86 billion neurons connected by synapses, forming the basis of thought. Scientists drew inspiration to create artificial neurons.

生物神经元 Biological Neuron

  • 接收信号 → 处理信号 → 决定是否传递 → 输出信号 Receive signals → Process → Decide whether to pass on → Output

人工神经元 Artificial Neuron

  • 输入 × 权重 → 求和 → 激活函数 → 输出 Inputs × Weights → Sum → Activation function → Output
💡

类比：想象你和朋友投票决定去哪吃饭。每个人的意见（输入）有不同的影响力（权重），最终投票结果超过某个门槛（激活函数），就做出决定（输出）。 Analogy: Imagine you and friends voting on where to eat. Each opinion (input) has different influence (weight), the vote exceeds a threshold (activation function), and a decision is made (output).

3.2 神经网络的结构 3.2 Structure of Neural Networks

多个人工神经元连接在一起，就形成了神经网络。 Multiple artificial neurons connected together form a neural network.

输入层 Input Layer

接收原始数据（如图片的像素值） Receives raw data (e.g., pixel values of an image)

隐藏层 Hidden Layer(s)

处理和提取特征（深度学习 = 更多隐藏层） Process and extract features (deep learning = more hidden layers)

输出层 Output Layer

给出最终结果（如"这是一只猫"） Gives the final result (e.g., "this is a cat")

深度学习之所以叫"深度"，就是因为它有很多隐藏层。层数越多，能学到的特征越复杂。类比：流水线加工——每一层做一道工序，最终产出成品。 Deep learning is called "deep" because it has many hidden layers. More layers = more complex features. Analogy: assembly line—each layer does one step, producing the final product.

3.3 AI 怎么"看"东西 3.3 How AI "Sees" Things

对于计算机来说，图片就是一堆数字。一张100×100的灰度图片就是10000个0-255之间的数字。 For computers, images are just numbers. A 100×100 grayscale image is 10,000 numbers between 0-255.

ℹ️

彩色图片有3个通道（红、绿、蓝），所以100×100的彩色图片 = 30000个数字！ Color images have 3 channels (R, G, B), so a 100×100 color image = 30,000 numbers!

卷积层——特征提取器 Convolution Layer—Feature Extractor

+

用小窗口（滤波器）在图片上滑动，提取局部特征。第一层提取边缘，第二层提取纹理，更深的层提取更复杂的特征（眼睛、耳朵等）。 A small window (filter) slides across the image extracting local features. First layer extracts edges, second extracts textures, deeper layers extract complex features (eyes, ears, etc.).

池化层——信息压缩 Pooling Layer—Information Compression

+

把特征图缩小，保留最重要的信息，减少计算量。就像把高清照片缩成缩略图，主体信息还在。 Shrinks feature maps, keeping the most important info and reducing computation. Like shrinking a HD photo to a thumbnail—key info remains.

全连接层——做决策 Fully Connected Layer—Making Decisions

+

把提取到的所有特征综合起来，做出最终判断："这是一只猫！" Combines all extracted features to make the final judgment: "This is a cat!"

3.4 训练的秘密 3.4 The Secret of Training

1

前向传播 Forward Propagation

数据从输入层经过各层传到输出层，得到一个预测结果。 Data passes through layers from input to output, producing a prediction.

2

计算误差 Calculate Error

比较预测结果和正确答案，计算差距有多大。 Compare prediction with the correct answer, calculate how far off it is.

3

反向传播 Backpropagation

把误差信号从输出层往回传，找出每个权重对误差的"贡献"。 Pass error signal backward through layers, finding each weight's "contribution" to the error.

4

更新权重 Update Weights

调整权重，让下次预测更准确。这个过程重复成千上万次。 Adjust weights to make the next prediction more accurate. This repeats thousands of times.

💡

就像考试 → 对答案 → 找到错误原因 → 下次改正。AI就是在不断地"考试、纠错、进步"中学会的！ Like taking a test → checking answers → finding errors → correcting next time. AI learns through continuous "test, correct, improve"!

3.7 小测验 3.7 Quiz

Q1: 人工神经元的灵感来源是什么？ Q1: What inspired artificial neurons?

人工神经网络的设计灵感直接来源于生物大脑中神经元的工作方式。 Artificial neural networks are directly inspired by how neurons work in the biological brain.

Q2: "深度学习"中的"深度"指的是什么？ Q2: What does "deep" in "deep learning" refer to?

深度学习的"深度"指的是神经网络中隐藏层的数量。更多层意味着可以学习更复杂的特征。 "Deep" refers to the number of hidden layers. More layers means learning more complex features.

Q3: 卷积神经网络（CNN）主要用于处理什么类型的数据？ Q3: What type of data is CNN mainly used for?

CNN专门为处理图像数据而设计，通过卷积操作提取图像中的空间特征。 CNNs are designed for image data, extracting spatial features through convolution operations.

Q4: 在训练过程中，反向传播的作用是什么？ Q4: What is the role of backpropagation in training?

反向传播通过计算误差对每个权重的梯度，指导权重的调整方向和幅度。 Backpropagation calculates error gradients for each weight, guiding adjustment direction and magnitude.

Q5: 下面哪个说法关于图片在计算机中的表示是正确的？ Q5: Which statement about how computers represent images is correct?

对计算机来说，图片就是数字矩阵。每个像素用数字表示颜色和亮度。 For computers, images are number matrices. Each pixel is a number representing color and brightness.
