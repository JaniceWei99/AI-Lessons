## 导入环节（10 分钟）

> 同学们有跟 AI 聊过天吗？ChatGPT、文心一言、通义千问——这些都是聊天机器人。但你们知道吗，世界上第一个聊天机器人诞生于 1966 年！

**在大屏幕上展示聊天机器人进化时间线：**
- **1966 年 ELIZA**：只会简单的模式匹配，像心理咨询师一样把你的话变成问题反问你
- **2011 年 Siri**：苹果的语音助手，能听懂语音并回答
- **2022 年 ChatGPT**：能理解上下文、写文章、写代码

> 今天我们要从最简单的开始，自己造三种聊天机器人，亲身体验从"笨"到"聪明"的进化过程。

**趣味互动：** 在屏幕上展示一段 ELIZA 的经典对话记录，让学生猜哪些回复是机器人说的。

## 核心概念讲解

### 聊天机器人的两大流派

> 聊天机器人本质上分两种：
> 1. **规则型：** 人工设定关键词和回复规则，像查字典一样
> 2. **AI 型：** 通过大量对话数据训练，自己学会怎么回答

**类比：** 规则型就像一本写好的剧本，演员只能照着念；AI 型就像一个真正的演员，能即兴发挥。

## 实验指导

### Part A：规则型聊天机器人（20 分钟）

> 我们先造一个最简单的机器人。它的工作原理很简单——看到关键词就回复对应的答案。

**教师先演示代码结构，然后让学生补充更多规则：**

```python
def simple_chatbot(user_input):
    user_input = user_input.lower()
    
    if "你好" in user_input or "hello" in user_input:
        return "你好！很高兴见到你！"
    elif "天气" in user_input:
        return "今天天气不错，适合写代码！"
    elif "名字" in user_input:
        return "我叫小智，是一个简单的聊天机器人。"
    elif "再见" in user_input or "bye" in user_input:
        return "再见！下次再聊！"
    else:
        return "抱歉，我还不理解这个问题。"

# 对话循环
print("简易聊天机器人已启动（输入'退出'结束）")
while True:
    user = input("你: ")
    if user == "退出":
        break
    response = simple_chatbot(user)
    print(f"机器人: {response}")
```

**教学策略：** 让学生先运行基础版本，然后给 10 分钟时间自由添加更多关键词和回复。

> 大家试试跟它聊天，看看它能回答什么。然后想想——它有什么明显的缺点？（引导学生发现：不能理解同义词、不记住上下文、无法处理没见过的问题）

### Part B：AI 聊天机器人（25 分钟）

> 规则型机器人很"死板"对吧？现在我们来用 AI 模型，让机器人真正"思考"。

**此环节建议教师先完整演示一轮对话，再让学生动手：**

```python
from transformers import pipeline, Conversation

chatbot = pipeline("conversational", model="microsoft/DialoGPT-small")

conversation = Conversation("Hello! How are you today?")
result = chatbot(conversation)
print(result.generated_responses[-1])
```

> DialoGPT 是微软基于 Reddit 对话数据训练的模型。它能理解上下文，回复更自然。

**多轮对话演示：**

```python
conversation.add_user_input("What do you like to do for fun?")
result = chatbot(conversation)
print(result.generated_responses[-1])
```

**给学生 10 分钟自由对话时间。** 提醒学生记录有趣的对话片段，后续分享。

**注意事项：** DialoGPT-small 模型较小，回复质量有限。提前告知学生它可能会给出奇怪的回答，这是正常的——因为模型太小了。

### Part C：个性化机器人（20 分钟）

> 最后一个实验最有趣——我们要给 AI 赋予"性格"！通过系统提示词（System Prompt），我们可以让同一个 AI 变成海盗、科学家或诗人。

**教师展示三种性格模板：**

```python
personalities = {
    "海盗": "You are a friendly pirate. Use pirate language like 'Arr!' and 'matey'. Talk about treasure and the sea.",
    "科学家": "You are a brilliant scientist. Explain everything with scientific facts. Be curious and enthusiastic about discoveries.",
    "诗人": "You are a romantic poet. Speak in poetic language with metaphors. See beauty in everything."
}
```

> 大家看，仅仅通过改变提示词的描述，AI 的说话方式就完全不同了。这就是 Prompt Engineering（提示词工程）的力量，我们在第 9 课会深入学习。

**此时让学生自由设计角色：** 每人创造一个独特的 AI 性格（比如：美食家、太空探险家、古代皇帝），写出对应的系统提示词并测试。

**小组展示：** 每组选一个最有趣的角色，向全班展示对话效果。

## 常见问题

| 问题 | 解决方案 |
|------|----------|
| 规则机器人无法识别输入 | 检查 `lower()` 转换，确保关键词匹配大小写一致 |
| DialoGPT 下载失败 | 检查网络连接，或使用教师预下载的模型缓存 |
| AI 回复出现乱码或重复 | 调整 `max_length` 和 `temperature` 参数 |
| `input()` 在 Notebook 中无法使用 | 确保使用标准 Jupyter Notebook（非 JupyterLab 旧版本） |
| 模型回复内容不当 | 提醒学生：小模型缺乏安全过滤机制，这是 AI 安全的重要话题 |

## 总结与回顾（10 分钟）

> 今天我们造了三种聊天机器人，来做个对比：

**板书总结表：**

| 类型 | 优点 | 缺点 |
|------|------|------|
| 规则型 | 可控、可预测、速度快 | 死板、需人工维护 |
| DialoGPT | 能理解语义、自动回复 | 可能回答奇怪、需算力 |
| 个性化 Bot | 角色丰富、可定制 | 依赖提示词质量 |

> 真正的 ChatGPT 比我们今天用的模型大几千倍，所以它才能那么聪明。但基本原理是相通的！
