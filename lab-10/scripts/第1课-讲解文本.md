# 第1课：AI 实验入门 (Getting Started)

## 课程标题

**AI 实验室启航——环境搭建与第一次 AI 体验**

## 教学目标

1. 学生能够独立启动并使用 Jupyter Notebook 编写 Python 代码
2. 掌握 Python 基础语法：`print()` 输出、基本数学运算
3. 了解如何安装第三方库（`openai`、`transformers`）
4. 成功调用 Hugging Face 情感分析 API，理解 AI 模型的基本使用方式
5. 体验 GPT-2 文本生成，感受 AI 的创造力

## 课时建议

**总时长：90 分钟**（2 课时）

## 教学流程

| 环节 | 时间 | 内容 |
|------|------|------|
| 导入环节 | 10 分钟 | AI 应用展示与课程介绍 |
| 环境搭建 | 15 分钟 | 安装 Jupyter Notebook，熟悉界面 |
| Python 基础 | 15 分钟 | print、变量、数学运算 |
| 安装 AI 库 | 10 分钟 | pip install 演示与操作 |
| 情感分析实验 | 15 分钟 | 调用 Hugging Face 模型 |
| 文本生成实验 | 15 分钟 | GPT-2 文本生成体验 |
| 总结与挑战 | 10 分钟 | 回顾要点，布置挑战任务 |

## 导入环节（10 分钟）

> 同学们好！欢迎来到 AI 实验室的第一课。大家平时有没有用过 Siri、小爱同学，或者 ChatGPT？（等待学生回应）这些都是人工智能的应用。今天我们要做一件很酷的事情——自己动手写代码，让 AI 帮我们分析文字的情绪、甚至帮我们写故事！

先在大屏幕上展示几个 AI 应用的例子：智能翻译、图像识别、聊天机器人。让学生对接下来十节课的内容有一个整体印象。

**教学提示：** 此时用 1-2 分钟快速演示一个已经写好的情感分析代码运行结果，给学生一个"终点预览"，激发学习动力。

## 核心概念讲解

### 环境搭建（15 分钟）

> 我们写 AI 代码的工具叫 Jupyter Notebook，它就像一个交互式的笔记本，可以一边写代码一边看结果。

**此时让学生打开 Jupyter Notebook。** 教师在大屏幕上同步演示：
1. 打开终端/命令行
2. 输入 `jupyter notebook` 启动服务
3. 在浏览器中新建一个 Notebook

**注意观察：** 巡视教室，确认每位学生都成功打开了 Notebook 界面。常见问题是浏览器没有自动弹出，提醒学生手动打开 `http://localhost:8888`。

### Python 基础速览（15 分钟）

> Python 是最适合做 AI 的编程语言，语法非常简洁。我们先学三个最基本的操作。

**教师演示代码（投屏），学生跟着敲：**

```python
# 第一行代码
print("你好，AI 世界！")

# 基本数学
print(2 + 3)
print(10 * 5)

# 变量
name = "小明"
print(f"欢迎 {name} 来到 AI 实验室！")
```

**教学策略：** 每写一段代码，先让学生猜测输出结果，再运行验证。这样能保持互动性。

### 安装 AI 库（10 分钟）

> 接下来我们要安装几个 AI 工具包。就像手机装 App 一样，Python 也需要安装扩展包。

**教师演示，学生跟做：**
```python
!pip install openai transformers torch
```

**常见问题提醒：** 安装过程可能需要 2-3 分钟，此时可以给学生解释什么是"库"和"包管理器"。如果安装报错，先检查网络连接。

## 实验指导

### 实验一：情感分析（15 分钟）

> 现在到了最激动人心的环节！我们要让 AI 读懂文字背后的情绪。

**教师先演示完整代码运行，然后让学生独立输入：**

```python
from transformers import pipeline
classifier = pipeline("sentiment-analysis")
result = classifier("I love this class!")
print(result)
```

> 大家看，AI 告诉我们这句话是"POSITIVE"（积极的），置信度接近 1.0。现在请大家试试输入不同的英文句子，看看 AI 的判断准不准确。

**给学生 5 分钟自由探索时间。** 巡视教室，鼓励学生尝试不同类型的句子。

### 实验二：GPT-2 文本生成（15 分钟）

> 接下来我们让 AI 来写故事！GPT-2 是 OpenAI 开发的文本生成模型。

**此环节建议教师先完整演示一次，再让学生动手：**

```python
generator = pipeline("text-generation", model="gpt2")
story = generator("Once upon a time", max_length=50)
print(story[0]['generated_text'])
```

> 每个人生成的故事可能都不一样，这就是 AI 的有趣之处。请大家尝试不同的开头，比如 "The robot walked into..."。

## 常见问题

| 问题 | 解决方案 |
|------|----------|
| `ModuleNotFoundError` | 重新运行 `!pip install` 命令 |
| 模型下载很慢 | 检查网络；教师可提前下载好模型分发 |
| Jupyter 无法启动 | 确认 Python 环境已正确安装 |
| 中文乱码 | 确保 Notebook 使用 UTF-8 编码 |
| 内存不足 | 关闭其他程序，重启 Kernel |

## 总结与回顾（5 分钟）

> 今天我们完成了三件大事：搭建了 AI 实验环境、写了人生第一段 Python 代码、还让 AI 帮我们分析情感和写故事。大家回想一下，`pipeline` 这个函数的作用是什么？（等待回答）对，它是一个"快捷通道"，能帮我们快速加载和使用 AI 模型。

让学生用一句话总结今天学到的最有趣的东西。
