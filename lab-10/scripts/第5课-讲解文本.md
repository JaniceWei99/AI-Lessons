## 导入环节（8 分钟）

> 同学们，先来看几幅画。（展示 3-4 幅 AI 生成的艺术作品）猜猜看，这些画是人画的还是 AI 画的？

**等待学生讨论后揭晓答案：** 全部由 AI 生成。

> 2022 年，一幅 AI 生成的画作获得了美国科罗拉多州博览会的艺术比赛冠军，引发了巨大争议。今天我们也要成为"AI 艺术家"，用文字指挥 AI 来画画。

**关键一问：** AI 只认识数字，它怎么理解"画一只在星空下飞翔的猫"这种描述呢？带着这个问题，我们进入原理讲解。

## 核心概念讲解（10 分钟）

### 扩散模型的工作原理

> 扩散模型的原理其实很优美。想象一下：

**类比（配合板书或PPT动画）：**

> 第一步"加噪"：拿一张清晰的照片，不断往上撒"数字噪点"，就像往照片上撒沙子，越撒越模糊，最后变成完全随机的噪点图——一团灰。
>
> 第二步"去噪"：AI 学会了一个本领——从一堆随机噪点中，一步一步去掉噪点，还原出清晰的图片。而且，AI 在去噪的时候会参考你给的文字描述来决定"还原"成什么样子。

> 简单说就是：**噪点 + 文字描述 → AI 一步步去噪 → 生成新图片**

### 提示词工程（Prompt Engineering for Art）

> 写提示词就像给画家下指令，越具体效果越好。好的提示词包含五个要素：

**板书公式：** `主体 + 风格 + 色彩 + 光照 + 细节`

举例：
- ❌ 差提示词："一只猫"
- ✅ 好提示词："A cute orange cat sitting on a windowsill, watercolor painting style, warm sunset lighting, soft pastel colors, detailed fur texture"

## 实验指导

### Part A：Craiyon 在线体验（12 分钟）

> 我们先用最简单的方式体验 AI 绘画。

**此时让学生打开浏览器访问 craiyon.com。** 教师同步演示：

1. 在输入框中输入提示词（英文效果更好）
2. 点击"Draw"按钮
3. 等待约 1 分钟生成结果

> 请大家输入自己想画的内容。先用简单描述试一次，然后用我们刚学的五要素公式改进提示词再试一次，对比两次的效果差异。

**给学生 8 分钟探索。** 巡视时引导学生观察：提示词越详细，生成效果越接近预期。

### Part B：Python SDXL-turbo 代码绘画（20 分钟）

> 在线工具虽然方便，但不够灵活。让我们用代码来生成画作。

**教师先完整演示一次，确认代码运行正常：**

```python
from diffusers import AutoPipelineForText2Image
import torch

pipe = AutoPipelineForText2Image.from_pretrained(
    "stabilityai/sdxl-turbo",
    torch_dtype=torch.float16,
    variant="fp16"
)

prompt = "A magical forest with glowing mushrooms, fantasy art style, moonlight, vibrant colors"
image = pipe(prompt=prompt, num_inference_steps=1, guidance_scale=0.0).images[0]
image.save("my_ai_art.png")
image.show()
```

> SDXL-turbo 是 Stability AI 的快速版模型，只需要 1 步就能生成图片！普通版的模型需要 20-50 步。

**让学生修改 prompt 变量的内容，生成自己的作品。** 给 10 分钟探索时间。

**重要提示：** 如果教室电脑没有 GPU，模型运行会较慢。可提前告知学生需要等待 30 秒到 1 分钟。

### Part C：批量风格生成（20 分钟）

> 现在我们用代码同时生成多种风格的画作，这就是编程的力量！

**教师演示代码框架，让学生理解循环生成的思路：**

```python
subject = "a cozy coffee shop on a rainy day"
styles = [
    "watercolor painting",
    "pixel art, 8-bit style",
    "anime illustration, Studio Ghibli style",
    "photorealistic, DSLR photo"
]

for i, style in enumerate(styles):
    prompt = f"{subject}, {style}, detailed, high quality"
    image = pipe(prompt=prompt, num_inference_steps=1, guidance_scale=0.0).images[0]
    image.save(f"style_{i+1}_{style.split(',')[0].replace(' ', '_')}.png")
    print(f"已生成风格 {i+1}: {style}")
```

> 同一个主题，四种完全不同的风格！请大家选择自己喜欢的主题，修改 `subject` 变量，生成四种风格的画作。

**此环节可以让学生两人一组合作。**

### Part D：提示词模板系统（10 分钟）

> 我们可以把提示词做成模板，方便复用。

```python
def create_prompt(subject, style, lighting, colors):
    return f"{subject}, {style}, {lighting} lighting, {colors} color palette, highly detailed, professional"

prompt = create_prompt(
    subject="a dragon flying over mountains",
    style="digital painting",
    lighting="dramatic sunset",
    colors="warm golden and red"
)
print(prompt)
```

> 这样我们只需要改参数，就能快速生成不同的提示词。这是程序员的思维方式——把重复的工作自动化。

## 常见问题

| 问题 | 解决方案 |
|------|----------|
| Craiyon 网站加载慢 | 检查网络；可改用其他在线工具如 Bing Image Creator |
| `diffusers` 安装失败 | `!pip install diffusers transformers accelerate` |
| GPU 内存不足 | 使用 `torch.float16` 精度，或减小图片尺寸 |
| 生成图片变形 | 改进提示词，添加"high quality, detailed"等描述 |
| 人物面部失真 | 当前小模型的通病，可添加"clear face, beautiful"等提示 |
