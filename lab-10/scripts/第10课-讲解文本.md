### 项目 A：智能花朵分类器（推荐初学者）

> 使用经典的 Iris 数据集，比较四种机器学习算法的分类效果。

**教师展示项目模板代码：**

```python
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.neighbors import KNeighborsClassifier
from sklearn.tree import DecisionTreeClassifier
from sklearn.svm import SVC
from sklearn.ensemble import RandomForestClassifier
from sklearn.metrics import accuracy_score, classification_report
import matplotlib.pyplot as plt

# 加载数据
iris = load_iris()
X_train, X_test, y_train, y_test = train_test_split(
    iris.data, iris.target, test_size=0.3, random_state=42
)

# 四种模型对比
models = {
    'KNN (K近邻)': KNeighborsClassifier(n_neighbors=3),
    'Decision Tree (决策树)': DecisionTreeClassifier(),
    'SVM (支持向量机)': SVC(),
    'Random Forest (随机森林)': RandomForestClassifier()
}

results = {}
for name, model in models.items():
    model.fit(X_train, y_train)
    pred = model.predict(X_test)
    acc = accuracy_score(y_test, pred)
    results[name] = acc
    print(f"{name}: 准确率 = {acc:.2%}")

# 可视化对比
plt.figure(figsize=(10, 5))
plt.bar(results.keys(), results.values(), color=['#FF6384', '#36A2EB', '#FFCE56', '#4BC0C0'])
plt.title('四种算法分类准确率对比')
plt.ylabel('准确率')
plt.ylim(0.8, 1.0)
plt.xticks(rotation=15)
plt.tight_layout()
plt.savefig('model_comparison.png')
plt.show()
```

> 项目 A 的核心任务：运行四种算法，对比准确率，分析哪个最好以及为什么。

### 项目 B：手写数字识别（推荐有一定基础的学生）

```python
from sklearn.datasets import load_digits
from sklearn.model_selection import train_test_split
from sklearn.ensemble import RandomForestClassifier
from sklearn.metrics import confusion_matrix, accuracy_score
import matplotlib.pyplot as plt
import numpy as np

# 加载 MNIST 简化版（8x8 像素）
digits = load_digits()

# 展示样本
fig, axes = plt.subplots(2, 5, figsize=(12, 5))
for i, ax in enumerate(axes.flat):
    ax.imshow(digits.images[i], cmap='gray')
    ax.set_title(f'标签: {digits.target[i]}')
    ax.axis('off')
plt.suptitle('手写数字样本', fontsize=16)
plt.tight_layout()
plt.show()

# 训练模型
X_train, X_test, y_train, y_test = train_test_split(
    digits.data, digits.target, test_size=0.3, random_state=42
)

model = RandomForestClassifier(n_estimators=100)
model.fit(X_train, y_train)
pred = model.predict(X_test)
print(f"准确率: {accuracy_score(y_test, pred):.2%}")
```

> 项目 B 的核心任务：训练模型识别手写数字，分析哪些数字最容易被混淆（如 3 和 8），可视化混淆矩阵。
