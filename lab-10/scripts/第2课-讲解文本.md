## 导入环节（8 分钟）

> 同学们，大家解锁手机时用的人脸识别，拍照时的自动对焦，甚至超市的自助结账机识别商品——这些都是图像识别技术。但你们有没有想过，计算机明明没有眼睛，它是怎么"看到"图片的呢？

**在大屏幕上展示：**
- 一张普通照片
- 同一张照片放大到像素级别（变成彩色方块）

> 大家看到了吗？放大之后，图片就变成了一堆彩色小方块。每个方块就是一个"像素"，计算机看到的就是这些数字。

## 核心概念讲解（12 分钟）

### 计算机如何看图像

> 一张图片在计算机里就是一个数字矩阵。每个像素有三个值：红（R）、绿（G）、蓝（B），范围是 0-255。

**教师用简单示例解释：**
- 纯红色 = (255, 0, 0)
- 纯白色 = (255, 255, 255)
- 纯黑色 = (0, 0, 0)

> 一张 224×224 的图片，计算机看到的就是 224×224×3 = 150,528 个数字！AI 模型的工作就是从这些数字中找出规律。

### 图像分类流程

用板书或PPT画出流程图：**输入图像 → 特征提取（边缘、纹理、形状）→ 分类器 → 输出标签（猫/狗/车...）**

> 今天我们用的 ViT（Vision Transformer）模型，是谷歌开发的，它把图片切成 16×16 的小块，就像拼图一样理解每个部分，再综合判断。

## 实验指导

### Part A：Google Teachable Machine（20 分钟）

> 我们先用一个不需要写代码的工具来体验图像识别。

**此时让学生打开浏览器，访问 teachablemachine.withgoogle.com。** 教师在大屏幕同步演示：

1. 选择"Image Project" → "Standard image model"
2. 建立两个分类，比如"举手"和"放下"
3. 用摄像头采集每个类别约 30 张样本
4. 点击"Train Model"开始训练
5. 测试模型效果

> 请大家用自己的摄像头训练一个模型，识别两个不同的手势。给大家 10 分钟时间。

**教学策略：** 让学生两人一组互相帮助。巡视教室时重点关注摄像头权限问题和样本数量不足的情况。

**关键提醒：** 训练样本要多样化——不同角度、不同距离、不同背景，这样模型才更准确。

### Part B：Python ViT 模型（25 分钟）

> 现在我们用代码来做图像识别，体验专业 AI 工程师的工作方式。

**教师先完整演示，确认代码能运行，然后让学生跟做：**

```python
from transformers import pipeline
from PIL import Image

# 加载预训练模型
classifier = pipeline("image-classification", 
                       model="google/vit-base-patch16-224")

# 加载并识别图片
image = Image.open("test_image.jpg")
results = classifier(image)

for result in results:
    print(f"{result['label']}: {result['score']:.2%}")
```

**重要：** 课前需准备好 3-5 张测试图片（猫、狗、汽车等常见物体），放在学生可以访问的共享文件夹中。

> 大家看输出结果，模型给出了前 5 个最可能的类别和对应的概率。最高概率的就是模型的"最佳猜测"。

**给学生 10 分钟尝试不同图片。** 鼓励学生从网上下载各种图片来测试。

### 批量分类实验（15 分钟）

> 如果有 100 张图片需要分类，总不能一张一张来吧？让我们写一个批量处理的程序。

**此环节教师演示代码框架，让学生补充关键部分：**

```python
import os
image_folder = "test_images/"
for filename in os.listdir(image_folder):
    if filename.endswith(('.jpg', '.png')):
        image = Image.open(os.path.join(image_folder, filename))
        results = classifier(image)
        top = results[0]
        print(f"{filename}: {top['label']} ({top['score']:.2%})")
```

## 常见问题

| 问题 | 解决方案 |
|------|----------|
| 摄像头权限被拒绝 | 在浏览器设置中允许摄像头访问 |
| Teachable Machine 训练失败 | 每个类别至少需要 30 张样本 |
| `PIL` 未安装 | 运行 `!pip install Pillow` |
| 图片格式不支持 | 转换为 jpg 或 png 格式 |
| ViT 模型下载慢 | 教师提前缓存模型，或使用离线包 |
| 识别结果不准确 | 确保图片清晰，物体居中 |
