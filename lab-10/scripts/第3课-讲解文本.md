## 核心概念讲解（12 分钟）

### 什么是 NLP？

> NLP 就是"自然语言处理"，让计算机理解人类的语言。你们每天用的输入法联想、搜索引擎、语音助手，背后都有 NLP 技术。

### 情感分析的数值化

> 情感分析就是把文字的情绪变成数字。我们用 -1 到 1 的范围：
> - **1** 表示非常积极（开心、满意、兴奋）
> - **0** 表示中性（没有明显情绪）
> - **-1** 表示非常消极（愤怒、失望、悲伤）

**类比：** 就像温度计测量温度一样，情感分析就是给文字的"情绪温度"打分。

**在白板上画一条数轴，标注 -1、0、1，让学生直观理解。**

## 实验指导

### Step 1：TextBlob 基础情感分析（15 分钟）

> TextBlob 是一个简单但实用的文本分析工具。我们先从它开始。

**教师先演示安装和基本用法：**

```python
!pip install textblob
from textblob import TextBlob

text = TextBlob("I love this amazing course!")
print(f"情感极性: {text.sentiment.polarity}")
print(f"主观性: {text.sentiment.subjectivity}")
```

> `polarity` 就是情感分数，`subjectivity` 表示这句话有多主观。数值越接近 1 越主观，越接近 0 越客观。

**此时让学生独立尝试 5 分钟：** 输入不同的英文句子，观察分数变化。

**互动环节：** 请 3 位学生分享他们测试的句子和得到的分数，全班讨论 AI 的判断是否合理。

### Step 2：Transformer 高级情感分析（15 分钟）

> TextBlob 是基于规则的，比较简单。现在我们用 Transformer 模型，它是通过大量数据训练出来的，通常更准确。

**教师演示代码，学生跟做：**

```python
from transformers import pipeline

analyzer = pipeline("sentiment-analysis")
result = analyzer("This movie is absolutely wonderful!")
print(result)
```

> 注意看，Transformer 给出的是分类标签（POSITIVE/NEGATIVE）加置信度分数，而 TextBlob 给出的是连续数值。两种方式各有优势。

**对比实验：** 让学生用同一句话分别在 TextBlob 和 Transformer 上分析，比较结果差异。

### Step 3：批量分析与报告（20 分钟）

> 实际工作中，我们需要同时分析大量文本。现在让我们处理一批产品评论。

**教师展示代码框架，让学生补充核心逻辑：**

```python
reviews = [
    "Great product, works perfectly!",
    "Terrible quality, broke after one day",
    "It's okay, nothing special",
    "Best purchase I've ever made!",
    "Waste of money, very disappointed",
    "Decent product for the price",
    "Absolutely love it, highly recommend!",
    "Poor customer service, won't buy again"
]

positive = 0
negative = 0

for review in reviews:
    result = analyzer(review)[0]
    label = result['label']
    score = result['score']
    print(f"[{label}] ({score:.2%}) {review}")
    if label == "POSITIVE":
        positive += 1
    else:
        negative += 1

total = len(reviews)
print(f"\n--- 分析报告 ---")
print(f"总评论数: {total}")
print(f"积极评论: {positive} ({positive/total:.0%})")
print(f"消极评论: {negative} ({negative/total:.0%})")
```

> 大家看，8 条评论中，AI 一下就帮我们统计出了积极和消极的比例。这在真实的商业场景中非常有用。

**让学生修改评论内容，加入自己写的句子，重新运行分析。** 给 10 分钟独立完成。

## 常见问题

| 问题 | 解决方案 |
|------|----------|
| TextBlob 安装后无法 import | 需要额外下载语料库：`python -m textblob.download_corpora` |
| Transformer 分析中文效果差 | 默认模型针对英文优化，中文需要用专门的中文模型 |
| 讽刺句子判断错误 | 向学生解释：讽刺是 NLP 的难题，当前模型难以理解反语 |
| 分析速度慢 | 模型首次加载需要时间，后续分析会快很多 |
| 内存警告 | 关闭无关 Notebook，重启 Kernel |

## 总结与回顾（5 分钟）

> 今天我们用了两种工具做情感分析：TextBlob 简单但基于规则，Transformer 更强大但需要更多资源。在实际应用中，选哪个取决于你的需求。谁能总结一下两者的主要区别？（等待学生回答）

**关键对比总结（板书）：**
- TextBlob：轻量、快速、返回数值、基于词典规则
- Transformer：准确度高、需要 GPU/时间、返回分类标签、基于深度学习
