## 导入环节（8 分钟）

> 同学们，我先给大家看两样东西。

**在大屏幕上展示：**
1. 一个密密麻麻的数字表格（100 行数据）
2. 同样数据画成的一张折线图

> 哪个让你更快看出趋势？（学生回答：图表）没错！科学研究表明，人脑处理图像的速度比处理文字快 **6 万倍**。这就是数据可视化存在的意义——把数字变成图片，让人一眼就能看出规律。

**再展示一个经典案例：** 南丁格尔的玫瑰图（历史上最著名的可视化之一），她用这张图说服了英国政府改善军医院卫生条件。

> 今天我们就要学会用代码画出漂亮的图表，让数据讲故事。

## 核心概念讲解（10 分钟）

### 图表选择指南

**板书/PPT 展示：**

| 数据类型 | 推荐图表 | 适用场景 |
|----------|----------|----------|
| 趋势变化 | 折线图 | 股价走势、温度变化 |
| 比例构成 | 饼图 | 市场份额、预算分配 |
| 数量比较 | 柱状图 | 各科成绩、销售额对比 |
| 相关关系 | 散点图 | 身高vs体重 |
| 分布情况 | 直方图 | 考试成绩分布 |

> 记住一个原则：**选图表就像选交通工具——不是最贵的最好，而是最适合的最好。** 短途走路，长途坐飞机；比较数量用柱状图，看趋势用折线图。

## 实验指导

### 实验一：AI 研究趋势折线图（20 分钟）

> 我们先画一张 AI 领域 2015-2024 年论文发表数量的趋势图。

**教师在大屏幕演示，学生跟做：**

```python
import matplotlib.pyplot as plt
import numpy as np

# 设置中文字体
plt.rcParams['font.sans-serif'] = ['SimHei', 'DejaVu Sans']
plt.rcParams['axes.unicode_minus'] = False

# AI 研究论文数据（模拟真实趋势）
years = list(range(2015, 2025))
papers = [5800, 7200, 9500, 13200, 18500, 24000, 31000, 42000, 58000, 78000]

plt.figure(figsize=(10, 6))
plt.plot(years, papers, marker='o', color='#2196F3', linewidth=2, markersize=8)
plt.fill_between(years, papers, alpha=0.1, color='#2196F3')
plt.title('AI 研究论文发表趋势 (2015-2024)', fontsize=16, fontweight='bold')
plt.xlabel('年份', fontsize=12)
plt.ylabel('论文数量', fontsize=12)
plt.grid(True, alpha=0.3)
plt.tight_layout()
plt.savefig('ai_trend.png', dpi=150)
plt.show()
```

> 大家看这条曲线，2015年到2024年，AI论文数量几乎是指数增长！特别是2022年以后，ChatGPT发布，增长更加猛烈。

**此时让学生修改代码：** 改变颜色、线条样式、标题，给 5 分钟自由探索 matplotlib 的参数。

**常见问题提醒：** 中文字体显示可能出现方框。如果出现这个问题，添加字体设置代码或者使用英文标题替代。

### 实验二：AI 技术分布饼图与柱状图（20 分钟）

> 接下来我们看看 AI 各个方向的研究热度分布。

**教师演示饼图：**

```python
# AI 技术分布数据
categories = ['NLP\n自然语言处理', '计算机视觉', '强化学习', 
              '推荐系统', '语音识别', '其他']
percentages = [28, 25, 15, 12, 10, 10]
colors = ['#FF6384', '#36A2EB', '#FFCE56', '#4BC0C0', '#9966FF', '#FF9F40']
explode = (0.05, 0.05, 0, 0, 0, 0)

fig, axes = plt.subplots(1, 2, figsize=(14, 6))

# 饼图
axes[0].pie(percentages, labels=categories, colors=colors, explode=explode,
            autopct='%1.1f%%', startangle=90, textprops={'fontsize': 10})
axes[0].set_title('AI 技术研究方向分布', fontsize=14, fontweight='bold')

# 柱状图
axes[1].bar(categories, percentages, color=colors, edgecolor='white', linewidth=1.5)
axes[1].set_title('AI 技术研究方向对比', fontsize=14, fontweight='bold')
axes[1].set_ylabel('占比 (%)', fontsize=12)

plt.tight_layout()
plt.savefig('ai_distribution.png', dpi=150)
plt.show()
```

> 同一份数据，饼图强调"占比"关系，柱状图强调"数量"比较。选哪个取决于你想突出什么信息。

**让学生讨论：** 在这个例子中，你觉得饼图还是柱状图更有说服力？为什么？

### 实验三：模型训练可视化（20 分钟）

> 最后我们来画 AI 工程师每天都要看的图——训练曲线。

**教师解释概念后展示代码：**

```python
# 模拟模型训练数据
epochs = list(range(1, 21))
train_loss = [2.5 * np.exp(-0.15 * x) + 0.1 + np.random.normal(0, 0.05) for x in epochs]
val_loss = [2.5 * np.exp(-0.12 * x) + 0.2 + np.random.normal(0, 0.08) for x in epochs]
train_acc = [1 - 0.8 * np.exp(-0.2 * x) + np.random.normal(0, 0.02) for x in epochs]
val_acc = [1 - 0.85 * np.exp(-0.17 * x) + np.random.normal(0, 0.03) for x in epochs]

fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(14, 5))

# 损失曲线
ax1.plot(epochs, train_loss, 'b-o', label='训练损失', markersize=4)
ax1.plot(epochs, val_loss, 'r-s', label='验证损失', markersize=4)
ax1.set_title('模型损失曲线', fontsize=14, fontweight='bold')
ax1.set_xlabel('训练轮次 (Epoch)')
ax1.set_ylabel('损失值 (Loss)')
ax1.legend()
ax1.grid(True, alpha=0.3)

# 准确率曲线
ax2.plot(epochs, train_acc, 'b-o', label='训练准确率', markersize=4)
ax2.plot(epochs, val_acc, 'r-s', label='验证准确率', markersize=4)
ax2.set_title('模型准确率曲线', fontsize=14, fontweight='bold')
ax2.set_xlabel('训练轮次 (Epoch)')
ax2.set_ylabel('准确率 (Accuracy)')
ax2.legend()
ax2.grid(True, alpha=0.3)

plt.tight_layout()
plt.savefig('training_curves.png', dpi=150)
plt.show()
```

> 左边的损失曲线应该下降（越来越好），右边的准确率曲线应该上升（越来越准）。如果验证损失开始上升而训练损失还在下降，说明模型"过拟合"了——就像一个学生只会做练习题，考试却考不好。

**此环节重点讲解图表含义，让学生理解"读图"比"画图"更重要。**

## 常见问题

| 问题 | 解决方案 |
|------|----------|
| 中文显示为方框 | 安装中文字体并配置 matplotlib |
| 图片保存后打不开 | 检查文件路径，确认 `savefig` 在 `show` 之前 |
| 子图重叠 | 使用 `plt.tight_layout()` 自动调整间距 |
| 颜色搭配不好看 | 使用预定义配色方案如 `plt.cm.Set2` |
| plotly 安装后无法显示 | Jupyter 中需要额外配置：`pip install nbformat` |
