**现场演示：** 打开手机的语音输入功能，随机说一句话，展示实时转文字的效果。

## 核心概念讲解（12 分钟）

### 语音识别的三步流程

**板书或 PPT 展示流程图：**

> **第一步：声波采集** —— 麦克风把空气振动变成电信号，再转换为数字波形
>
> **第二步：特征提取** —— AI 把声波切成很多小片段（通常 25 毫秒一片），提取每片的频率特征。就像把一首歌拆成一个个音符
>
> **第三步：文本输出** —— 模型根据这些特征，预测最可能的文字序列

**类比：** 想象你在听一首歌，即使歌声模糊，你也能通过旋律猜出歌词。Whisper 就是这样一个"音乐达人"，只不过它听的是说话声，猜的是文字。

### Whisper 模型介绍

> Whisper 是 OpenAI 在 2022 年发布的，用 68 万小时的多语言音频数据训练。它最厉害的地方是：支持 99 种语言，包括中文！而且它能自动检测语言，不需要我们提前告诉它。

### 创建测试音频（10 分钟）

> 我们先用代码生成几段测试用的音频文件。

**教师演示，学生跟做：**

```python
from gtts import gTTS

# 生成英文音频
tts_en = gTTS("Hello, welcome to the AI laboratory. Today we will learn speech recognition.", lang='en')
tts_en.save("test_english.mp3")

# 生成中文音频
tts_zh = gTTS("你好，欢迎来到人工智能实验室。今天我们学习语音识别。", lang='zh-CN')
tts_zh.save("test_chinese.mp3")

print("音频文件已生成！")
```

> 大家可以双击生成的文件听听看效果。这是合成语音，虽然不如真人自然，但用来测试语音识别是够用的。

**让学生生成自己想说的句子的音频，给 3 分钟探索。**

### 语音识别实验（20 分钟）

> 现在到了重头戏！让 Whisper 来识别我们刚才生成的音频。

**教师演示英文识别：**

```python
import whisper

model = whisper.load_model("base")

# 识别英文
result = model.transcribe("test_english.mp3")
print("识别结果:", result["text"])
print("检测语言:", result["language"])
```

> 大家看，Whisper 不仅准确识别了文字内容，还自动检测出了语言是英文！

**接着演示中文识别：**

```python
result_zh = model.transcribe("test_chinese.mp3")
print("识别结果:", result_zh["text"])
print("检测语言:", result_zh["language"])
```

**此时让学生独立运行代码。** 给 5 分钟验证自己的识别结果。

**互动实验：** 如果条件允许（有麦克风），让学生录制自己的语音并识别。否则用 gTTS 生成不同内容的音频来测试。

> 大家注意观察：Whisper 对长句子和短句子的识别效果有区别吗？有没有识别错误的地方？

### 语音命令系统（20 分钟）

> 最后我们来做一个实用的东西——语音命令系统！就像 Siri 一样，根据语音命令执行不同操作。

**教师展示代码框架，讲解逻辑后让学生补充命令：**

```python
import datetime

def execute_command(text):
    text = text.lower().strip()
    
    if "time" in text or "时间" in text:
        now = datetime.datetime.now()
        return f"现在时间是 {now.strftime('%H:%M:%S')}"
    elif "weather" in text or "天气" in text:
        return "今天天气晴朗，气温 25°C，适合户外活动！"
    elif "joke" in text or "笑话" in text:
        return "为什么程序员总是分不清万圣节和圣诞节？因为 Oct 31 = Dec 25！"
    elif "hello" in text or "你好" in text:
        return "你好！我是你的语音助手，有什么可以帮你的？"
    elif "calculate" in text or "计算" in text:
        return "请告诉我你想计算什么？例如：2 + 3"
    else:
        return f"抱歉，我不理解命令：{text}"

# 测试命令系统
test_commands = ["What time is it?", "Tell me a joke", "Hello there"]
for cmd in test_commands:
    print(f"命令: {cmd}")
    print(f"回复: {execute_command(cmd)}")
    print()
```

> 这个系统把语音识别和命令执行结合起来了。在实际产品中，Siri 和 Alexa 也是类似的架构——先识别语音，再根据内容执行操作。

**让学生添加更多命令：** 给 10 分钟时间扩展功能。建议方向：日期查询、计时器、翻译请求等。

## 常见问题

| 问题 | 解决方案 |
|------|----------|
| `ffmpeg not found` 错误 | 需要系统级安装 ffmpeg：`apt install ffmpeg` |
| Whisper 模型下载慢 | 使用 `base` 或 `tiny` 模型，体积更小 |
| gTTS 需要网络连接 | 提前生成好音频文件供离线使用 |
| 中文识别出英文 | 指定语言参数：`model.transcribe("file.mp3", language="zh")` |
| 识别结果有错别字 | 正常现象；可使用更大的模型（medium/large）提升准确率 |
| 音频格式不支持 | 转换为 mp3 或 wav 格式 |
