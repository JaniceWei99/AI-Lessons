1.2 背景知识1.2 Background

什么是 API？ API（Application Programming Interface）就像一个"服务窗口"。你把请求递过去，AI 在后台处理，然后把结果递回来。就像点外卖 —— 你下单，厨房做菜，外卖员送到你手上。 What is an API? An API (Application Programming Interface) is like a "service window." You hand over a request, AI processes it behind the scenes, and hands back the result. It's like ordering food — you place the order, the kitchen cooks, and the delivery person brings it to you.

Jupyter Notebook 是一个在本地运行的交互式编程环境。它就像一个电子笔记本，可以同时写文字说明和运行代码。你可以用 pip install jupyter 安装，然后在终端输入 jupyter notebook 启动。 Jupyter Notebook is an interactive coding environment that runs locally on your computer. It's like a digital notebook where you can mix text and executable code. Install it with pip install jupyter, then launch it by typing jupyter notebook in your terminal.

Python 是目前最流行的 AI 编程语言。它的语法简单易懂，非常适合初学者。在接下来的 10 个实验中，我们都会用 Python 来编写 AI 项目。 Python is the most popular programming language for AI. Its syntax is simple and easy to read, making it perfect for beginners. We'll use Python for all 10 labs in this course.

步骤 4：调用你的第一个 AI APIStep 4: Make Your First AI API Call

现在让我们用一个免费的 AI 模型来完成一个任务。我们将使用 Hugging Face 的免费 API： Now let's use a free AI model to complete a task. We'll use Hugging Face's free API:

Python 
import requests

# 使用 Hugging Face 的免费情感分析 API
# Using Hugging Face's free sentiment analysis API
API_URL = "https://api-inference.huggingface.co/models/distilbert-base-uncased-finetuned-sst-2-english"

def analyze_sentiment(text):
    """Send text to AI and get sentiment analysis"""
    response = requests.post(API_URL, json={"inputs": text})
    return response.json()

# 测试几句话 / Test some sentences
sentences = [
    "I love learning about AI!",
    "This homework is so boring.",
    "Today is a beautiful day!"
]

for sentence in sentences:
    result = analyze_sentiment(sentence)
    print(f"Text: {sentence}")
    print(f"Result: {result}")
    print("---")

预期输出：AI 会告诉你每句话是"积极的"（POSITIVE）还是"消极的"（NEGATIVE），以及它的信心程度（一个 0 到 1 之间的数字）。 Expected Output: The AI will tell you if each sentence is "POSITIVE" or "NEGATIVE," along with a confidence score (a number between 0 and 1).

步骤 5：让 AI 帮你写故事Step 5: Have AI Write a Story for You

Python 
# 使用 Hugging Face 的文本生成 API
# Using Hugging Face's text generation API
API_URL = "https://api-inference.huggingface.co/models/gpt2"

def generate_text(prompt):
    """Generate text from a prompt using AI"""
    response = requests.post(API_URL, json={
        "inputs": prompt,
        "parameters": {"max_length": 100}
    })
    return response.json()

# 给 AI 一个故事开头 / Give AI a story beginning
story_start = "Once upon a time, a young student discovered a magical AI that could"
result = generate_text(story_start)
print("AI Generated Story:")
print(result[0]["generated_text"])
