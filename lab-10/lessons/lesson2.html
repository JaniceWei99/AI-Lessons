2.2 背景知识2.2 Background

计算机是怎么"看"图片的呢？对于人类来说，看一张猫的照片就知道是猫。但对于计算机来说，一张图片只是一堆数字 —— 每个像素都是一个数字，表示颜色的深浅。 How does a computer "see" images? For humans, looking at a photo of a cat instantly tells us it's a cat. But for computers, an image is just a bunch of numbers — each pixel is a number representing color intensity.

图像分类的原理：AI 通过分析大量的标注图片来学习规律。比如，它看了上万张猫的图片后，就能总结出"猫有什么特征" —— 尖耳朵、胡须、圆瞳孔等。这个过程叫做"训练"。 How image classification works: AI learns patterns by analyzing tons of labeled images. After seeing thousands of cat pictures, it can summarize "what makes a cat" — pointy ears, whiskers, round pupils, etc. This process is called "training."

第1步：收集数据Step 1: Collect Data

准备大量标注好的图片（比如"这是猫"、"这是狗"）Prepare lots of labeled images (like "this is a cat," "this is a dog")

第2步：训练模型Step 2: Train the Model

让 AI 反复学习这些图片，找到规律Let AI study these images repeatedly to find patterns

第3步：测试预测Step 3: Test & Predict

给 AI 一张新图片，它会告诉你这是什么Give AI a new image and it tells you what it sees

Part B：用 Python 构建图像分类器Part B: Build an Image Classifier with Python

在 Jupyter Notebook 中运行以下代码，使用预训练模型来识别图片： Run the following code in Jupyter Notebook to classify images using a pre-trained model:

Python 
# 安装必要的库 / Install required libraries
!pip install transformers Pillow torch

from transformers import pipeline
from PIL import Image
import requests
from io import BytesIO

# 加载预训练的图像分类模型
# Load a pre-trained image classification model
classifier = pipeline("image-classification", model="google/vit-base-patch16-224")

# 从网上下载一张图片 / Download an image from the web
url = "https://upload.wikimedia.org/wikipedia/commons/thumb/4/4d/Cat_November_2010-1a.jpg/1200px-Cat_November_2010-1a.jpg"
response = requests.get(url)
image = Image.open(BytesIO(response.content))

# 让 AI 识别图片 / Let AI classify the image
results = classifier(image)

# 显示结果 / Show results
print("AI thinks this image contains:")
for result in results[:3]:
    label = result["label"]
    score = result["score"]
    print(f"  {label}: {score:.1%} confidence")

预期输出：AI 应该会识别出这是一只猫，置信度很高（通常超过 90%）。你可以把 URL 换成任何图片链接来测试！ Expected Output: AI should identify this as a cat with high confidence (usually over 90%). Try changing the URL to any image link to test!

实验检查Lab Check

1. 计算机看到一张图片时，实际处理的是什么？1. When a computer "sees" an image, what is it actually processing?

图片在计算机中以像素数字的形式存储，每个像素表示一个颜色值。Images are stored as pixel numbers in computers, where each pixel represents a color value.

2. 训练图像分类器需要什么？2. What do you need to train an image classifier?

训练分类器需要大量带标签的图片，这样 AI 才能学习每种类别的特征。Training a classifier requires lots of labeled images so AI can learn the features of each category.

3. Teachable Machine 的优点是什么？3. What is the advantage of Teachable Machine?

Teachable Machine 是一个无代码平台，让任何人都能在浏览器中训练 AI 模型。Teachable Machine is a no-code platform that lets anyone train AI models right in the browser.

4. 置信度（confidence score）表示什么？4. What does a confidence score represent?

置信度是 0 到 1 之间的数字，表示 AI 对自己预测结果的确信程度。越接近 1 越确定。Confidence score is a number from 0 to 1 showing how sure the AI is about its prediction. Closer to 1 means more certain.

5. "预训练模型"是什么意思？5. What does "pre-trained model" mean?

预训练模型是已经用大量数据训练好的 AI 模型，我们可以直接拿来使用，无需自己训练。A pre-trained model is an AI model already trained on large datasets — we can use it directly without training ourselves.
