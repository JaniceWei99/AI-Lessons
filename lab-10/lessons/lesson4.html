4.3 动手实验4.3 Hands-On

Part A：构建规则型聊天机器人Part A: Build a Rule-Based Chatbot

Python 
import random

# 定义机器人的回复规则 / Define bot response rules
responses = {
    "hello": ["Hi there! How can I help you?", "Hello! Nice to meet you!"],
    "how are you": ["I'm doing great, thanks!", "I'm a bot, but I'm feeling good!"],
    "what is ai": ["AI is artificial intelligence - machines that can learn and think!",
                    "AI stands for Artificial Intelligence. It's about making smart machines."],
    "tell me a joke": ["Why did the robot go to school? To improve its algorithms!",
                       "What do you call a computer that sings? A-Dell!"],
    "bye": ["Goodbye! Have a great day!", "See you later! Keep learning!"],
}

def rule_bot(user_input):
    """Simple rule-based chatbot"""
    user_input = user_input.lower().strip()

    for key in responses:
        if key in user_input:
            return random.choice(responses[key])

    return "I don't understand that yet. Try asking about AI or telling me hello!"

# 测试机器人 / Test the bot
test_messages = ["Hello", "What is AI?", "Tell me a joke", "How are you?", "What's the weather?"]

print("=== Rule-Based Chatbot ===\n")
for msg in test_messages:
    reply = rule_bot(msg)
    print(f"You: {msg}")
    print(f"Bot: {reply}\n")

注意：规则型机器人很有限！试试问它"What's the weather?"——它不知道怎么回答。这就是我们需要 AI 的原因。 Note: The rule-based bot is very limited! Try asking "What's the weather?" — it can't answer. This is why we need AI.

Part B：升级为 AI 聊天机器人Part B: Upgrade to an AI Chatbot

Python 
!pip install transformers

from transformers import pipeline

# 加载对话 AI 模型 / Load conversational AI model
chatbot = pipeline("text-generation", model="microsoft/DialoGPT-small")

def ai_chat(user_message, chat_history=""):
    """AI-powered chatbot using DialoGPT"""
    prompt = f"{chat_history}User: {user_message}\nBot:"
    result = chatbot(prompt, max_length=200, do_sample=True,
                     temperature=0.7, top_p=0.9)
    generated = result[0]["generated_text"]

    # Extract just the bot's response
    bot_reply = generated.split("Bot:")[-1].strip()
    # Take only the first sentence
    bot_reply = bot_reply.split("\n")[0].strip()
    return bot_reply

# 测试 AI 聊天机器人 / Test the AI chatbot
print("=== AI Chatbot ===\n")
conversations = [
    "Hi! What's your name?",
    "Can you help me with my homework?",
    "What do you think about robots?",
    "Tell me something interesting about space.",
]

for msg in conversations:
    reply = ai_chat(msg)
    print(f"You: {msg}")
    print(f"AI Bot: {reply}\n")

Part C：添加个性化Part C: Add Personality

Python 
# 给机器人设定角色 / Give the bot a character
def themed_bot(user_input, theme="pirate"):
    """A chatbot with personality!"""
    themes = {
        "pirate": "You are a friendly pirate. Respond using pirate language with 'Arrr' and 'matey'.",
        "scientist": "You are a young scientist. Explain things using simple science facts.",
        "poet": "You are a poet. Respond in short rhyming verses."
    }

    system_prompt = themes.get(theme, themes["pirate"])
    prompt = f"[System: {system_prompt}]\nUser: {user_input}\nBot:"

    result = chatbot(prompt, max_length=150, do_sample=True, temperature=0.8)
    reply = result[0]["generated_text"].split("Bot:")[-1].strip().split("\n")[0]
    return reply

# 测试不同角色 / Test different personalities
print("=== Pirate Bot ===")
print(f"Bot: {themed_bot('Hello!', 'pirate')}\n")

print("=== Scientist Bot ===")
print(f"Bot: {themed_bot('Why is the sky blue?', 'scientist')}\n")

print("=== Poet Bot ===")
print(f"Bot: {themed_bot('Tell me about the moon', 'poet')}")

实验检查Lab Check

1. 第一个聊天机器人叫什么名字？1. What was the name of the first chatbot?

ELIZA 是 1966 年在麻省理工学院开发的第一个聊天机器人程序。ELIZA was the first chatbot program, developed at MIT in 1966.

2. 规则型聊天机器人的主要局限是什么？2. What is the main limitation of rule-based chatbots?

规则型机器人只能回答程序员预设的问题模式，遇到新问题无法灵活应对。Rule-based bots can only respond to preset question patterns — they can't handle new questions flexibly.

3. ChatGPT 属于什么类型的 AI？3. What type of AI is ChatGPT?

ChatGPT 是基于大语言模型（LLM）的 AI，通过学习大量文本来理解和生成语言。ChatGPT is based on Large Language Models (LLMs) that learn from vast amounts of text to understand and generate language.

4. temperature 参数控制 AI 回复的什么特性？4. What does the temperature parameter control in AI responses?

temperature 越高，AI 的回复越有创意和随机性；越低则越保守和确定。Higher temperature means more creative and random responses; lower temperature means more conservative and deterministic.

5. 在 Python 中，random.choice(list) 的作用是什么？5. In Python, what does random.choice(list) do?

random.choice() 从列表中随机选择并返回一个元素。random.choice() randomly selects and returns one element from a list.
