6.2 背景知识6.2 Background

语音识别的原理：声音其实就是空气中的振动波。麦克风把这些振动转换成数字信号（波形数据）。AI 模型分析这些波形的模式，找出对应的文字。现代语音识别模型（如 Whisper）可以识别多种语言，准确率超过 95%。 How speech recognition works: Sound is just vibration waves in the air. A microphone converts these vibrations into digital signals (waveform data). AI models analyze these waveform patterns to find matching text. Modern speech recognition models (like Whisper) can recognize many languages with over 95% accuracy.

声音输入Sound Input

麦克风捕捉声波，转换为数字波形Microphone captures sound waves, converts to digital waveform

特征提取Feature Extraction

AI 分析波形中的频率、音调、节奏等特征AI analyzes frequency, pitch, rhythm and other features in the waveform

文字输出Text Output

将特征匹配成文字，输出识别结果Matches features to words and outputs recognized text

提示：Whisper 模型有多种大小：tiny、base、small、medium、large。越大的模型越准确，但也越慢。在本地使用 base 或 small 就够了。 Tip: Whisper comes in sizes: tiny, base, small, medium, large. Larger = more accurate but slower. For local use, base or small is sufficient.

步骤 4：构建语音指令系统Step 4: Build a Voice Command System

Python 
import datetime

def process_command(text):
    """Process voice commands and perform actions"""
    text = text.lower().strip()

    if "time" in text or "what time" in text:
        now = datetime.datetime.now().strftime("%H:%M")
        return f"The current time is {now}"

    elif "weather" in text:
        return "I can't check real weather, but it's always sunny in the AI lab!"

    elif "joke" in text:
        return "Why do programmers prefer dark mode? Because light attracts bugs!"

    elif "hello" in text or "hi" in text:
        return "Hello! I'm your AI voice assistant. How can I help?"

    elif "calculate" in text or "math" in text:
        return "I'm ready for math! Try saying 'calculate 5 plus 3'"

    elif "thank" in text:
        return "You're welcome! Happy to help!"

    else:
        return f"I heard: '{text}', but I don't know that command yet."

# 模拟语音指令测试 / Simulate voice command tests
test_commands = [
    "Hello there!",
    "What time is it?",
    "Tell me a joke",
    "How's the weather?",
    "Thank you very much"
]

print("=== Voice Command System ===\n")
for cmd in test_commands:
    response = process_command(cmd)
    print(f"You said: \"{cmd}\"")
    print(f"Assistant: {response}\n")

实验检查Lab Check

1. 声音在物理层面上是什么？1. What is sound at the physical level?

声音是空气中的振动波（声波），由物体振动产生，通过空气传播到我们的耳朵。Sound is vibration waves in the air (sound waves), produced by vibrating objects and transmitted through the air to our ears.

2. OpenAI 的 Whisper 模型主要用来做什么？2. What is OpenAI's Whisper model primarily used for?

Whisper 是 OpenAI 开发的语音识别模型，专门用于将语音音频转换为文字。Whisper is a speech recognition model by OpenAI, specifically designed to convert speech audio to text.

3. gTTS 库的功能是什么？3. What does the gTTS library do?

gTTS（Google Text-to-Speech）将文字转换为语音音频文件。gTTS (Google Text-to-Speech) converts text into speech audio files.

4. Whisper 的哪个模型版本最准确？4. Which Whisper model version is the most accurate?

Whisper 的 large 模型最准确，但需要更多计算资源和时间。Whisper's large model is the most accurate, but requires more computing resources and time.

5. 语音识别中的"特征提取"步骤做什么？5. What does the "feature extraction" step in speech recognition do?

特征提取步骤分析音频波形中的频率、音调、节奏等关键特征，为后续的文字匹配做准备。Feature extraction analyzes key characteristics like frequency, pitch, and rhythm in the audio waveform to prepare for text matching.
