5.2 背景知识5.2 Background

AI 是怎么画画的？AI 绘画模型（如 Stable Diffusion、DALL-E）通过学习数十亿张图片来理解"什么样的描述对应什么样的画面"。当你输入文字提示，AI 从随机噪点开始，一步步去噪，最终生成一张与你描述匹配的图片。这个过程叫做"扩散模型"（Diffusion Model）。 How does AI draw? AI art models (like Stable Diffusion, DALL-E) learn from billions of images to understand "what descriptions match what visuals." When you type a text prompt, AI starts from random noise and gradually removes it step by step until an image matching your description appears. This is called a "Diffusion Model."

提示词工程（Prompt Engineering）是与 AI 绘画交互的关键技能。好的提示词能让 AI 生成令人惊叹的作品，而模糊的提示词可能得到奇怪的结果。一个好的提示词通常包含：主题、风格、颜色、光线和细节。 Prompt Engineering is the key skill for AI art. Good prompts produce stunning artwork, while vague prompts may give odd results. A good prompt typically includes: subject, style, colors, lighting, and details.

模糊提示词Vague Prompt

"画一只猫""Draw a cat"

精准提示词Detailed Prompt

"一只橘色小猫坐在窗台上，阳光透过窗帘照进来，水彩画风格，温暖色调""An orange kitten sitting on a windowsill, sunlight streaming through curtains, watercolor style, warm tones"

Part B：用 Python 生成 AI 艺术Part B: Generate AI Art with Python

Python 
# 安装所需库 / Install required libraries
!pip install diffusers transformers accelerate torch

from diffusers import DiffusionPipeline
import torch
from IPython.display import display

# 加载轻量级 AI 绘画模型
# Load a lightweight AI art model
pipe = DiffusionPipeline.from_pretrained(
    "stabilityai/sdxl-turbo",
    torch_dtype=torch.float16,
    variant="fp16"
)
pipe.to("cuda")  # Use GPU for faster generation

# 用文字生成图片 / Generate image from text
prompt = "A friendly robot teaching a class of kids about AI, colorful cartoon style"
image = pipe(prompt, num_inference_steps=4, guidance_scale=0.0).images[0]

# 显示生成的图片 / Display the generated image
display(image)

提示：如果你的电脑有 NVIDIA GPU，代码会自动使用 GPU 加速生成。如果没有 GPU，可以将 pipe.to("cuda") 改为 pipe.to("cpu")，生成速度会慢一些。 Tip: If your computer has an NVIDIA GPU, the code will automatically use it for faster generation. If you don't have a GPU, change pipe.to("cuda") to pipe.to("cpu") — generation will be slower.

步骤 3：批量生成不同风格Step 3: Batch Generate Different Styles

Python 
# 同一个主题，不同风格 / Same subject, different styles
subject = "a magical treehouse"
styles = [
    "watercolor painting, soft colors",
    "pixel art, retro 8-bit game style",
    "Japanese anime style, studio ghibli",
    "realistic photograph, golden hour lighting"
]

for i, style in enumerate(styles):
    full_prompt = f"{subject}, {style}"
    print(f"\nGenerating style {i+1}: {style}")
    image = pipe(full_prompt, num_inference_steps=4, guidance_scale=0.0).images[0]
    display(image)
    image.save(f"treehouse_style_{i+1}.png")
    print(f"Saved as treehouse_style_{i+1}.png")

步骤 4：提示词模板Step 4: Prompt Templates

Python 
# 创建提示词模板来系统地探索
# Create prompt templates for systematic exploration
def build_prompt(subject, style, mood, details):
    """Build a structured prompt from components"""
    return f"{subject}, {style}, {mood} atmosphere, {details}"

# 使用模板生成 / Generate using the template
my_prompt = build_prompt(
    subject="a young explorer discovering an ancient AI temple",
    style="digital concept art",
    mood="mysterious and awe-inspiring",
    details="dramatic lighting, highly detailed, 4k"
)

print(f"Prompt: {my_prompt}")
image = pipe(my_prompt, num_inference_steps=4, guidance_scale=0.0).images[0]
display(image)

实验检查Lab Check

1. AI 绘画模型生成图片的起点是什么？1. What does an AI art model start with when generating an image?

扩散模型从随机噪点开始，逐步去噪来生成与提示词匹配的图片。Diffusion models start from random noise and gradually remove it to generate images matching the prompt.

2. 什么是"提示词工程"？2. What is "Prompt Engineering"?

提示词工程是精心设计文字提示来引导 AI 输出理想结果的技能。Prompt Engineering is the skill of carefully crafting text prompts to guide AI toward producing ideal results.

3. 一个好的 AI 绘画提示词应该包含什么？3. What should a good AI art prompt include?

详细的提示词包含主题、艺术风格、颜色、光线和细节描述，能帮助 AI 生成更好的结果。Detailed prompts with subject, art style, colors, lighting, and details help AI generate better results.

4. DALL-E 和 Stable Diffusion 都属于哪种类型的 AI 模型？4. What type of AI model are DALL-E and Stable Diffusion?

DALL-E 和 Stable Diffusion 都是基于扩散原理的图像生成 AI 模型。DALL-E and Stable Diffusion are both image generation AI models based on the diffusion principle.

5. 为什么 AI 绘画在 GPU 上运行更快？5. Why does AI art generation run faster on a GPU?

GPU（图形处理器）有数千个核心，擅长并行计算，非常适合 AI 需要的大量矩阵运算。GPUs have thousands of cores that excel at parallel computing, making them ideal for the massive matrix operations AI requires.
