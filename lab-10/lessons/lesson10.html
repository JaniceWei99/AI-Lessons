10.3 项目模板代码 10.3 Project Template Code

以下是项目 A（花卉分类器）的完整模板代码。其他项目可以参考类似结构。 Below is the complete template for Project A (Flower Classifier). Other projects can follow a similar structure.

Python 
# === 毕业项目：花卉智能分类器 ===
# === Final Project: Smart Flower Classifier ===

from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split, cross_val_score
from sklearn.preprocessing import StandardScaler
from sklearn.neighbors import KNeighborsClassifier
from sklearn.tree import DecisionTreeClassifier
from sklearn.svm import SVC
from sklearn.ensemble import RandomForestClassifier
from sklearn.metrics import classification_report, confusion_matrix
import matplotlib.pyplot as plt
import numpy as np

# 步骤1: 加载和准备数据 / Step 1: Load and prepare data
print("=" * 50)
print("毕业项目：花卉智能分类器")
print("=" * 50)

iris = load_iris()
X, y = iris.data, iris.target

# 数据标准化 / Standardize data
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

# 划分训练集和测试集 / Split train/test
X_train, X_test, y_train, y_test = train_test_split(
    X_scaled, y, test_size=0.2, random_state=42
)

print(f"\n数据集大小: {len(X)} 样本")
print(f"训练集: {len(X_train)} | 测试集: {len(X_test)}")

# 步骤2: 定义多个模型 / Step 2: Define multiple models
models = {
    "KNN (k=5)": KNeighborsClassifier(n_neighbors=5),
    "决策树 / Decision Tree": DecisionTreeClassifier(random_state=42),
    "SVM": SVC(kernel="rbf", random_state=42),
    "随机森林 / Random Forest": RandomForestClassifier(
        n_estimators=100, random_state=42
    ),
}

# 步骤3: 训练和评估 / Step 3: Train and evaluate
print("\n--- 模型对比结果 / Model Comparison ---")
results = {}
for name, model in models.items():
    # 交叉验证 / Cross-validation
    cv_scores = cross_val_score(model, X_train, y_train, cv=5)
    model.fit(X_train, y_train)
    test_score = model.score(X_test, y_test)
    results[name] = {"cv": cv_scores.mean(), "test": test_score}
    print(f"\n{name}:")
    print(f"  交叉验证: {cv_scores.mean():.4f} (+/- {cv_scores.std():.4f})")
    print(f"  测试准确率: {test_score:.4f}")

# 步骤4: 找出最佳模型 / Step 4: Find the best model
best_name = max(results, key=lambda k: results[k]["test"])
print(f"\n{'='*50}")
print(f"最佳模型 / Best Model: {best_name}")
print(f"测试准确率 / Test Accuracy: {results[best_name]['test']:.4f}")
print(f"{'='*50}")

print("\n项目完成！/ Project Complete!")

10.4 项目报告模板 10.4 Project Report Template

完成代码后，请按以下结构撰写项目报告： After completing the code, write your project report following this structure:

Python 
# 项目报告生成器 / Project report generator

def generate_report(project_name, results):
    report = f"""
{'='*60}
  项目报告 / Project Report: {project_name}
{'='*60}

1. 项目目标 / Objective
   描述你要解决的问题和目标...
   Describe the problem and your goals...

2. 数据描述 / Data Description
   - 数据来源 / Source: ______
   - 样本数量 / Samples: ______
   - 特征数量 / Features: ______
   - 类别数量 / Classes: ______

3. 方法 / Methodology
   - 数据预处理 / Preprocessing: ______
   - 使用的算法 / Algorithms: ______
   - 评估方法 / Evaluation: ______

4. 结果 / Results
"""
    for name, scores in results.items():
        report += f"   {name}: CV={scores['cv']:.4f}, Test={scores['test']:.4f}\n"

    report += """
5. 结论 / Conclusion
   最佳模型是什么？为什么效果好？
   Which model is best? Why does it perform well?

6. 收获 / What I Learned
   这个项目让我学到了...
   From this project I learned...
"""
    return report

# 示例使用 / Example usage
sample_results = {
    "KNN": {"cv": 0.9583, "test": 1.0},
    "SVM": {"cv": 0.9667, "test": 1.0},
}
print(generate_report("花卉分类器", sample_results))

10.6 毕业测验 10.6 Final Quiz

Q1. 在机器学习中，将数据分为训练集和测试集的目的是什么？ Q1. In ML, why do we split data into training and test sets?

测试集模拟真实世界的新数据，用来检验模型是否真正学到了规律而不是死记硬背。 The test set simulates real-world new data, checking if the model truly learned patterns rather than memorizing.

Q2. 以下哪种是无监督学习算法？ Q2. Which of these is an unsupervised learning algorithm?

K-Means 是无监督学习——它不需要标签数据，而是自动发现数据中的自然分组。 K-Means is unsupervised learning—it doesn't need labeled data, automatically discovering natural groupings.

Q3. 交叉验证（cross-validation）相比简单的训练/测试拆分，主要优势是什么？ Q3. What is the main advantage of cross-validation over a simple train/test split?

交叉验证通过多次分组训练和测试，减少了单次拆分的随机性，给出更稳定可靠的性能评估。 Cross-validation trains/tests on multiple splits, reducing randomness from a single split, giving more stable performance estimates.

Q4. 在文本分析中，TF-IDF 的作用是什么？ Q4. What does TF-IDF do in text analysis?

TF-IDF 把文本变成数字向量，同时通过逆文档频率降低常见词的权重，突出有区分度的词语。 TF-IDF converts text to numeric vectors while down-weighting common words via inverse document frequency, highlighting distinctive terms.

Q5. 一个好的 AI 项目应该包含以下哪些步骤？ Q5. A good AI project should include which of these steps?

完整的 AI 项目流程：理解问题 → 数据准备 → 选择和训练模型 → 评估效果 → 撰写报告。每一步都不可少。 Complete AI project pipeline: understand the problem → prepare data → select & train models → evaluate → write report. Every step matters.
