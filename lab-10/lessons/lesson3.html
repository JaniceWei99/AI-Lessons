3.2 背景知识3.2 Background

什么是 NLP？ NLP（Natural Language Processing，自然语言处理）是 AI 的一个分支，专门研究如何让计算机理解和处理人类语言。你每天用的搜索引擎、翻译工具、语音助手背后都有 NLP 技术。 What is NLP? NLP (Natural Language Processing) is a branch of AI focused on helping computers understand and process human language. Search engines, translation tools, and voice assistants all use NLP behind the scenes.

情感分析是 NLP 中最常见的应用之一。商家用它来分析用户评论，社交媒体用它来检测有害内容，新闻机构用它来了解公众情绪。 Sentiment analysis is one of the most common NLP applications. Businesses use it to analyze reviews, social media platforms use it to detect harmful content, and news agencies use it to gauge public opinion.

步骤 2：使用 TextBlob 进行简单情感分析Step 2: Simple Sentiment Analysis with TextBlob

Python 
from textblob import TextBlob

# 创建一个简单的情感分析函数
# Create a simple sentiment analysis function
def analyze(text):
    blob = TextBlob(text)
    polarity = blob.sentiment.polarity  # -1 (negative) to 1 (positive)

    if polarity > 0.1:
        mood = "Positive"
    elif polarity < -0.1:
        mood = "Negative"
    else:
        mood = "Neutral"

    return mood, polarity

# 测试不同的句子 / Test different sentences
test_sentences = [
    "I absolutely love this new game! It's so much fun!",
    "The weather today is terrible and I feel awful.",
    "I went to the store to buy some milk.",
    "This is the best day of my life!",
    "I'm so disappointed with this product."
]

print("=== Sentiment Analysis Results ===\n")
for sentence in test_sentences:
    mood, score = analyze(sentence)
    print(f"Text: {sentence}")
    print(f"Mood: {mood} (score: {score:.2f})")
    print()

提示：情感分数范围是 -1 到 1。接近 1 表示非常积极，接近 -1 表示非常消极，接近 0 表示中性。 Tip: The sentiment score ranges from -1 to 1. Close to 1 = very positive, close to -1 = very negative, close to 0 = neutral.

步骤 3：使用 AI 模型进行高级情感分析Step 3: Advanced Sentiment Analysis with an AI Model

Python 
from transformers import pipeline

# 加载预训练的情感分析模型
# Load a pre-trained sentiment analysis model
sentiment_model = pipeline("sentiment-analysis")

# 分析电影评论 / Analyze movie reviews
reviews = [
    "The special effects were incredible and the story kept me on the edge of my seat!",
    "Waste of money. The acting was wooden and the plot made no sense.",
    "It was okay, nothing special but not bad either.",
    "A masterpiece! One of the best films I've ever seen.",
    "I fell asleep halfway through. So boring!"
]

print("=== Movie Review Analysis ===\n")
for review in reviews:
    result = sentiment_model(review)[0]
    label = result["label"]
    score = result["score"]
    emoji = "👍" if label == "POSITIVE" else "👎"
    print(f"{emoji} [{label} {score:.1%}]")
    print(f"   \"{review}\"")
    print()

步骤 4：创建评论分析报告Step 4: Create a Review Analysis Report

Python 
# 批量分析并生成报告 / Batch analyze and generate a report
all_reviews = [
    "Great product, works perfectly!",
    "Terrible quality, broke after one day.",
    "Exactly what I expected, good value.",
    "Don't buy this! Total waste of money.",
    "My kids love it! Best purchase this year.",
    "Average product, nothing impressive.",
    "Absolutely fantastic! Highly recommend!",
    "Poor customer service, very disappointed."
]

positive = 0
negative = 0

for review in all_reviews:
    result = sentiment_model(review)[0]
    if result["label"] == "POSITIVE":
        positive += 1
    else:
        negative += 1

total = len(all_reviews)
print("=== Product Review Report ===")
print(f"Total reviews analyzed: {total}")
print(f"Positive: {positive} ({positive/total:.0%})")
print(f"Negative: {negative} ({negative/total:.0%})")
print(f"\nOverall sentiment: {'Mostly Positive' if positive > negative else 'Mostly Negative'}")

实验检查Lab Check

1. NLP 代表什么？1. What does NLP stand for?

NLP 是 Natural Language Processing（自然语言处理）的缩写，是 AI 中处理人类语言的分支。NLP stands for Natural Language Processing — the AI branch that deals with human language.

2. 情感分析中，分数接近 0 表示什么？2. In sentiment analysis, what does a score near 0 mean?

情感分数接近 0 表示文本是中性的，没有明显的积极或消极情感。A sentiment score near 0 means the text is neutral — no strong positive or negative emotion.

3. 情感分析的常见应用场景是什么？3. What is a common use case for sentiment analysis?

企业经常使用情感分析来自动分析大量的用户产品评论，了解客户满意度。Businesses frequently use sentiment analysis to automatically process large volumes of product reviews and gauge customer satisfaction.

4. TextBlob 的 polarity 值范围是什么？4. What is the range of TextBlob's polarity value?

TextBlob 的 polarity 值范围是 -1（最消极）到 1（最积极）。TextBlob's polarity ranges from -1 (most negative) to 1 (most positive).

5. 对 AI 情感分析来说，以下哪种情况最难处理？5. Which is the hardest for AI sentiment analysis to handle?

反讽和讽刺是情感分析中最难处理的，因为字面意思和真实意思是相反的。Irony and sarcasm are hardest for sentiment analysis because the literal meaning is opposite to the intended meaning.
